A near-autonomous AI chemist improves a challenging reaction in medicinal chemistry (1 minute read)

OpenAI connected GPT-5.4 to Molecule.one's Maria, an agentic chemistry system tied to a high-throughput lab, and tasked it with improving Chan-Lam coupling, a reaction used to form carbon-nitrogen bonds. The model identified primary sulfonamides as a hard substrate class and proposed mild oxidants like TEMPO, lifting mean yield from 16.6% to 25.2% across two experimental cycles.

Predicting model behavior before release by simulating deployment (1 minute read)

OpenAI introduced Deployment Simulation, a method that replays prior conversations in a privacy-preserving way against a candidate model to estimate how it will behave before release. Across multiple GPT-5-series Thinking deployments, it improved estimates of undesired behavior rates, surfaced novel forms of misalignment, and reduced the risk that models could tell they were being tested.

Z.ai released GLM-5.2, a 753B-parameter open-weights model under an MIT license that beats GPT-5.5 on several long-horizon coding benchmarks at roughly one-sixth the cost. It ships with a 1M-token context window, $12.60-per-month enterprise tiers, and an IndexShare architecture that reuses one indexer across every four sparse attention layers to cut per-token compute by 2.9x.

Where Did the Variability Go? From Vibe Coding to Product Lines by Regeneration (1 minute read)

This paper examines what happens to software variability in vibe coding, where an LLM generates an entire program from a natural language prompt. An analysis of 10 vibe-coded C/C++ projects found near-zero in-artifact variability, since all configuration decisions get resolved at generation time. The authors propose Variability by Regeneration, a product-line approach where the LLM generates a separate dead-code-free binary for each variant from a declarative spec, with a dispatcher routing requests to the matching build.

Explaining Attention with Program Synthesis (1 minute read)

This paper approximates transformer attention heads with executable Python programs by prompting a language model to generate code that reproduces each head's attention patterns, then re-ranking by held-out fit. Fewer than 1,000 programs match heads in GPT-2, TinyLlama-1.1B, and Llama-3B above 75% IoU on TinyStories, and replacing 25% of heads with these surrogates raises perplexity by only 16% while preserving QA benchmark performance.

STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability (1 minute read)

STARE is a reinforcement learning method that stabilizes policy entropy in GRPO-style post-training by reweighting the effective advantages of entropy-critical tokens, identified through batch-internal surprisal quantiles and regulated by a target-entropy closed-loop gate. Tested across 1.5B to 32B models on short CoT, long CoT, and multi-turn tool use, it sustains stable training over thousands of steps and beats DAPO on AIME24 and AIME25 by 4% to 8% in average accuracy.

User as Engram: Internalizing Per-User Memory as Local Parametric Edits (1 minute read)

This paper proposes User as Engram, which stores a user's facts as surgical edits to an Engram model's hash-keyed memory table while carrying reasoning skill in one shared adapter. The layered design matches per-user LoRA on direct recall while delivering 5.6x higher indirect-reasoning accuracy and a roughly 33,000x smaller memory footprint, since disjoint hash slots let many users' edits compose losslessly in one table.

Detecting Hidden ML Training With Zero-Overhead Telemetry (1 minute read)

This paper tests whether GPU compute-governance monitoring can survive adversarial evasion, using only zero-overhead NVML telemetry that observes physical effects of computation without touching weights, training data, or hyperparameters. Across 5 rounds of monitor-evader iteration covering 20 evasion strategy families and 9 GPU models, the authors' classifier hit 98.2% binary accuracy at flagging training workloads and 43-87% against the hardest adversarially disguised cases.

Sumi: Open Uniform Diffusion Language Model from Scratch (1 minute read)

Sumi is a fully open 7B uniform diffusion language model pretrained from scratch on 1.5T tokens, the first UDLM trained at this combined scale. It performs competitively with autoregressive models at comparable token budgets on knowledge, reasoning, and coding benchmarks while underperforming on commonsense tasks, and the authors released model weights, checkpoints, and the full training recipe including the data mixture.

PhantomSkill: Malicious Code Injection in Agent Skill Ecosystems (1 minute read)

PhantomSkill is an attack framework that hides malicious behavior in an agent skill's auxiliary resources rather than its textual description, exposing a new supply-chain attack surface for LLM coding agents. Its core technique, VulMask, rewrites overt malicious scripts into vulnerability-shaped code that activates only under attacker-controlled triggers, evading automated reviewers while preserving benign utility.

CodeSentinel: A Three-Layer Defense Against Indirect Prompt Injection in Code Contexts (1 minute read)

CodeSentinel is an inference-time sanitizer that defends code LLMs against indirect prompt injection hidden in comments, strings, identifiers, or decoy code retrieved from repositories and agent environments. It uses Tree-sitter to extract high-risk CST nodes, then applies syntax-guided pre-filtering, CST-guided Dynamic Min-K% scoring, and node perturbation analysis, hitting 0.80 average node-level F1 across six attack families and beating CodeGarrison, DePA, and KillBadCode.

Rethinking Reward Supervision: Rubric-Conditioned Self-Distillation (1 minute read)

This paper introduces Rubric-Conditioned Self-Distillation, a post-training method that conditions a teacher model on criterion-level rubrics to give token-level guidance on a student's own sampled reasoning trajectories. The two-stage pipeline first learns to generate task-specific rubrics, then trains a rubric-guided reasoner, beating GRPO by 1.0 points and OPSD by 0.9 points on average across science reasoning benchmarks.
