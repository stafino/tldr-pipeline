Agent Beacon: Open-source telemetry layer for AI agents (1 minute read)

Asymptote Labs released Agent Beacon, an open-source telemetry layer that configures runtimes like Claude Code, Codex CLI, and Cursor to export OpenTelemetry data and writes a normalized JSON log of agent file edits, commands, and tool calls across local, CI, and cloud surfaces. Written in Go under the MIT license, it runs locally without an account, binds its dashboard to loopback, and supports configurable retention modes plus forwarding to customer-managed SIEMs.
