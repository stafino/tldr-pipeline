Agentic coding and persistent returns to expertise (5 minute read)

Anthropic analyzed roughly 400,000 Claude Code sessions from October 2025 to April 2026 to study how people direct agentic coding. Users typically make the planning decisions while Claude handles execution, and the more domain expertise someone brings, the more work Claude does per instruction and the more often the session succeeds. Over the seven months, time spent debugging fell by nearly half, usage shifted toward end-to-end agentic work, and the estimated value of a typical task rose about 25%.

The 100k whys of AI (1 minute read)

The author argues that AI-generated text is distinguishable from human writing not because LLMs have alien mannerisms, but because they fall back on the same complex set of mannerisms in response to almost any prompt. The evidence is a collage of roughly 150 near-identical Amazon book covers surfaced by searching "100000 whys," many featuring the same roaring dinosaur, rocket, or golden retriever. Because the tools are quasi-deterministic, a hundred authors giving similar prompts get functionally identical output around 80% of the time.

LLMs Are Complicated Now (1 minute read)

The author argues that LLM architectures have grown as messy as the recommendation systems they once contrasted with, citing the jump from Llama 3 to Nemotron 3 Ultra. Attention now comes in grouped, compressed, sparse, linear, and sliding-window variants, Mixture-of-Experts routing has spread well beyond feed-forward layers, and inference spans multiple GPUs. The takeaway: performance has shifted from optimization to necessity, so research iteration needs fused, fast baselines before a new variant can even be evaluated.

Why the Human Genome's Tangled Physicality May Confound AI (5 minute read)

The author argues that the human genome resists the kind of understanding biologists once expected, since barely 2% of its 3 billion bases code for genes and the real difficulty lies in how those genes are regulated. Genomic foundation models like Evo 2, Genos, and DeepMind's AlphaGenome sidestep that complexity by learning correlations from known cases rather than mechanisms. MolmoMotion: Language-guided 3D motion forecasting (5 minute read)

MolmoMotion is a language-guided model from Allen AI that predicts the future 3D trajectories of object points given a video frame, query points, and a written action description, outperforming existing motion forecasting methods. Alongside the model weights, the team released MolmoMotion-1M, a dataset of 3D point trajectories paired with action descriptions drawn from 1.16M videos, and PointMotionBench, a human-validated benchmark of 2.7K clips.

MosaicLeaks: Can your research agent keep a secret? (5 minute read)

MosaicLeaks is a benchmark that tests whether deep-research agents leak private enterprise data through their external web queries, where an adversary watching only the cumulative query log can reassemble fragments into sensitive facts. The authors found that agents frequently leaked private information, and their privacy-aware RL method, PA-DR, cut answer leakage from 34.0% to 9.9% while raising strict chain success from 48.7% to 58.7%.

Amazon S3 annotations: attach rich, queryable context directly to your objects (1 minute read)

Amazon S3 introduced annotations, a metadata capability that lets you attach up to 1,000 named annotations per object, each up to 1 MB and totaling 1 GB, in JSON, XML, YAML, or plain text. Annotations can be modified or deleted without rewriting objects, travel with objects during copy and replication, and flow into managed tables that are queryable with Amazon Athena when S3 Metadata is enabled.

Intel and AMD's new ACE CPU extensions bring an efficient AI-oriented instruction set to x86 — a new design makes matrix multiplication more power- and density-efficient (1 minute read)

Intel and AMD released the full specification for ACE, an x86 instruction set extension that adds dedicated matrix-multiplication silicon on top of existing AVX10 registers. For the same number of input vectors, ACE performs 16x as many operations as AVX10, reducing instruction overhead and letting ML frameworks like PyTorch and TensorFlow target one implementation-agnostic code path.

Enterprise-Managed Authorization: Zero-touch OAuth for MCP (1 minute read)

The Enterprise-Managed Authorization (EMA) extension for the Model Context Protocol is now stable, letting organizations manage MCP server access centrally through their existing identity provider. Instead of per-app OAuth, the client obtains an Identity Assertion JWT Authorization Grant during single sign-on and exchanges it for an access token, giving users zero-touch setup and security teams central policy and audit control. Anthropic, Microsoft, and Okta are among early adopters.

New framework for auditing machine unlearning (5 minute read)

Google Research introduced Regularized f-Divergence Kernel Tests, a framework presented at AISTATS 2026 for auditing machine unlearning by querying a model and analyzing its output samples. The tests provably control false positives at any sample size and drive false negatives toward zero as samples grow, addressing the loss of statistical power that makes standard two-sample testing computationally expensive on large models.
