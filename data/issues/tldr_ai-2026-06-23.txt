The text in Claude Code’s “Extended Thinking” output (1 minute read)

The author found that Claude Code's on-disk "thinking blocks" contain only a 600-character encrypted signature, not readable reasoning. Per Anthropic's docs, extended thinking is encrypted with a key Anthropic holds, and the API returns only a summary of the model's reasoning rather than the reasoning itself, with full access gated behind an enterprise agreement. The author warns that this breaks any promise of a genuine audit trail, since the logic that actually drove an agent's actions can't be reconstructed from local files.

Agentic coding and persistent returns to expertise (5 minute read)

Anthropic analyzed roughly 400,000 Claude Code sessions from October 2025 to April 2026 to study how people direct agentic coding. Users typically make the planning decisions while Claude handles execution, and the more domain expertise someone brings, the more work Claude does per instruction and the more often the session succeeds. Over the seven months, time spent debugging fell by nearly half, usage shifted toward end-to-end agentic work, and the estimated value of a typical task rose about 25%.

Prompt Injection as Role Confusion (1 minute read)

This post argues that prompt injection stems from a fundamental flaw in how LLMs perceive roles, since everything a model receives, including system prompts, user messages, tool outputs, and its own reasoning, arrives as one continuous token stream partitioned only by role tags. From this framing the authors construct new attacks, explain existing mechanistic interpretability results, and predict when injections succeed, then sketch a research agenda for a science of roles.

The 100k whys of AI (1 minute read)

The author argues that AI-generated text is distinguishable from human writing not because LLMs have alien mannerisms, but because they fall back on the same complex set of mannerisms in response to almost any prompt. The evidence is a collage of roughly 150 near-identical Amazon book covers surfaced by searching "100000 whys," many featuring the same roaring dinosaur, rocket, or golden retriever. Because the tools are quasi-deterministic, a hundred authors giving similar prompts get functionally identical output around 80% of the time.

Using Reddit to manipulate AI search results is surprisingly easy (1 minute read)

A Cornell Tech study found that a short Reddit comment, sometimes as few as 13 words, can poison the outputs of deep-research agents that gather sources and generate cited reports. The attack, dubbed Web Agent Retrieval Poisoning (WARP), targets threads and forum pages that recur across related searches, then plants text that blends into the discussion. It requires no access to the model, its prompts, or its retrieval infrastructure, and user-generated sources made up 16.7% to 23.4% of retrieved URLs.
