Popping the GPU Bubble (1 minute read)

Photon hides GPU bubbles using pipelined decoding, which overlaps GPU and CPU work instead of running them as a sequential baton pass. In standard autoregressive decoding, the GPU finishes one token's math, then sits idle while the CPU handles fixed-cost housekeeping like selecting requests, setting up metadata, and committing the chosen token. Photon launches the next forward pass while the current token is still being committed, keeping the GPU busy across the loop.

EvalSafetyGap: A Hybrid Survey and Conceptual Framework for LLM Evaluation-Safety Failures (1 minute read)

This paper argues that LLM evaluation and AI safety share a measurement problem: benchmark, reward-model, and safety scores can climb while the latent properties they track stay unverified. It introduces EvalSafetyGap, a framework built on Goodhart's Law plus an Instability Decomposition and Alignment Trilemma, and runs a ten-model audit measuring capability, behavioral safety, and governance separately. The audit finds capability and adversarial robustness statistically unrelated (Pearson r = +0.232, p = 0.520), with the open-closed safety gap driven mainly by disclosure rather than behavioral robustness.

Pessimism's Paradox: Conservative Offline Training Amplifies Reward Hacking During Online Adaptation in Reasoning Models (1 minute read)

This paper challenges the assumption that conservative offline training makes online adaptation safer, showing that more conservatism monotonically worsens reward hacking. Training a Qwen3-14B policy under DPO at increasing conservatism levels, the authors find that higher conservatism compresses policy entropy, reduces response diversity, and raises ensemble disagreement that online optimization exploits faster, arguing the field needs calibrated rather than maximal conservatism.

VLK: Learning Humanoid Loco-Manipulation from Synthetic Interactions in Reconstructed Scenes (1 minute read)

VLK is a pipeline that generates synthetic vision-language-kinematics supervision for humanoid robots by using 3D Gaussian Splatting to reconstruct metric-scale indoor scenes, then synthesizing navigation and object-interaction trajectories with privileged scene data and rendering paired egocentric views. The authors produced 48,000 paired trajectories with no human intervention and trained a policy that transfers to a physical Unitree G1 for navigation and single-object transport.

Forensic Trajectory Signatures for Agent Memory Poisoning Detection (1 minute read)

This paper identifies a behavioral invariant in LLM agents under memory poisoning: successful exfiltration attacks call memory_recall_fact before email_send_email, a transition benign sessions rarely show. A Random Forest classifier over 19 trajectory features detects the attack at AUC 0.9904, generalizes to frontier models like GPT-4.1 without retraining, and distinguishes memory-channel attacks from prompt injection using tool-call logs alone.

DiScoFormer: One transformer for density and score, across distributions (5 minute read)

Allen AI introduced DiScoFormer, a transformer that estimates both the density and the score of a distribution from a set of data points in a single forward pass without retraining. The model uses cross-attention to evaluate density and score at any point, with a shared backbone and two output heads that exploit the relationship between score and the gradient of log-density to stay accurate as dimensionality grows.

Entity Binding Failures in Tool-Augmented Agents (1 minute read)

This paper identifies entity binding failures, where a tool-augmented agent picks the correct tool but acts on the wrong real-world entity, such as emailing the wrong Alex or updating the wrong account. Across 60 tasks, five backends, and six tool-use methods, baselines hit 0% wrong-tool error but 24-26% wrong-entity actions, which entity-aware mechanisms eliminated at the cost of lower task completion.

MuonSSM: Orthogonalizing State Space Models for Sequence Modeling (1 minute read)

MuonSSM is a framework that stabilizes state space model training by conditioning the geometry of memory updates rather than the recurrent transition matrix, using a momentum-based pathway and a lightweight Newton-Schulz transformation on low-rank input injections. The authors report bounded, spectrally conditioned updates that preserve parallel scan complexity and deliver consistent accuracy and long-context gains across language, vision, and time-series benchmarks.

MCP Server Architecture Patterns for LLM-Integrated Applications (1 minute read)

This paper catalogues five recurring MCP server architecture patterns (Resource Gateway, Tool Orchestrator, Stateful Session Server, Proxy Aggregator, and Domain-Specific Adapter) observed across fifteen production and public servers, plus four anti-patterns and cross-cutting concerns around auth, versioning, and observability. A tool-count study found selection accuracy drops below 90% at 10-15 tools per context for Claude Haiku 4.5 and 20-30 tools for Sonnet 4.

GROW$^2$: Grounding Which and Where for Robot Tool Use (1 minute read)

GROW² is an open-world affordance grounding method that lets robots select an unconventional object as a tool and localize its specific region of action, such as using a plate to cut a cake when no knife is available. It splits grounding into a semantic level, where a VLM parses the instruction and picks task-relevant parts, and a geometric level, where vision foundation models ground those parts into 3D regions from a single RGB-D image.

Words Speak Louder Than Code: Investigating Cognitive Heuristics in LLM-Based Code Vulnerability Detection (1 minute read)

This paper presents the first systematic study of cognitive heuristics in LLM-based code vulnerability detection, holding the code fixed while varying surrounding context to trigger halo, framing, and anchoring effects across eight models and three languages. All models proved susceptible, with framing biasing verdicts 33.2% of the time on average, and a proof-of-concept black-box attack suppressed up to 97% of previously detected vulnerabilities.

TraceLab: Characterizing Coding Agent Workloads for LLM Serving (1 minute read)

TraceLab releases a trace of roughly 4,300 coding-agent sessions spanning about 350,000 LLM steps and 430,000 tool calls, collected from day-to-day use of Claude Code and Codex. The analysis finds long autonomous loops, long contexts with short outputs, heavily-tailed tool calls, and high but imperfect prefix cache hit rates, pointing to serving optimizations like append-length-aware prefill and better KV-cache management.

When Is a Draft Accepted? A Theory of Acceptance in Speculative Decoding (1 minute read)

This paper develops a theory of acceptance for speculative decoding in the deterministic regimes that real systems actually use, including greedy decoding, relaxed acceptance rules, and tree-based candidate sets. The authors derive exact KL-divergence certificates and margin-based bounds for when a drafter's tokens are accepted, and show on Qwen3 models that relaxed and tree-based criteria substantially enlarge the certified acceptance region, especially on low-margin decoding steps.

The attack that hijacked Claude Code came through Sentry. Datadog, PagerDuty, and Jira have the same exposure. (3 minute read)

Tenet Security's June "agentjacking" disclosure showed that a single crafted Sentry error event, sent through a public DSN credential that requires no breach or authentication, can inject attacker instructions that Claude Code, Cursor, and Codex execute as trusted diagnostic output with the developer's full privileges. Tenet hit an 85% success rate across 100-plus controlled targets and found 2,388 organizations with exposed Sentry credentials, exposing the same blind spot in any agent connected to Datadog, PagerDuty, or Jira.

Researchers Demo New Claude Code Attack Using Harmless-Looking Repositories to Hijack Developer Machines (1 minute read)

Mozilla's 0Din researchers demonstrated an attack that hijacks developer machines through normal-looking repositories that contain no malicious code, relying instead on Claude Code following a setup error message that triggers a hidden reverse shell. The payload lives in a base64-encoded DNS TXT record rather than the repo, splitting the attack across the repository, DNS infrastructure, and the agent's trust so no component looks malicious in isolation.

New BioShocking Attack Tricks AI Browsers Into Leaking User Credentials (1 minute read)

Security firm LayerX disclosed BioShocking, an indirect prompt injection attack that tricked six AI browsers and assistants, including OpenAI's ChatGPT Atlas, Perplexity's Comet, and Anthropic's Claude extension, into copying a user's credentials and sending them to an attacker. The attack disguises malicious commands as game rules a puzzle page, and vendor responses were uneven, with OpenAI patching Atlas while Perplexity closed the report without acting.
