The Log is the Agent (1 minute read)

ActiveGraph is a runtime that inverts the usual agent stack by making an append-only event log the source of truth, with the working graph as a deterministic projection and behaviors reacting to graph changes and emitting new events. The design enables deterministic replay of any run from its log, cheap forking that branches at any event without re-executing the shared prefix, and end-to-end lineage from a goal down to the individual model call behind each artifact.
