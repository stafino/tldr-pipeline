AI’s energy tax was already concerning. Research says AI agents are over hundred times worse (1 minute read)

A KAIST study presents the first comprehensive analysis of the energy cost of AI agents, finding they can consume up to 136.5 times more energy per query than conventional generative models. The overhead comes from agents repeatedly calling LLMs, browsing, and running tools, which leaves GPUs idle and drawing power up to 54.5% of execution time while waiting on external tasks. At Google-scale query volume, the authors estimate agent infrastructure would need roughly 198.9 gigawatts.

Does code cleanliness affect coding agents? A controlled minimal-pair study (1 minute read)

This paper introduces a minimal-pair evaluation protocol to test whether code cleanliness affects coding agents, using repository pairs matched on architecture, dependencies, and behavior but differing on static-analysis violations and cognitive complexity. Across 660 trials with Claude Code, cleanliness did not change pass rates, but agents working on cleaner code used 7 to 8% fewer tokens and reduced file revisitations by 34%.

The Log is the Agent (1 minute read)

ActiveGraph is a runtime that inverts the usual agent stack by making an append-only event log the source of truth, with the working graph as a deterministic projection and behaviors reacting to graph changes and emitting new events. The design enables deterministic replay of any run from its log, cheap forking that branches at any event without re-executing the shared prefix, and end-to-end lineage from a goal down to the individual model call behind each artifact.
