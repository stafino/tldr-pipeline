DeepSeek cut prices 75%. The 100x problem remains (3 minute read)

DeepSeek cut prices 75% on its V4-Pro model, but cheaper inference isn't fixing enterprise margins. The reason is token amplification: a chatbot turns one user question into roughly one model call, while an agent turns it into a loop of planning, retrieval, tool use, and verification, pushing the input-to-billed ratio from about 1:5 to 1:700 or higher. Falling prices don't repair a product architecture that turns one prompt into dozens of billable operations.

Nvidia, CoreWeave, and Nebius: Inside the Circular Financing of the GPU Boom (1 minute read)

Neoclouds like CoreWeave and Nebius have ridden hyperscaler demand to soaring revenue, backlog, and share prices, but this analysis argues the buildout rests on shaky financing. Both have contracted 3.5 GW of power while running unprofitable, cash-thin operations funded by GPU-backed debt. The bigger risk is circular financing, where Nvidia's own investments and backstopping prop up the customers buying its chips.

Claude Code sends 33k tokens before reading the prompt; OpenCode sends 7k (1 minute read)

Claude Code sends roughly 33,000 tokens of system prompt, tool schemas, and injected scaffolding before a user's prompt arrives, while OpenCode sends about 7,000 on the same model and machine. A 72KB CLAUDE.md adds another 20,000 tokens per request and Claude Code rewrites prompt-cache tokens mid-session, writing up to 54x more cache tokens than OpenCode on the same task, though its tool-call batching can still win a multi-step session outright.

Mesh LLM: distributed AI computing on iroh (1 minute read)

Mesh LLM pools GPUs and memory across the machines you already have and exposes them as a single OpenAI-compatible API, built on iroh endpoints. It serves each request one of three ways: locally, routed to a peer that has the model loaded, or split across several machines for models too large for one box, and ships a catalog of 40+ models ranging from sub-billion-parameter models to 235B mixture-of-experts giants.
