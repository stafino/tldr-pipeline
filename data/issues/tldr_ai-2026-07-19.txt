Transcribe.cpp (1 minute read)

transcribe.cpp is a ggml-based speech-to-text library from Handy's maintainer that supports 16 ASR families and over 60 models, with GPU acceleration via Vulkan, Metal, CUDA, and TinyBLAS. Every supported model has been numerically validated and WER tested against its reference implementation, and the v0.1.0 release works as a near drop-in whisper.cpp replacement with streaming, batch transcription, and maintainer-supported bindings.

VulnHunter: Capital One's agentic AI code security tool (1 minute read)

Capital One open-sourced VulnHunter, an agentic AI tool that analyzes source code from an attacker's perspective to find exploitable defects. Rather than scanning for pattern matches, it runs an agentic reasoning workflow that maps prospective attack paths and proposes targeted code remediations, with the stated design goal of shifting developer effort away from triaging findings.
