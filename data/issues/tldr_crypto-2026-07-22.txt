TrapHunter: Exposing Covert Pathways in Trap Token Contracts (1 minute read)

This paper introduces TrapHunter, a framework for detecting ERC-20 trap tokens that follow the standard exactly while embedding covert logic to defraud holders. The authors classify traps across the token lifecycle (generation, circulation, persistence, observation), then combine abstract behavior trees and augmented path graphs with LLM reasoning about intent deviation from reference implementations, confirming exploitability through fork-based dynamic validation. On 269 real-world contracts, it hit 81.8% average precision and 85.4% recall across three LLM backends.

Tracing the Shadows: Automatic Tracking and Analysis of Crypto Money Laundering via Transaction Semantic Analysis (1 minute read)

This paper introduces AMLGuard, an anti-money-laundering framework that traces stolen funds from known malicious addresses by inferring the intent behind DeFi transactions rather than following raw token transfers. It combines rule-based static analysis with retrieval-augmented LLM reasoning to abstract each transaction into a DeFi Semantic Unit, and parses call arguments to recover cross-chain bridge semantics. Across 82 real laundering cases covering over $1B in illicit assets, it reports 98.4% address recall on single-chain flows and 95.8% cross-chain.
