Announcing Apps on Databricks Marketplace (1 minute read)

Databricks launched Apps on Databricks Marketplace in Public Preview, letting customers discover, install, and run third-party data and AI applications natively inside their Unity Catalog-governed workspaces with no data egress. Instead of moving data to an external tool, the app deploys to the data, and providers get a no-egress distribution channel to monetize proprietary apps across thousands of Databricks enterprises.

Introducing OpenSharing SecureConnect (1 minute read)

Databricks released OpenSharing SecureConnect, a managed proxy that routes storage access for Delta Sharing recipients so providers allowlist Databricks endpoints once instead of configuring firewall rules per recipient. Data stays in the provider's own bucket with live sharing from the source, and providers can optionally use Network Connectivity Configuration for private link connectivity between the proxy and their storage.

AI governance at Data + AI Summit 2026: What’s new with Unity AI Gateway (1 minute read)

Databricks expanded Unity AI Gateway at Data + AI Summit 2026, extending Unity Catalog governance to models, agents, MCP services, and skills with runtime guardrails, unified tracing, and cross-provider cost controls including spend visibility, hard spend caps, and intelligent routing. The pitch is one governance layer for a multi-model, multi-agent estate, where teams cap token spend, attribute costs per agent, and monitor coding-agent activity without stitching together separate tools per provider.

Milvus invents Vector Lakebase (5 minute read)

Zilliz launched Vector Lakebase, extending Milvus from real-time vector search into a unified S3-based foundation that runs production queries, discovery sessions, and multi-petabyte training pipelines against the same vectors without copies or migration. It stores data on Vortex, an open-source columnar format that claims roughly 100x faster random access and 10-20x faster scans than Parquet, with tiered serving for production and on-demand compute for batch analytics.
