Accelerating Presto with GPUs (1 minute read)

Meta and Presto's maintainers extended Presto to run query operators on GPUs, using NVIDIA's cuDF library to keep data in GPU memory across distributed exchanges instead of round-tripping through the CPU. On TPC-H-derived benchmarks across a multi-GPU cluster, the team reports up to 6x cost/performance gains over CPU Presto. The code is upstream in open-source Presto/Velox and already running some customer production workloads, making it a real option for analytics teams weighing GPU spend against warehouse compute.

One Index for Subsumption and Roll-up across Time, Geography, and Ontology (1 minute read)

This paper introduces OEH, a single index that handles both subsumption testing ("is x under y?") and hierarchical roll-up across time, geospatial, and ontology hierarchies, which today each live in separate silos. It encodes trees as nested-set order-embeddings and low-width DAGs as chain decompositions, picked by a cheap structural probe. On five real hierarchies it matched a 2-hop index on latency using half the space and building 6-7x faster, while matching TimescaleDB's continuous aggregates and adding roll-up that 2-hop cannot.

Unified Dominance Graph for Interval-Predicate Approximate Nearest Neighbor Search (1 minute read)

This paper introduces the Unified Dominance Graph (UDG), a graph index for approximate nearest neighbor search constrained by interval predicates like containment and overlap. Treating an object's interval endpoints as two independent scalar filters bloats intersection overhead, so UDG maps both endpoints into a normalized 2D dominance space and reuses one construction across predicates. On standard benchmarks it beats hybrid search baselines while keeping indexing overhead low.

Entity Resolution via Batched Oracle Queries (1 minute read)

This paper formalizes batched entity resolution, where an oracle clusters only a limited batch of records at a time and no batch is guaranteed to hold all of an entity's records. The authors prove that selecting optimal batches is NP-hard, give an optimal solution under a natural constraint on entity sizes, and frame the work as pay-as-you-go, trading oracle consults for recall at every step. Across six datasets, their method beats state-of-the-art baselines on recall per query budget.

Grading the Grader: Lessons from Evaluating an Agentic Data Analysis System (1 minute read)

This paper tackles how to reliably grade agentic data analysis systems, whose outputs mix code, numbers, and verbal diagnostics that single-turn eval methods can't handle. Running LAMBDA on 153 numerical QRData tasks, the authors built a three-layer cascade of strict regex matching, lenient LLM grading, and human snippet inspection, both automated graders hitting 100% precision with 97% lenient recall. An iterative nudge mechanism that cues the answer template lifted grading run success from 36% to 97%.

FlowPipe: LLM-Enhanced Conditional Generative Flow Networks for Data Preparation Pipeline Construction (1 minute read)

FlowPipe reframes automated data-prep pipeline construction as conditional probabilistic flow generation over a DAG, using Conditional GFlowNets with a Trajectory Balance objective to tie terminal validation rewards back to early operator choices. LLM-derived priors condition the policy through Feature-wise Linear Modulation so dataset semantics steer the search, while failure awareness steers it away from invalid states. Across 74 real-world datasets it beats Multi-DQN baselines by 11.96% on accuracy and converges 12.5x faster.
