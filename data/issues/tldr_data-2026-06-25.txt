Optimizing cloud economics with linear elastic caching (1 minute read)

Google researchers introduced linear elastic caching, a CIDR paper that treats cache memory as a variable cost integrated over time rather than a fixed pre-allocated pool, framing each eviction as a ski-rental problem solved with lightweight ML. Instead of sizing a cache for peak demand and paying for idle RAM, the system dynamically resizes against real-time workloads to minimize total cost. The pitch matters where serverless memory runs up to $3 per GiB per day.
