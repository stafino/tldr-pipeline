WAL-RUS: a Rust Rewrite of WAL-G for PostgreSQL Backups (1 minute read)

ClickHouse released WAL-RUS, an open-source Rust rewrite of WAL-G that handles Postgres backups and WAL archival while staying compatible with WAL-G's format. The goal is a predictable memory profile: Go's garbage-collected runtime produces a sawtooth memory pattern that forces operators to over-provision, leaving less headroom for Postgres queries, shared buffers, and page cache in resource-constrained environments.
