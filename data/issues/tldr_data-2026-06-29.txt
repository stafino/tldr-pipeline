WAL-RUS: a Rust Rewrite of WAL-G for PostgreSQL Backups (1 minute read)

ClickHouse released WAL-RUS, an open-source Rust reimplementation of WAL-G that handles Postgres WAL archival and backups while staying wire-compatible with existing WAL-G storage. The motivation is memory predictability: Go's garbage-collected runtime produces a sawtooth memory profile that forces operators to over-reserve RAM, and Rust's fixed footprint frees that budget for Postgres queries, shared buffers, and page cache.
