Zilliz has a Loonatic storage engine (5 minute read)

Zilliz released Loon, a storage engine for Zilliz Cloud that serves real-time vector search, large-scale discovery, and batch analytics from a single copy of data on object storage, using Vortex for vectors and Parquet for scalar fields under a shared versioned manifest. Zilliz's internal tests show Loon's layout pulls roughly 135x less data per record read than Parquet, and adding a new embedding model becomes a column-level version update rather than a multi-hundred-gigabyte rewrite.

MongoDB embeds reranking into Atlas as enterprises look to simplify AI stacks for scale (2 minute read)

MongoDB added Native Reranking to Atlas, a Voyage AI-powered feature now in public preview that runs inside the aggregation pipeline and improves retrieval quality by up to 30%. Folding reranking into the database removes a separate vendor, API, and orchestration layer from RAG stacks, cutting the retry logic and version juggling that a standalone reranking service forces on teams as apps scale.
