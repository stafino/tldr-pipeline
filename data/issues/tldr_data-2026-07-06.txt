Google Cloud proposes Open Knowledge Format for AI-ready data sharing (5 minute read)

Google Cloud proposed the Open Knowledge Format, a vendor-neutral spec that represents organizational context as a directory of markdown files with YAML frontmatter for fields like type, title, and tags. OKF v0.1 formalizes the LLM-wiki pattern so metadata, runbooks, and metric definitions written by one producer can be consumed by different agents without translation, no new runtime or SDK required.
