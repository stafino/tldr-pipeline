Terascale Query Processing in the Browser: Rethinking GPU Acceleration (1 minute read)

This paper introduces WGLog, a WebGPU-based engine that runs compute-bound recursive queries entirely inside the browser. The authors replace hash-table joins with atomic-free sorted-array joins to avoid the serialization hit on skewed graphs, and use WebGPU indirect dispatch to drop per-iteration GPU-host syncs. Reported results are a 1.48x to 4.68x speedup over native GPU systems and orders of magnitude over CPU and WebAssembly.

Extending Elle for Transaction Workloads with Duplicate Values (1 minute read)

Elle, the black-box isolation checker behind most Jepsen-style database tests, only works soundly when every written value is unique, which rules out the duplicate-value workloads where many isolation bugs actually surface. This paper extends it with a finer-grained dependency model that reasons about individual operations rather than whole values, with soundness and completeness proofs and a working prototype. For teams validating transactional guarantees in Postgres, CockroachDB, or a homegrown OLTP layer, it closes a gap between synthetic test traffic and production write patterns.

From Blind Search to Memory-Aware Evolution: Efficient DBMS Tuning via Collaborative Diagnosis and Utility-Aware Retrieval (1 minute read)

This paper introduces EvoTune, a framework that tunes DBMS knobs, query hints, and indexes together by narrowing to a query-specific high-impact subspace instead of searching the full combinatorial space. It pairs lightweight pattern learning with LLM reasoning and retrieves past tuning observations by their long-term performance payoff rather than similarity, with feedback stored in a hierarchical memory that needs no fine-tuning. The authors report up to 44.5% better performance at the same tuning budget and 3.9x faster convergence to baseline-best results.

Panache: One-Pass Motif Discovery at Every Window Length (1 minute read)

This paper introduces Panache, a one-pass streaming algorithm for z-normalized pan matrix profile motif discovery that replaces one quadratic self-join per window length with a single near-linear scan. It maintains each subsequence's Fourier spectrum online, hashes on that state, and uses a Parseval bound to reject most candidate pairs before exact computation. On a 5M-sample series across 51 window lengths, it emits exact motifs in 6 minutes against 7.95 hours for the fastest exact CPU baseline.

The Database Is Not the Data Model (5 minute read)

The author argues that schema design and data modeling are different disciplines, and that the industry collapsed them because DDL was visible, executable, and committable while conceptual models were not. The piece traces the drift from Codd's data independence and Chen's ER modeling through ORMs shipping `models.py` and dbt calling a SELECT statement a model. The result is that "model" now means an ERD, a table definition, a SQL transform, and a transformer, which leaves teams arguing about implementation before they have agreed on what the entities mean.

SAGA: Synthetic Agentic Graph Architecture for Temporal Benchmark Generation (1 minute read)

This paper introduces SAGA, a pipeline that generates synthetic temporal graphs by separating structure from semantics: an O(1)-per-edge generator builds power-law skeletons, then LLM agents inject domain rules via RAG across finance/AML, network IDS, cyber, and transportation. Anomaly labels fall out of the state alignment step rather than needing manual annotation. On one H100 with vLLM batching, SAGA produces 500K temporal edges across 100K nodes in under 90 minutes, which makes GNN benchmark data cheap enough to generate instead of license.

AlloyDB added columnar engine acceleration for pgvector's HNSW index in preview, delivering up to 4x higher QPS than standard PostgreSQL HNSW at comparable recall.
