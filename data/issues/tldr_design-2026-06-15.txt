The experience of rendering Arabic typography and its technical debt (1 minute read)

Arabic typography requires cursive joining, contextual letterforms, and full bidirectional layout, demands that most rendering stacks bolt on after the fact rather than design around. The author walks through where that technical debt surfaces, from broken letter-shaping in common UI frameworks to the gap between Unicode's logical order and the visual order text must actually render in.

What’s !important #13: @function, alpha(), CSS Wordle, and More (1 minute read)

The latest issue of CSS-Tricks' What's !important roundup covers the new `@function` rule for authoring custom CSS functions, the `alpha()` function for adjusting opacity on existing colors, and Grid Lanes for more flexible track placement. It also surfaces lesser-known `dialog` element behaviors and a CSS-only Wordle build worth studying for layout tricks.
