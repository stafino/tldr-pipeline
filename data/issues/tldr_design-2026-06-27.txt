At Config 2026, Figma is expanding its canvas into code layers, motion, shaders, and agent-driven workflows, a direct response to AI pulling product work into code editors and structured pipelines. "A James Bond opening is the one thing that doesn't need to follow trends": how 007 First Light blurred the line between film and gaming (1 minute read)

Territory Studio's Andrew Popplestone breaks down how the team built the 007 First Light opening title sequence, applying its cross-disciplinary film and games experience to push game-ready assets up to film-production quality with finer detail and definition. The interview is a useful look at how title-design craft translates across mediums, and why a Bond opening earns its refusal to chase current motion trends.

How to Detect Which Font Is Actually Rendering in a Browser (Not Just the CSS Stack) (2 minute read)

A developer explains why getComputedStyle returns the CSS font stack, not the font that actually rendered, and walks through canvas-based detection that measures glyph widths to determine which font in the cascade the browser selected. The distinction is mostly moot for Latin text but critical for Japanese, where Hiragino, Yu Gothic, and Noto Sans JP differ enough that a site designed on macOS looks meaningfully different on Windows.

Launches & Tools

Krea 2 Technical Report (5 minute read)

Krea released Krea 2, a series of foundation image-generation models built for creative exploration rather than the single polished default most diffusion tools converge toward. The system pairs a from-scratch data and distributed-training stack with a custom diffusion transformer using grouped-query attention, sigmoid-gated attention, and Qwen3-VL text encoding, trained through pretraining, SFT, preference optimization, and RL to span a wider range of styles and moods.

Libre Barcode Project (1 minute read)

Libre Barcode is an open-source font collection that renders scannable barcodes in formats like Code 128, Code 39, and EAN-13, with optional human-readable text below each code. Because the barcodes are fonts, designers can drop them into any layout tool and generate valid codes from plain text without a separate encoder, image asset, or generator service.
