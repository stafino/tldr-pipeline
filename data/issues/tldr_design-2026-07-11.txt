shadcn/typeset vs Tailwind Typography: Styling Markdown (3 minute read)

shadcn released shadcn/typeset on July 10, 2026, a single CSS file you copy into your project to style rendered markdown and HTML wrapped in a `.typeset` container: headings, lists, tables, code, blockquotes, even MathML and `<details>`. Unlike Tailwind Typography's plugin or shadcn/ui's authored components, there's no npm package or CLI install, so you own the file outright and layer presets like `.typeset-docs` to tune each context.
