Linux 7.1 (1 minute read)

Linus Torvalds released Linux 7.1, the latest mainline kernel, following the standard two-week stabilization period after the final release candidate. As with past major-number bumps, the jump carries no special significance beyond keeping minor version numbers from growing unwieldy, with the bulk of the changes landing as routine driver updates, filesystem fixes, and incremental subsystem improvements.

Phoenix LiveView 1.2 (1 minute read)

Phoenix LiveView 1.2 introduces colocated hooks and colocated JS, which let developers write client-side JavaScript directly inside HEEx templates instead of maintaining separate hook files, with the JS extracted at compile time and bundled by the asset pipeline. The release also adds a new portal API for rendering content outside the current DOM tree, useful for modals and tooltips, and ships compatibility fixes for the latest Phoenix and Elixir versions.

Arch Linux Now Believes Malware Incident Under Control: More Than 1,500 Packages (1 minute read)

Arch Linux says its AUR malware incident is now contained after more than 1,500 packages were found carrying malicious code injected through compromised maintainer accounts. The affected packages have been flagged and removed, and the team is urging users who installed anything from the AUR during the window to audit their systems for the injected payloads.

Arch Linux locks down AUR signups amid wave of malicious commits (1 minute read)

Arch Linux froze new account signups on the Arch User Repository after attackers flooded it with malicious commits that poisoned package updates. The community-maintained AUR ships build scripts rather than vetted binaries, so the lockdown buys maintainers time to clean compromised packages and review the backlog before reopening registration to contributors.
