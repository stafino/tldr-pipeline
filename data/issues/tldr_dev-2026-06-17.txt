Making ast.walk 220x Faster (1 minute read)

Reflex rewrote Python's `ast.walk` for their code-generation linter, replacing the generator-based traversal with an inlined iterative loop that appends to a list instead of yielding nodes. Eliminating the `yield` suspensions and inlining `iter_child_nodes` cut per-node overhead from ~285 nanoseconds to a fraction of that, yielding a 220x speedup on the hot path that validates AI-generated code.

Apple Foundation Models (1 minute read)

Anthropic released Claude for Foundation Models, a Swift package that conforms Claude to the `LanguageModel` protocol in Apple's Foundation Models framework, so apps drive it through the same `LanguageModelSession` API used for Apple's on-device model. Requests go straight to the Claude API rather than through Apple, and developers swap between on-device and server-side models per session to add larger context, frontier reasoning, or server tools like web search and code execution.

Show HN: cuTile Rust: Safe, data-race-free GPU kernels in Rust (1 minute read)

cuTile Rust is a tile-based system from NVIDIA Labs for writing memory-safe, data-race-free GPU kernels in idiomatic Rust, extending ownership across the launch boundary by partitioning mutable tensors into disjoint tiles and sharing immutable ones. A #[cutile::module] macro captures each kernel's AST and JIT-compiles it through CUDA Tile IR into a cubin, supporting synchronous launches, async pipelines, and CUDA graph replay, with local opt-outs when lower-level control is needed.

RFC 10008: The new HTTP Query Method (1 minute read)

The IETF published RFC 10008, which standardizes the HTTP QUERY method for safe, idempotent requests that carry their input in the request body rather than the URI. QUERY fills the gap between GET and POST, letting large or sensitive query payloads be sent without URI length limits while remaining cacheable and automatically retriable since it makes no partial state changes.
