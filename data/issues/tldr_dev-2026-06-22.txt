The only scalable delete in Postgres is DROP TABLE (3 minute read)

The author argues that large DELETEs in Postgres are "work added, not work done," since MVCC leaves dead tuples behind that don't free disk space, force replication overhead, and burden other writers until autovacuum catches up. The recommendation is to design schemas so bulk cleanup can be expressed as a DROP TABLE or TRUNCATE, which reclaim space immediately instead of leaving the heap and indexes to garbage-collect later.

Memory Safe Inline Assembly (1 minute read)

Fil-C, the memory-safe C dialect, now supports GCC/clang inline assembly syntax in a pre-release build, preserving the exact assembly template you write while guaranteeing memory safety. The post details how mistakes that would silently miscompile in standard C, like omitting a `+` constraint or a `cc` clobber, instead degrade to a panic or an illegal instruction trap at worst.

Deno Desktop (1 minute read)

Deno v2.9.0 introduces `deno desktop`, a command that turns any Deno project from a single TypeScript file to a Next.js, SvelteKit, or Vite SSR app into a self-contained, cross-compilable binary bundling the runtime and a web rendering engine. It defaults to the OS webview for small binaries with optional bundled Chromium, swaps socket IPC for in-process bindings, and ships built-in bsdiff auto-update, though it currently runs only in the canary build.

Skip Recompiling 70+ iOS Packages on Every Build (2 minute read)

Expo SDK 56 ships its iOS modules as precompiled XCFrameworks distributed through npm, so apps link prebuilt binaries instead of recompiling React Native core, Expo modules, and native dependencies from source on every clean build. The change works in existing apps with no migration and advances React Native's broader move off Ruby-based CocoaPods, which goes read-only in December 2026, toward Swift Package Manager.

Show HN: Recall – Local project memory for Claude Code (1 minute read)

Recall is a local memory plugin for Claude Code that logs each session to an append-only `history.md` and condenses it into a compact `context.md` summary using a classical Python summarizer rather than an LLM call. It solves the cold-start problem without spending model tokens or sending transcripts to any external API, so resuming a project loads from a ~1-2K token digest instead of re-explaining everything each session.

Codex logging bug may write TBs to local SSDs (1 minute read)

A GitHub issue reports that OpenAI's Codex CLI continuously writes its SQLite feedback logs to disk at a rate of roughly 640 TB/year, enough to exhaust a consumer SSD's rated write endurance in under a year. The reporter traced about 96% of the writes to verbose TRACE-level logging and mirrored telemetry, including high-frequency inotify and websocket payload entries that filtering could eliminate.

North Korean Hackers Blamed for Mastra NPM Supply Chain Attack (1 minute read)

Microsoft attributed last week's Mastra supply chain attack to North Korean group Sapphire Sleet, which compromised an NPM maintainer account and added a malicious dayjs typosquat to 141 packages totaling roughly 8 million weekly downloads. The obfuscated postinstall dropper ran during npm install on Windows, macOS, and Linux, harvesting system data and targeting 160-plus crypto browser extensions, so any workstation or CI/CD pipeline that installed during the June 17 window is exposed.

Hundreds of AI-powered iOS apps found exposing credentials (1 minute read)

Researchers at Wake Forest University analyzed 444 iOS apps with LLM features and found 282, or 64%, exposed exploitable credentials, including 54 plaintext API keys and 136 leaked authentication tokens. The most popular affected app had over 2.3 million ratings, and 55% of leaks routed traffic through custom developer backends, meaning provider-side fixes alone cannot close the gap.

PixelSmash – Critical FFmpeg Vulnerability Turns Media Files into Weapons (1 minute read)

JFrog Security Research disclosed PixelSmash (CVE-2026-8461), a critical heap out-of-bounds write in FFmpeg's MagicYUV decoder that escalates from a crash to reliable remote code execution via a single crafted 50KB AVI file. Because nearly every media app links libavcodec, the blast radius spans Jellyfin, Nextcloud, mpv, Kodi, OBS, and more, with researchers demonstrating full RCE against Jellyfin through an automatic library scan.
