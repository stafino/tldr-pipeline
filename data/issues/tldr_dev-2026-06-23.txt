The text in Claude Code’s “Extended Thinking” output (1 minute read)

The author argues that Claude Code's local "Extended Thinking" logs are not an authentic record of the model's reasoning, since Anthropic encrypts the actual thinking into a signature it holds the key to and returns only a lossy summary over the API. The practical takeaway for engineers is to stop treating those local files as an audit trail, because reconstructing the real logic that drove an agent's actions requires an enterprise agreement and is impossible from your machine alone.

The Coming Loop (1 minute read)

The author argues that agentic engineering is shifting from the in-agent tool loop to a harness-level loop, where a queue feeds tasks to a machine and an outer harness decides whether the work is actually finished or needs another session. He resists adopting it for code he cares about, citing models that write overly defensive, locally-reasoned code and his own unwillingness to ship systems he cannot fully explain.

Prompt Injection as Role Confusion (1 minute read)

The author argues that prompt injection is fundamentally a role-confusion problem: because an LLM receives system prompts, user messages, tool outputs, and its own reasoning as one undifferentiated token stream, it cannot reliably tell its own thoughts from injected text. Framing roles as the core abstraction lets the post derive new attacks, explain existing mech-interp results, and predict when an injection will succeed.
