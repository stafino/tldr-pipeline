The text in Claude Code’s “Extended Thinking” output (1 minute read)

The practical takeaway for engineers is to stop treating those local files as an audit trail, because reconstructing the real logic that drove an agent's actions requires an enterprise agreement and is impossible from your machine alone.

Prompt Injection as Role Confusion (1 minute read)

The author argues that prompt injection is fundamentally a role-confusion problem: because an LLM receives system prompts, user messages, tool outputs, and its own reasoning as one undifferentiated token stream, it cannot reliably tell its own thoughts from injected text. Framing roles as the core abstraction lets the post derive new attacks, explain existing mech-interp results, and predict when an injection will succeed.

CCCL Runtime: A Modern C++ Runtime for CUDA (1 minute read)
