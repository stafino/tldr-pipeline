IntentTester: Intent-Driven Multi-agent Framework for Cross-Library Test Migration (1 minute read)

IntentTester is a multi-agent framework that migrates unit tests across libraries and languages by abstracting them into a language-agnostic Test Description Language, aligning them to a repository graph, and synthesizing executable tests through LLM reasoning and iterative validation. Across nine Java and Python projects it produced 2,776 syntactically correct tests at 85% correctness versus 51% and 43% for structure-mapping baselines, and surfaced real defects including stack overflows and null dereferences that maintainers have patched.

Constraint Tax in Open-Weight LLMs: An Empirical Study of Tool Calling Suppression Under Structured Output Constraints (1 minute read)

This paper documents Tool Suppression, a reproducible failure where multiple open-weight models stop calling tools once JSON Schema constraints and tool calling are enabled together, even though each works fine in isolation. The authors trace it to grammar-based token masks that make tool-call tokens unreachable during decoding, and propose Transparent Two-Pass Execution to decouple tool calls from schema-constrained generation without retraining.
