How we built saga rollbacks for Cloudflare Workflows (1 minute read)

Cloudflare shipped saga rollbacks for Workflows, letting developers declare compensation logic inside each step.do() rather than tracking what succeeded and what to undo in separate orchestration code outside the steps. Each rollback runs as its own durable step that must be idempotent and continue even if a prior compensation fails, so a partially completed multi-step transaction can semantically reverse committed operations like a debit at one bank after a downstream credit fails.

Scaling AI Inference Across Multiple GPUs Using NVIDIA TensorRT with Multi-Device Inference Support (1 minute read)

TensorRT 11.0 adds multi-device inference support, bringing native multi-GPU inference to the TensorRT runtime while preserving kernel fusions, memory planning, and quantization across devices. The feature integrates directly with NCCL for transport optimization across NVLink, NVSwitch, PCIe, and InfiniBand, and supports tensor and context parallelism so layer weights that exceed a single GPU's memory can be sharded across multiple devices.

Interesting Paper Exploring Prompt Injection (1 minute read)

A new paper argues that LLMs fall for prompt injection because role tags are a formatting convention that never becomes a real boundary in the model's internal representations, with the model leaning on the style of text in each block rather than the tags themselves. The authors frame injection as role confusion and conclude that without genuine role perception, defenses stay a whack-a-mole game vulnerable to attacks that subtly shift model state through innocuous-looking text.

Deno 2.9 (1 minute read)

Deno 2.9 ships deno desktop, an experimental command that wraps a script or auto-detected web framework project (Next.js, Astro, SvelteKit, and others) into a single native binary with the UI in a webview and logic in Deno, no Electron or Tauri toolchain required. The release also lets deno install read npm, pnpm, yarn, and Bun lockfiles directly, and adds CSS module imports, a stronger test runner, faster startup, and Node.js 26 compatibility.

Zig's new bitCast semantics and LLVM back end improvements (1 minute read)

Zig's latest main-branch changes redefine `@bitCast` as a pure value operation rather than memory reinterpretation, and rework the LLVM backend to store arbitrary bit-width integers in zero- or sign-extended ABI-sized slots instead of LLVM's poorly-tested bit-int types. The new lowering matches how Clang emits C's `_BitInt(N)`, fixing missed optimizations and miscompilations the old path triggered.
