A single config file in a cloned repository could steal your AWS credentials through Amazon Q Developer (1 minute read)

Wiz Research disclosed CVE-2026-12957, a high-severity flaw in Amazon Q Developer that auto-registered and started attacker-controlled MCP servers from a cloned repo's config file with no consent prompt, handing over the developer's full AWS credentials, IAM role, and environment variables. Amazon patched it on May 12 by requiring explicit approval before any MCP server starts and restricting the environment variables those servers can access.

Miasma Malware Targets npm Packages and GitHub Actions in Supply Chain Attack (1 minute read)

Researchers tied a new wave of npm supply chain attacks to the Mini Shai-Hulud, Miasma, and Hades malware family, which trojanized two dozen packages including LeoPlatform and RStreams after compromising a maintainer's npm token and pushing malicious versions within a six-second window. The campaign skips package.json lifecycle hooks in favor of binding.gyp install-time execution, pulls in the Bun runtime to stage a credential stealer, and abuses GitHub Actions to exfiltrate CI/CD secrets.

Miasma campaign poisons 20-plus npm packages, hunts for developer secrets (1 minute read)

A supply-chain campaign dubbed Miasma has compromised more than 20 npm packages with malicious code that scans infected machines for developer secrets like API keys, cloud credentials, and tokens. The packages exfiltrate harvested data to attacker-controlled servers, extending the run of registry poisoning attacks that make every untrusted dependency install a credential-theft risk.

Amazon Q Developer Flaw Could Let Malicious Repos Run Code via MCP Configs (1 minute read)

Amazon patched a high-severity flaw in Amazon Q Developer (CVE-2026-12957, CVSS 8.5) that let a malicious repo run arbitrary code and steal a developer's live AWS session by dropping a single `.amazonq/mcp.json` config that the assistant auto-launched. Wiz Research showed the MCP servers inherited the developer's full environment, so update Language Servers for AWS to 1.69.0 across all four bundled IDE plugins.

Oracle promises to open up MySQL governance, but the community wants guarantees (1 minute read)

Oracle pledged to open up MySQL's governance after years of community frustration over its unilateral control of the open-source database. Maintainers and downstream forks like MariaDB want binding guarantees rather than promises, citing Oracle's track record of steering roadmap and licensing decisions without external input.

Samsung will start charging for SmartThings API access (1 minute read)

Samsung will start charging for SmartThings API access beginning in October, introducing a $5 monthly plan for non-commercial individual developers that hits anyone controlling Samsung devices through third-party tools like Home Assistant. The standard SmartThings app stays free, but the change puts custom and open-source smart home setups behind a cloud paywall, with Samsung citing unspecified enterprise-grade investments and a forthcoming Developer Center.
