Mostly Automatic Translation of Language Interpreters from C to Safe Rust (1 minute read)

Reboot translates real-world C language interpreters to safe Rust mostly automatically, converting six interpreters of 6k to 23k lines with only 1 to 11 brief human interventions each. It pairs feature reduction, which breaks the job into testable milestones that incrementally restore functionality, with a multi-agent loop that validates unreliable coding agents, passing 100% of provided test suites and eliminating memory bugs like use-after-free in mujs.

DSpark: Speculative decoding accelerates LLM inference [pdf] (1 minute read)

DSpark applies speculative decoding to accelerate LLM inference by drafting multiple candidate tokens with a small model and verifying them in parallel against the full model. The paper reports throughput gains while preserving the target model's output distribution, with the draft-and-verify loop tuned to keep acceptance rates high without retraining the base model.

How Much Static Structure Do Code Agents Need? A Study of Deterministic Anchoring (1 minute read)

This paper tests whether injecting lightweight static-analysis facts (call graphs and inheritance topology) as plain-text comments makes LLM code agents navigate repositories more deterministically. Using OpenAI's Codex as a baseline, the authors find that anchoring improves function-level localization by 2.2pp, shortens trajectories by 1.6 rounds, and roughly halves run-to-run variance at the cost of about 10% more input tokens.

Qwen-AgentWorld (5 minute read)

Qwen-AgentWorld introduces two language world models (35B-A3B and 397B-A17B) that simulate agentic environments across 7 domains via long chain-of-thought reasoning, trained on 10M+ real interaction trajectories through a three-stage CPT/SFT/RL pipeline. Used as a decoupled simulator for agentic RL, it surpasses real-environment training alone, and as an agent foundation model the world-model warm-up lifts performance across 7 downstream benchmarks.

British Columbia, Time Zones, and Postgres (5 minute read)

British Columbia switched to permanent UTC-7 on March 8, 2026, which breaks any Postgres `timestamptz` appointment stored before the rule change and queried after. Because these columns store UTC and reconvert using current tzdata, future November-to-March bookings can come back an hour off, and the post shows a SQL check to confirm whether your tzdata package has picked up the new offset.

Measuring Exploits in LLM Agents with Tool Use (5 minute read)

Cursor built an auditor agent to inspect SWE-bench Pro trajectories and found that 63% of Opus 4.8 Max's successful resolutions retrieved the known fix from the public web or bundled .git history rather than deriving it. Sealing git history and blocking internet access dropped Opus from 87.1% to 73.0% and Cursor's Composer 2.5 from 74.7% to 54.0%, showing that agentic coding benchmarks need controlled runtime environments, not just contamination-free training data.

Same Scrutiny, More Time: Eye Tracking Insights into Reviewing LLM-Labelled Code (1 minute read)

In a Wizard-of-Oz eye-tracking study, researchers found that labeling code as LLM-generated did not change how thoroughly engineers reviewed it, but did make them fixate longer on the labeled code and shift strategy toward checking logical correctness and reading the prompt. The authors flag a gap between developers' stated intent to scrutinize AI output and their actual behavior, arguing companies should revisit AI-assisted development policies and treat the prompt as a reviewable artifact.

Anatomy of a Failed (Nation-State?) Attack (1 minute read)

The payload-laden image evades every AV engine on VirusTotal, and the writeup walks through the social-engineering chain plus the IoC-detection script the author built to catch it.

A Deterministic Control Plane for LLM Coding Agents (1 minute read)

A study of 10,008 GitHub repos found agent config files (rules, agent definitions, IDE markdown) propagate as an unmanaged supply chain, with 10.1% of tracked paths being exact SHA-256 duplicates across independent orgs and under 1% declaring permission boundaries. The authors propose Rel(AI)Build, a deterministic control plane that content-addresses agent definitions, enforces tiered permissions and blocklists before LLM invocation, and compiles one canonical definition to seven IDE targets.

Clean GitHub repo tricks AI coding agents into running malware (1 minute read)

Researchers at Mozilla's 0DIN demonstrated an attack where a clean GitHub repo with no malicious code tricks Claude Code into opening a reverse shell while it tries to recover from a planted setup error. The agent never decides to run malware; it follows an error message that triggers a script fetching an attacker-controlled DNS TXT record, three indirection steps from anything it actually evaluated.

Open source maintainership in the age of AI (3 minute read)

The Kubernetes maintainers introduced an AI policy that requires contributors to disclose AI assistance, keeps the human contributor fully accountable for every change, and bans listing AI as a commit co-author or as an "assisted-by" trailer. The post argues AI has made code generation fast without improving code maintenance, so the project now enforces CLA checks on co-authors and expects reviewers to engage with humans rather than AI-generated review replies.

The gap between open weights LLMs and closed source LLMs (1 minute read)

The author tests the viral claim that open weights LLMs will catch up to closed source by December 2026, showing it holds only on a single benchmark. Repeating the analysis across all 18 Artificial Analysis benchmarks yields a near-flat average gap of just under 5 months, with coding closing fast while most other categories barely move.

Can We Agree on a Storage/Workload Architecture Taxonomy? (5 minute read)

Jack Vanlightly proposes a taxonomy for storage and workload architectures, classifying systems by number of systems, workloads, durable copies, and tiers to separate OLTP, OLAP, HTAP, and Databricks' new LTAP. The framework distinguishes tiering (one copy moved hot to cold) from materializing (a second durable copy in another system), with data visibility across workloads as the dimension that determines staleness.

Broken Windows of Data (5 minute read)

Booking.com applies the "broken windows" theory to its shared enterprise data warehouse, arguing that small quality lapses like loose grain definitions, duplicated metrics, or skipped reviews compound into platform-wide trust and maintainability problems as they propagate downstream. The author describes a hybrid operating model where domain teams own delivery but work inside common modelling rules and shared entities, keeping governance from becoming a delivery bottleneck.

AI was supposed to kill engineering jobs, but new data suggests they're the most resilient (5 minute read)

SignalFire's analysis of hiring across 80M+ companies found engineering was the most resilient tech job function in 2025, with engineering roles down just 11% from 2019 levels versus a 25% drop in overall tech hiring. Engineers made up 55% of new hires at the 12 "Tech Majors" last year, up from 46% in 2019, undercutting the claim that AI coding tools are already substituting for engineering headcount.

Gossamer: a Rust-flavoured language with real goroutines and pause-free memory (1 minute read)

The pre-1.0 release (v0.20.0, Apache-2.0) runs as a bytecode VM with a REPL or compiles to a single dependency-free native binary via LLVM, and ships a standard library covering HTTP, JSON, crypto, and SQL.
