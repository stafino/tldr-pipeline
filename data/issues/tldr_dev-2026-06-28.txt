DLL that was not present in memory despite not being formally unloaded (1 minute read)

A debugging post-mortem traces a third-party program's crashes, initially blamed on shell32.dll, to a recursive exception-handling death spiral where an exception thrown inside RtlLookupFunctionEntry restarts KiUserExceptionDispatch on every pass until the stack overflows. Walking to the bottom of the call stack shows shell32 was only fingered as the apparent origin, and the post explains why the real fault sits in the exception dispatch loop rather than the library it was assigned to.

WAL-RUS: a Rust Rewrite of WAL-G for PostgreSQL Backups (1 minute read)

ClickHouse released WAL-RUS, an open-source Rust reimplementation of WAL-G for PostgreSQL WAL archival and backups, designed to be WAL-G compatible while delivering a predictable memory profile. The motivation is Go's garbage-collected runtime, whose sawtooth memory pattern and reserved virtual memory force operators to over-provision backup infrastructure, taking RAM away from Postgres queries, shared buffers, and page cache.

Wayfinder Router: deterministic routing of queries between local and hosted LLM (1 minute read)

Wayfinder Router scores a prompt's structure (length, headings, lists, code) and wording to decide whether it goes to your small local model or a big cloud one, in microseconds and fully offline with no API key, network call, or model judge. Unlike classifier- or LLM-based routers, the decision is deterministic and free, and it composes with gateways like LiteLLM or OpenRouter, which still pick the provider underneath.
