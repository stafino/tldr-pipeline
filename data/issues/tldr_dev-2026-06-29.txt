WATaBoy: JIT-Ing Game Boy Instructions to WASM Beats a Native Interpreter (1 minute read)

A developer building a Game Boy emulator for their final-year project generated WebAssembly bytecode from Game Boy instructions, exploiting the JIT compilation that JavaScriptCore applies to hot Wasm in browsers, the one path around Apple's iOS JIT ban. The JIT-to-Wasm approach stays cycle-accurate by predicting interrupts and falling back to an interpreter, and it beats a natively running interpreter on performance.

What happens when you run a CUDA kernel? (1 minute read)

This post traces a single CUDA vector-add kernel from source down to the warps that execute it and back, covering how nvcc fans out to multiple compilers (cicc, ptxas, fatbinary) to produce PTX, SASS, and an embedded fatbin. The walkthrough then follows the launch through the driver stack, accounting for the tens of millions of CPU instructions, two device files, ~900 ioctls, and one memory-mapped doorbell register that turn a million float additions into a result.

Factoring RSA Keys with Many Zeros (1 minute read)

Researchers found real-world RSA keys with regularly spaced blocks of all zeros in their moduli, a sparsity pattern that makes them far easier to factor, surfaced by scanning Certificate Transparency logs, TLS/SSH scans, and PGP dumps with the open-source badkeys project. The flaw traces to independent implementations failing the same way, including CompleteFTP's key generation and certificates once issued to Yahoo and Verizon, suggesting cryptanalytic attacks tuned for this failure class are worth building.

Mozilla warns of indirect prompt injection risk in AI coding agents (1 minute read)

Mozilla's 0DIN team showed that a malicious GitHub repo can compromise a developer's machine via indirect prompt injection, with no malicious code in the repo itself. The proof-of-concept tricks agents like Claude Code into running setup steps that pipe an attacker-controlled DNS TXT record to bash, fetching a reverse shell at runtime that's invisible to code review, static analysis, and the agent reading the repo.

Researchers Demo New Claude Code Attack Using Harmless-Looking Repositories to Hijack Developer Machines (1 minute read)

Mozilla's 0Din researchers demonstrated an attack where a clean-looking repository hijacks a developer's machine through Claude Code, with no malicious code in the repo itself. A deliberate install error tells the agent to run `axiom init`, which executes a script that pulls a base64-encoded reverse shell from a DNS TXT record, splitting the payload across three systems no single tool inspects together.

A native graphical shell for SSH (1 minute read)

Outer Shell is an open-source graphical shell for SSH, where each app is a minimal HTTP server exposed over a Unix domain socket rather than a localhost port, with encryption handled at the SSH layer instead of in the app. The shell provides a home screen of apps plus an API for inter-app lookups, so a text-editor app can register itself and open files double-clicked elsewhere.

Highlights from Git 2.55 (1 minute read)

Git 2.55 shipped with contributions from over 100 developers, headlined by `git repack --write-midx=incremental`, which writes incremental multi-pack index chains directly instead of rewriting a single MIDX over every pack. The append-only mode lets large repositories index newly created packs by adding a layer to the chain, avoiding the expensive full-file rewrite that single-MIDX maintenance forces on each small update.
