‘DirtyClone’ Linux Kernel Vulnerability Leads to Root Access (1 minute read)

JFrog published technical details and a proof of concept for DirtyClone (CVE-2026-43503, CVSS 8.8), a Linux kernel local privilege escalation bug that lets any local user gain root by corrupting file-backed page-cache memory through zero-copy socket buffer paths. The fix landed in v7.1-rc5, but only kernels carrying the full DirtyFrag patch chain are protected, leaving Debian, Fedora, and Ubuntu systems with unprivileged user namespaces exposed.
