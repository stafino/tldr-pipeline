WATaBoy: JIT-Ing Game Boy Instructions to WASM Beats a Native Interpreter (1 minute read)

Memory Safe Context Switching (1 minute read)

Fil-C implements longjmp/setjmp and the ucontext coroutine APIs (getcontext, setcontext, makecontext, swapcontext) so that no misuse can corrupt the stack or escape its capability model. Restoring a returned-from or freed stack, swapping into the currently executing context, or running on a dangling stack all become clean panics instead of the unprintable, exploitable crashes those same bugs produce in ordinary C.
