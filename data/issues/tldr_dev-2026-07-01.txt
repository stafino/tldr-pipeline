Parse, Don't Validate – In a Language That Doesn't Want You To (1 minute read)

GuardFall Exposes Open-Source AI Coding Agents to Decades-Old Shell Injection Risks (1 minute read)

Adversa AI's GuardFall research shows that ten of eleven popular open-source coding agents can be tricked into running blocked commands, because their blocklists match commands as plain text while bash rewrites that text before execution, so `r''m` sails past a filter watching for `rm`. Microsoft Warns Poisoned MCP Tool Descriptions Can Make AI Agents Leak Data (1 minute read)

Microsoft Incident Response and Defender researchers show that a poisoned MCP tool description can turn an AI agent into a data-exfiltration path, since the agent treats the description's hidden instructions as legitimate and every step looks routine enough to trip no alarm. Because every step is authorized, IAM, EDR, and network controls flag nothing, leaving cloud credentials, GitHub tokens, and CI/CD secrets exposed.
