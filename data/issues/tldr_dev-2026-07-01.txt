GuardFall Exposes Open-Source AI Coding Agents to Decades-Old Shell Injection Risks (1 minute read)

Adversa AI's GuardFall research shows that ten of eleven popular open-source coding agents can be tricked into running blocked commands, because their blocklists match commands as plain text while bash rewrites that text before execution, so `r''m` sails past a filter watching for `rm`. The authors frame this as a class of problems rather than a patchable bug, since the gap opens whenever an agent auto-executes shell commands with full account access against untrusted repos or packages.

Microsoft Warns Poisoned MCP Tool Descriptions Can Make AI Agents Leak Data (1 minute read)

Microsoft Incident Response and Defender researchers show that a poisoned MCP tool description can turn an AI agent into a data-exfiltration path, since the agent treats the description's hidden instructions as legitimate and every step looks routine enough to trip no alarm. The core weakness is that MCP tools ship with plain-text descriptions the agent trusts, and MCP picks up description changes on the fly, so a third-party tool approved once can go malicious without re-review.
