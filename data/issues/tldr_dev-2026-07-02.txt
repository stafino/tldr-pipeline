Factoring RSA Keys with Many Zeros (1 minute read)

Researchers found real-world RSA keys with regularly spaced blocks of all zeros in their moduli, a sparsity pattern that makes them far easier to factor, surfaced by scanning Certificate Transparency logs, TLS/SSH scans, and PGP dumps with the open-source badkeys project. RepoRescue: An Empirical Study of LLM Agents on Whole-Repository Compatibility Rescue (1 minute read)

RepoRescue benchmarks whether LLM agents can restore old repositories that passed historically but break after dependency and runtime drift, using 193 Python and 122 Java repos that each fail only after modernization. Under runtime blocking that stops agents editing failing tests, Kimi rescues 41.5% of repos and the union of systems hits 62.7%, with cross-file coordination the main failure mode and passing test suites proving only a weak signal of real fixes.

Are Performance-Optimization Benchmarks Reliably Measuring Coding Agents? (1 minute read)

This paper audits three repository-level performance-optimization benchmarks (GSO, SWE-Perf, SWE-fficiency) and finds their leaderboard scores conflate runtime instability, scoring-rule quirks, and task saturation, with reference patches passing cross-machine replay for only 39/102 GSO and 11/140 SWE-Perf tasks. The authors show public rankings flip on 9 of 28 pairwise comparisons depending on the scoring rule, arguing aggregate scores overstate coding-agent progress and hide the real per-task performance gaps.

We Ran 250 AI Agent Evals to Find Out if Skills Beat Docs. The Answer Is More Complicated Than We Expected (5 minute read)

Wix ran 250 controlled evals comparing AI agents on standard docs, AI-optimized docs, and purpose-built skills, and found that purpose-built skills only win when kept current. Text AI watermarks will always be trivial to remove (3 minute read)

The author argues that text AI watermarks will always be trivial to strip, because text is too compressed a medium to hide a signature without degrading output quality, unlike images.
