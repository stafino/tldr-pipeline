60% Fable cost cut by converting code to images and having the model OCR it (1 minute read)

pxpipe is a local proxy that renders the bulky, token-dense parts of a Claude Code request (system prompt, tool docs, older history) into compact PNGs and appends an OCR instruction, exploiting the fact that an image's token cost scales with pixel dimensions rather than character count. On dense content it packs roughly 3.1 chars per image-token versus 1 per text-token, cutting input tokens enough to land a 59-70% lower bill at current Fable prices.
