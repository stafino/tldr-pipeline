AI Writes Faster Than Humans Can Review: A Longitudinal Study of an Enterprise 2x Mandate (1 minute read)

A longitudinal study of an enterprise "2x" mandate tracked 802 developers and 196,212 pull requests from January 2024 to April 2026, finding per-capita merged-PR throughput reached 2.09x the pre-mandate baseline, with a staggered difference-in-differences design tying most of the gain to AI adoption and accumulated usage rather than the mandate itself. Automated review overtook human review as per-reviewer load roughly doubled, while merge and revert rates held steady.

How OpenAI Delivers Low-Latency Voice AI for 900M Users (5 minute read)

OpenAI serves voice AI to 900M weekly users on WebRTC, splitting the stack into a stateless edge relay for packet routing and a stateful transceiver that owns the heavy WebRTC state. Program-as-Weights: A Programming Paradigm for Fuzzy Functions (1 minute read)

This paper proposes fuzzy-function programming, which compiles a natural-language spec into a small locally-executable neural artifact instead of routing every call to an LLM API. A 4B compiler trained on the released 10M-example FuzzyBench emits adapters for a frozen 0.6B Qwen3 interpreter that matches Qwen3-32B prompting at roughly one fiftieth the inference memory, running at 30 tokens/s on a MacBook M3.

Postgres transactions are a distributed systems superpower (1 minute read)

The DBOS team argues that co-locating your durable workflow state in the same Postgres database as your application data lets both be updated in a single transaction, eliminating partial failures. The post shows how this collapses hard problems like idempotency and atomicity into ordinary transactional steps, dropping the usual application-level bookkeeping such as a separate applied_payments table used to guard against double-crediting on retry.

Too many tables are bad for you (5 minute read)

The Cybertec team benchmarks how a Postgres schema with tens of thousands of tables degrades performance, tracing the slowdown to bloated system catalogs, autovacuum churning through every relation, and slower planning as the catalog cache thrashes. This paper studies regression accumulation in multi-turn LLM coding sessions, where later suggestions break requirements from earlier turns. How To Corrupt An SQLite Database File (5 minute read)

The SQLite documentation catalogs the concrete ways a database file goes corrupt despite SQLite's automatic rollback, from processes overwriting the file with garbage to stale file descriptors leaking writes into a reopened database. A notable case hit the Fossil DVCS repository when stderr was closed before sqlite3_open_v2(), leaving fd 2 pointing at the database so a later assert() error message wrote directly into it, which motivated SQLite 3.8.1 refusing low-numbered file descriptors.

Phantom Squatting: AI-Hallucinated Domains as a Software Supply Chain Vector (5 minute read)

Unit 42 found that LLMs consistently hallucinate plausible domains for legitimate brands, and attackers are registering these nonexistent domains to intercept traffic from AI coding assistants and other tools, a technique the researchers call phantom squatting. Across 913 brands and 2.1 million generated URLs they surfaced roughly 250,000 unregistered hallucinated domains, and predicted one real attacker registration 23 days before a phishing kit went live.

Best Simple System for Now (2025) (1 minute read)

Dan North argues the perfectionism-versus-pragmatism split is a false trade-off, proposing a middle path he calls the Best Simple System for Now (BSSN): the simplest system that meets the product's current needs, written to an appropriate standard, with no over-engineered code and no more robustness than required. The framing rejects both quick-and-dirty tech debt and speculative gold-plating, treating "for now" as an explicit constraint rather than an excuse to cut corners.

Critical Cursor AI Code Editor Flaws Could Lead to OS-Level Remote Code Execution (1 minute read)

Cato Networks disclosed two critical vulnerabilities in Cursor, dubbed DuneSlide (CVE-2026-50548 and CVE-2026-50549, CVSS 9.8), that let a prompt-injected payload escape the IDE sandbox and achieve OS-level remote code execution. Both bugs abuse Cursor's auto-run terminal execution to overwrite the cursorsandbox executable, one via a working_directory allow-list flaw and the other via a symlink path-canonicalization bypass, and were patched in Cursor 3.0.

Since Linux 6.9, LUKS suspend stopped wiping disk-encryption keys from memory (1 minute read)

A regression introduced in Linux 6.9 broke `cryptsetup luksSuspend`, which is supposed to flush disk-encryption keys from kernel memory but silently stopped doing so. The bug leaves LUKS keys recoverable from RAM on suspended machines, defeating the exact threat model laptop users rely on suspend-and-lock to cover, and it went unnoticed across multiple stable releases.
