Web-based cryptography is always snake oil (1 minute read)

The author argues that all web-based end-to-end encryption is inherently broken, coining the rule that a cryptosystem is incoherent when its implementation is distributed by the same entity it claims to secure against. Because a website serves the client-side JavaScript, a malicious operator can always push modified code, and the author extends the same critique to Signal and WhatsApp for banning third-party clients.

The LLM API Failure Policy I Wish I Had Before My First Production Incident (2 minute read)

The author argues that HTTP status codes and SDK exception handling are too shallow for LLM API failures, since a single 429 can mean requests-per-minute, tokens-per-minute, daily quota, concurrency, or provider overload, each demanding a different response. The proposed fix is a failure policy layer near the LLM client that maps provider-specific errors into internal categories and actions like retry, queue, switch models, degrade, or page someone.

OpenPrinter (1 minute read)

OpenPrinter is an open-source, repairable printer/plotter built from standard components, with refillable cartridges and independent black and color channels so an empty color tank no longer blocks black printing. It supports roll or sheet paper up to A3 via an integrated cutter for banners and custom formats, and is designed to be assembled, maintained, and repaired to extend the life of both the machine and its consumables.
