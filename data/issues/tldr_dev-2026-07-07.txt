On the risk of coding before testing: An empirical study on LLM-based test generation workflow (1 minute read)

This paper challenges the assumption behind test-first and agentic LLM workflows: that a model's generated tests act as independent oracles. The authors show error propagation is prevalent, with faulty code biasing subsequent test generation and dropping fault-detection rates from 25% to 14%, since incorrect implementations and their tests become mutually consistent and mask the defect rather than expose it.

New additions include Math.sumPrecise for low-error summation, Array.fromAsync for building arrays from async iterables, Error.isError for reliable error detection, Uint8Array hex and base64 conversion methods, and Map getOrInsert helpers for default values on missing keys.

Apple Container: Open-source tool for Linux containers on the Mac (1 minute read)

Apple released Container 1.0.0, an open-source Swift tool that runs each Linux workload as its own lightweight VM on Apple silicon, using OCI-compatible images so developers can pull from and push to any standard registry. The release adds a container machine command for persistent Linux environments, a container cp command for host-to-container file transfers, and moves configuration to a config.toml file, a breaking change that requires macOS 26.

Python 3.14 compiled to metal – no interpreter (1 minute read)

pon is a JIT and AoT native compiler and runtime for Python 3.14, written in Rust with no interpreter or bytecode: it parses modules with the ruff parser, lowers them to one shared IR, and compiles to machine code through Cranelift, either in-process or into a standalone executable. It swaps reference counting for a Green Tea garbage collector and enforces correctness with a byte-exact differential harness against CPython 3.14.0.

Kani: A Model Checker for Rust (1 minute read)

Kani is an open-source bounded model checker for Rust that compiles proof harnesses from MIR into CBMC's verification engine, checking safety properties like unsafe soundness, functional correctness, and panic-freedom with no annotations required. Function and loop contracts extend it from bounded to unbounded verification, and it runs in production CI at over 16,000 harnesses per code change in the Rust standard library campaign, where it uncovered six previously unknown bugs.

North Korean Hackers Target Open Source Developers in Supply Chain Attacks (1 minute read)

Socket disclosed PolinRider, a North Korean supply chain campaign running since December 2025 that compromises maintainer accounts across NPM, Packagist, Go modules, and Chrome extensions to push the DEV#POPPER RAT and OmniStealer via obfuscated JavaScript loaders. Socket has found 162 malicious release artifacts across 108 packages, with attackers rewriting Git history to age the changes and pulling encrypted payloads from blockchain RPC infrastructure.
