A road to Lisp: Why Lisp (1 minute read)

The author argues that learning Lisp makes you a better programmer because its macro system lets you extend the language itself, growing it toward your problem rather than bending your problem to fit the language. Framing the difficulty through Paul Graham's Blub paradox, the piece walks through macros, the REPL, and conditions and restarts to show the capabilities that lower-power languages leave you unable to even perceive.

Show HN: Getting GLM 5.2 running on my slow computer (1 minute read)

Colibrì runs GLM-5.2, a 744B-parameter MoE model, on a consumer machine with about 25GB of RAM by keeping the ~17B dense params resident at int4 and streaming the 21,504 routed experts from disk with a per-layer LRU cache. The engine is a single ~2,400-line C file with no BLAS, Python, or GPU required, and uses GLM-5.2's own int8 MTP head for lossless speculative decoding at 39-59% draft acceptance.

Wiz disclosed GhostApproval, a vulnerability pattern in six AI coding assistants (Amazon Q Developer, Claude Code, Augment, Cursor, Google Antigravity, and Windsurf) where a malicious repository uses symlinks to escape the workspace sandbox while the approval prompt hides the dangerous target from the developer. The agent's own reasoning often flags the sensitive file, but the CWE-451 UI misrepresentation means users approve what looks like a harmless local edit, potentially yielding remote code execution.

npm 12 Disables Install Scripts by Default to Reduce Supply Chain Risk (1 minute read)

npm 12 ships with install scripts disabled by default, so lifecycle hooks (preinstall, install, postinstall) and implicit node-gyp builds no longer run unless developers opt in via npm approve-scripts and commit an allowlist to package.json. The release also deprecates 2FA-bypass granular access tokens, stripping their ability to publish directly and forcing automated publishing toward trusted publishing (OIDC) or staged publishing with human approval.

Interview with Mitchell Hashimoto about Ghostty and Zig (1 minute read)

He started it to relearn GPU and desktop systems programming after years on distributed systems, and argues against turning terminals into general application platforms since the monospaced-grid model is good at something the browser and desktop are not.
