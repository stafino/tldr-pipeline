Since Chromium 148, Math.tanh is now fingerprintable to link underlying OS (1 minute read)

Since Chromium 148, V8 replaced its bundled fdlibm port of `Math.tanh` with `std::tanh`, which routes through the host libm and returns OS-specific bits, so `Math.tanh(0.8)` now differs by 1 ULP between glibc, Apple's libsystem_m, and Windows UCRT. The post traces the leak across V8, Blink CSS trig, and the Web Audio compressor, and shows that spoofing it means bit-for-bit reproducing another OS's math library.

ZeroFS vs. Amazon S3 Files (1 minute read)

ZeroFS and Amazon S3 Files both mount a POSIX filesystem on top of object storage, but they invert the bucket's role: S3 Files (built on EFS) keeps a one-file, one-object mapping so images/cat.jpg exports back to the same S3 key, while ZeroFS packs compressed, client-side-encrypted extents into opaque immutable segments. The trade-off is direct S3 interoperability, since a ZeroFS pathname can never be fetched with GetObject and recovery requires the encryption password.
