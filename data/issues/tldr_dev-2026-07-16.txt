Show HN: misa77 - a codec that decodes 2x faster than LZ4 (at better ratios) (1 minute read)

Cursor Flaw Lets Malicious Cloned Repositories Trigger Windows Code Execution (1 minute read)

Cursor on Windows executes any `git.exe` sitting in a repository's root when the project loads, spawning it with `git rev-parse --show-toplevel` and re-running it for as long as the folder stays open. Mindgard reported the unqualified-binary lookup to Cursor on December 15, 2025 and published a proof of concept seven months later with no patch and no advisory; cloning an untrusted repo is the entire exploit chain.

Cursor 0day: When Full Disclosure Becomes the Only Protection Left (1 minute read)
