Prompt injection is becoming the XSS of the web agent era (1 minute read)

UC Berkeley researchers argue that prompt injection is the web-agent era's Cross-Site Scripting, since autonomous agents read untrusted page content like reviews and listings as executable instructions that attackers can exploit. Their system, Prismata, sits between agent and browser and uses page structure plus the 1977 Biba integrity model to filter which elements an agent can act on, after finding only 1.2% of untrusted content across 90,000 samples reached an actionable element.

Minikotlin (1 minute read)

Minikotlin is a Kotlin compiler written from scratch in C that emits WebAssembly GC bytecode by hand, with no JVM, LLVM, Binaryen, or Gradle in the pipeline. The compiler is itself compiled to WASM and runs entirely in the browser tab, lowering real Kotlin features like sealed hierarchies, coroutines as CPS over closures, generics, and vtable-based dynamic dispatch onto the WASM-GC type system.

Show HN: Leaves – A text-UI disk usage treemap visualizer (1 minute read)

Leaves is a text-mode disk usage analyzer inspired by WinDirStat and QDirStat that renders files and directories as a treemap of nested rectangles, with each rectangle's area proportional to its size. Written in Rust, it trades the precision of graphical tools for terminal-level resolution but runs over remote SSH sessions where a desktop environment is unavailable, and it respects .gitignore and .ignore rules by default.

AI Prototyper: A Figma Plugin for Decomposition-Based GUI Prototyping with LLMs (1 minute read)

AI Prototyper is an open-source Figma plugin that turns a natural-language screen description into fully editable, auto-layout Figma layers by decomposing the request into GUI features and pulling matches from a 32-primitive library via RAG. It runs on Gemini 2.5 Flash, adds a human-in-the-loop step to edit the feature list before rendering, and supports multilingual output in Thai, English, and Mandarin.
