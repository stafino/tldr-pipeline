Regressive JPEGs (1 minute read)

The author abused JPEG's progressive mode, where a file is split into scans that each fill in a spectral range, to concatenate multiple same-resolution images into one file that animates as it downloads over a slow connection. The trick caps out fast because most decoders stop after roughly nine scans as a zip-bomb guard, and baseline single-scan frames fail since progressive scans cannot mix DC and AC coefficients.

Transcribe.cpp (1 minute read)

Transcribe.cpp is a ggml-based speech-to-text library covering 16 ASR families and 60+ models, with acceleration via Vulkan, Metal, CUDA, and TinyBLAS. The author, who maintains the Handy desktop app, built it as a near drop-in whisper.cpp replacement after hitting the limits of ONNX's CPU-only performance, and every model ships numerically validated and WER tested against its reference implementation.

UnifiedIR for Julia (1 minute read)

A WIP PR proposes UnifiedIR, a new top-level Julia package that replaces SyntaxTree, SyntaxGraph, CodeInfo, and IRCode with a single dialect-aware, region-based IR in the style of MLIR. The core structure is extensible with per-statement columns and layered views, including an IncrementalCompact-style one-pass rewrite view and an LLVM-like linked-list view that uses an order maintenance algorithm to avoid quadratic behavior on large functions.

Hugging Face Breached Through a Malicious Dataset: What the Patches Reveal (2 minute read)

Hugging Face disclosed a compromise of part of its production infrastructure that started with a malicious dataset abusing a remote-code loader and jinja2 template injection in a dataset config, then escalated to node-level access and lateral movement across internal clusters. Public dataset-viewer commits show the response: an fsspec protocol allowlist, worker pods stripped of ServiceAccount tokens and Linux capabilities, and static S3 and Mongo credentials replaced with IRSA auth.

SleeperGem Uses Three Malicious RubyGems Packages to Target Developer Machines (1 minute read)

StepSecurity flagged three malicious RubyGems packages, dubbed SleeperGem, that impersonate Microsoft's Git Credential Manager or revive gems dormant since 2019. The loader checks roughly 30 CI environment variables from GitHub Actions, GitLab, CircleCI, and Jenkins, exiting if any are found, so it only drops its native daemon and persistence on developer machines rather than ephemeral runners.
