Nativ: Run frontier open models locally on your Mac (1 minute read)

Nativ is an open-source, MIT-licensed macOS app that downloads and runs open models from Google, Cohere, and Liquid AI locally on Apple Silicon, built on MLX-VLM and tuned for M-series unified memory and Metal. It recommends a model for your hardware, exposes live tokens/sec, memory pressure, thermal state, and time-to-first-token, and serves a local endpoint so existing coding agents can point at on-device models.

OpenAI reduced the default input context window for GPT-5.6 in the Codex CLI to 272,000 tokens from 372,000, prompting complaints from developers who say long-running sessions now trigger context compaction sooner. Analysts note the change mainly affects large codebases, repository-wide refactoring, and extended sessions, where the agent must summarize or reload context more often and developers spend more time re-establishing state than writing code.
