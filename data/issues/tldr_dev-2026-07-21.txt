Prefactory: Automated Discovery and Application of Library-Adoption Refactorings (1 minute read)

Prefactory finds places in a Python codebase where hand-written code could be replaced by a library API call, using an LLM to synthesize executable lexical and structural detectors instead of prompting over every file. On a 100-refactoring benchmark from 61 open-source projects, it detected 56 instances at the function level against 32 for Codex CLI, and produced 40 refactorings validated by project tests and generated differential tests.

Autoresearch with Coding Agents: Generalizers and Metric-Maximizers on Quran Recitation Data (1 minute read)

Researchers ran unsupervised "autoresearch" loops on a production task, giving Claude Code and OpenAI Codex the same dataset, eval script, and one editable file to optimize against a score. Adding a held-out test set eliminated the memorization entirely.

Transcribe.cpp is a ggml-based speech-to-text library covering 16 ASR families and 60+ models, with acceleration via Vulkan, Metal, CUDA, and TinyBLAS. The author, who maintains the Handy desktop app, built it as a near drop-in whisper.cpp replacement after hitting the limits of ONNX's CPU-only performance, and every model ships numerically validated and WER tested against its reference implementation.

Block launches Buzz, an open source collaboration workspace on Nostr protocol for humans and AI agents to share messages, code repositories, workflows, and more (Block) (1 minute read)

Block released Buzz, an open source collaboration workspace built on the Nostr protocol where humans and AI agents share messages, code repositories, and workflows in the same space. Building on Nostr's relay model means identity and data are keyed to user-controlled keypairs rather than a vendor account, so agents and teams can work across relays without a central server holding the workspace.

CommitLLM: A Fine-Tuned Pipeline for Git Commit Message Generation (1 minute read)

CommitLLM is a three-stage pipeline that turns code diffs into Conventional Commits-compliant messages using QLoRA-fine-tuned Mistral-7B, constrained decoding, and deterministic post-processing, running on a single 16GB T4. Format compliance jumped from 22% to 98% and average message length dropped from 154.8 to 37.9 characters, with the authors noting the post-processing layers contributed more than the fine-tuning itself.

Incremental – A library for incremental computations (1 minute read)

Jane Street's Incremental is an OCaml library for building computations that recompute only the parts affected when their inputs change, based on Umut Acar's work on self-adjusting computation. It targets spreadsheet-style calculation graphs, GUI views that fold in new data without full rebuilds, and derived data like filters or inverted mappings that must stay in sync with their source.

The Perfect Heist: NuGet Typosquat Targets Betting Platform to Rig Results (1 minute read)

JFrog found a typosquatted NuGet package, Newtonsoftt.Json.Net, that shipped a working Newtonsoft.Json fork with a payload targeting a single company, the betting platform Digitain, rigging round results and exfiltrating them to an attacker server. The package was discoverable from August to October 2025 under seven versions and is now unlisted, though all artifacts remain downloadable from its NuGet page.

AWS Kiro Flaw Let a Poisoned Web Page Rewrite Its Config and Run Code (1 minute read)

Hidden one-pixel text on a web page was enough to make Kiro, AWS's agentic coding IDE, write a malicious MCP server into `~/.kiro/settings/mcp.json` and execute it with the developer's privileges. Intezer and Kodem Security found that Kiro's fsWrite tool could edit that file with no approval prompt and auto-reload it, bypassing the human-in-the-loop click that the whole safety model depends on. AWS has patched it, with no CVE assigned.

Cursor, Codex, Gemini CLI, Antigravity hit by sandbox escapes (1 minute read)

Pillar Security published seven sandbox escapes affecting Cursor, OpenAI's Codex CLI, Gemini CLI, and Antigravity, none of which attack the sandbox directly. The agent writes a workspace file that a trusted process outside the sandbox later executes, turning prompt injection in a README or dependency into host command execution. Cursor patched the .claude hook config bug (CVE-2026-48124) in 3.0.0, and OpenAI fixed a git show allowlist gap in Codex CLI v0.95.

SleeperGem Uses Three Malicious RubyGems Packages to Target Developer Machines (1 minute read)

StepSecurity flagged three malicious RubyGems packages, dubbed SleeperGem, that impersonate Microsoft's Git Credential Manager or revive gems dormant since 2019. The loader checks roughly 30 CI environment variables from GitHub Actions, GitLab, CircleCI, and Jenkins, exiting if any are found, so it only drops its native daemon and persistence on developer machines rather than ephemeral runners.
