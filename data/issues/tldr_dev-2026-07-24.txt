Top AIs invent same fake PyPl and npm package names (1 minute read)

New research found that five leading LLMs hallucinated the same 127 nonexistent PyPI and npm package names, a phenomenon fueling "slopsquatting" attacks where hackers register the fake names to slip malware into real codebases. As of April, 53 of those names remained unregistered and no malicious use has surfaced yet, but the overlap across Claude, GPT, Gemini, and DeepSeek shows the attack surface is predictable enough to pre-empt.

LLMs reward expertise (3 minute read)

The author argues that the most important prompting skill is expertise in the domain you are prompting for, not generic prompt-crafting technique. Using Terence Tao's ChatGPT conversation on the Jacobian Conjecture as an example, he shows that signaling domain knowledge shifts the model into expert mode and lets you pull the right idea from its output, the same way a strong theory of your codebase lets you push an LLM harder than an unfamiliar one.

Buz – A fork of Bun using modern Zig, with sub-1s incremental builds (1 minute read)

Buz is a work-in-progress fork of Bun built on modern upstream Zig, forked from the last commit before Bun's Rust rewrite and vendoring JavaScriptCore into build.zig to enable sub-1s incremental builds. The author aims for a drop-in Bun replacement with a cleaner codebase, having already cut over 11,000 lines of dead code, though the fork is early, many imported tests still fail, and it is not production-ready.

Model Context Protocol is going stateless to make scaling simpler (2 minute read)

The Model Context Protocol's July 28 release candidate removes protocol-level sessions in favor of a stateless design, where every request carries the information any available server needs to process it independently. The change lets MCP servers scale horizontally like standard cloud services instead of routing requests back to a pinned machine, with the trade-off that apps needing cross-request context must now manage that state explicitly rather than leaning on the protocol.
