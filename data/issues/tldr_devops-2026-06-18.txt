AWS Continuum brings AI models to code vulnerability management (1 minute read)

AWS opened a gated preview of Continuum for code vulnerabilities, a model-agnostic system that runs discovery, prioritization, validation, and remediation across a finding's full lifecycle, building sandboxed exploits to filter false positives and routing patch recommendations through the same validation step. It reasons over infrastructure, permissions, network topology, and code to rank findings by reachability and production impact, and ships with blast-radius visibility and rollback paths, starting in a human-reviewed learn mode.
