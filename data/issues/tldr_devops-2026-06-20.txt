Bootimus – A Self-Contained PXE and HTTP Boot Server (1 minute read)

Bootimus is a self-contained PXE and HTTP boot server shipped as a single dependency-free Go binary with embedded iPXE, a web UI, and SQLite, and its built-in proxyDHCP answers PXE requests without touching your existing DHCP or router. It auto-detects 50+ distros, extracts kernels and initrds for Ubuntu, Debian, NixOS, Windows, and more, and exposes every UI action as a scriptable API with SSE log streaming.
