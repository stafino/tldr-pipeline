Agent Beacon: Open-source telemetry layer for AI agents (1 minute read)

Asymptote Labs released Beacon, an open-source Go tool that configures OpenTelemetry export for AI coding agents like Claude Code, Codex CLI, and Cursor, normalizing their file edits, command runs, and tool calls into a single JSON event log readable from a loopback dashboard or forwarded to a SIEM. It runs locally under MIT license with no account required, defaulting to full capture but offering redacted retention modes that strip prompt text, command output, and raw diffs.
