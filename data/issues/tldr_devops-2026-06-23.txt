GitHub Updates actions/checkout to Block Common Pwn Request Attack Patterns (1 minute read)

GitHub updated actions/checkout to refuse fork pull request code in pull_request_target and workflow_run workflows by default, blocking the most common pwn request pattern that lets attackers steal the privileged GITHUB_TOKEN and secrets. The change ships in v7 effective June 18, 2026, backports to all supported major versions on July 16, and requires authors to set allow-unsafe-pr-checkout to true to opt out.

AWS unveils sovereign-centric micro serverless virtual machines (1 minute read)

AWS released Lambda MicroVMs, a new Firecracker-backed resource that hands each user or session its own isolated environment with up to 16 vCPUs, 32 GB of memory, and 32 GB of disk for running untrusted or AI-generated code. Idle instances suspend via API or lifecycle policy while preserving memory and disk state, capping costs and targeting the data-sovereignty and isolation demands of multi-tenant AI workloads.

Linux and Secure Boot certificate expiration (2025) (1 minute read)

The Microsoft 2011 key used to sign the shim first-stage UEFI bootloader expires in September, after which Microsoft will stop signing the bootloader that Linux distributions rely on to boot under Secure Boot. A 2023 replacement key exists but is missing from many systems and often requires a vendor firmware update, so the breakage mainly hits new installs on Secure Boot hardware and adds work for distributors and users.

Verifiable, private AI: Google Cloud expands Confidential Computing frontiers (2 minute read)

Google Cloud expanded its Confidential Computing lineup with Confidential G4 VMs and Confidential GKE Nodes, pairing NVIDIA RTX PRO 6000 Blackwell GPUs with 5th-gen AMD EPYC Turin CPUs to keep AI inference and fine-tuning data encrypted inside hardware TEEs, including in transit between CPU and GPU. The preview ships in every region the standard G4 is available, across On Demand, Reservations, DWS Flex Start, and Spot consumption models.

The post walks through the schema design and the trade-offs against the index-heavy operational overhead of the older backends.

How we found a bug in the hyper HTTP library (1 minute read)

Cloudflare engineers spent six weeks tracing why their rearchitected Images binding intermittently returned truncated payloads, 200-status responses that cut a two-megabyte image down to a few hundred kilobytes, only for larger files. The culprit was a race condition in the hyper Rust HTTP library governing how processed image data streamed back to the client. The fix was four lines of code, a reminder that edge-network rollouts surface failure modes invisible under normal load.

Making secret scanning more trustworthy: Reducing false positives at scale (5 minute read)

GitHub partnered with Microsoft Security & AI's Agents Offense team to add LLM-based contextual verification to its secret scanning, reasoning about how a detected value appears in code rather than whether it merely matches a secret-like pattern. The goal is to bring AI-detected secrets closer to the precision GitHub already achieves on provider-pattern detections across billions of pushes. Fewer low-value alerts means less triage toil and more developer trust in the system, without sacrificing coverage.

Blinding the Watchmen: Abusing Cloud Logging Services for Defense Evasion and Visibility (5 minute read)

Unit 42 researchers detail how attackers target cloud logging services like AWS CloudTrail and Google Cloud Logging to blind defenders, breaking down two attack classes: defense evasion that modifies or disables logging resources to execute unnoticed, and continuous visibility that siphons logs into attacker-controlled accounts. The writeup maps the specific configurations and detections platform teams need to catch logging-service tampering before it erases the audit trail you rely on for incident response.

Run isolated sandboxes with full lifecycle control: AWS Lambda introduces MicroVMs (1 minute read)

AWS introduced Lambda MicroVMs, a Firecracker-backed serverless primitive that hands each user or session its own isolated, stateful execution environment with VM-level isolation, near-instant launch and resume, and pause-to-idle pricing. It targets multi-tenant apps that run untrusted user- or AI-generated code, such as coding assistants and analytics platforms, removing the choice between slow-booting VMs and containers that need custom kernel hardening.

Announcing Amazon EC2 G7 instances accelerated by NVIDIA RTX PRO 4500 Blackwell Server Edition GPUs (1 minute read)

AWS made general availability for EC2 G7 instances, the first major cloud offering of NVIDIA RTX PRO 4500 Blackwell Server Edition GPUs, delivering up to 4.6x AI inference and 2.1x graphics performance over G6. Each instance scales to 8 GPUs and 256GB of GPU memory, with 700 Gbps of EFA networking (7x G6) and 7.6TB of local NVMe to keep large models and datasets close to compute.

AWS debuts Lambda MicroVMs with up to 8 hours runtime (1 minute read)

AWS launched Lambda MicroVMs, extending the serverless runtime from a 15-minute cap to as long as 8 hours per invocation with hardware-isolated execution. The longer ceiling targets untrusted code, AI agents, and batch jobs that previously forced teams onto Fargate or EC2, though sustained 8-hour invocations shift the cost calculus closer to always-on compute than traditional event-driven Lambda.
