Cordyceps CI/CD Flaws Expose 300+ GitHub Repositories to Supply-Chain Attacks (1 minute read)

Novee Security disclosed Cordyceps, a class of CI/CD misconfiguration where untrusted pull requests trigger privileged workflows, and its scan of 30,000 high-impact repos found 300-plus fully exploitable by any unauthenticated user with a free account. Affected projects include Microsoft's Azure Sentinel, Google's adk-samples, Apache Doris, and Cloudflare's Workers SDK, where forged approvals or crafted branch names let attackers run code on CI runners and steal long-lived credentials.

Exploitable CI/CD Vulnerabilities Expose Millions of Repositories to Hijacking (1 minute read)

Security firm Novee disclosed Cordyceps, a class of CI/CD vulnerabilities in GitHub Actions YAML that lets unauthenticated attackers with a free account forge approvals, push code, and exfiltrate maintainer credentials across build tooling from Microsoft, Google, Apache, Cloudflare, and the Python Software Foundation. A single scan flagged 654 repositories and confirmed over 300 fully exploitable, with the agentic-coding boom reproducing the insecure workflow patterns that traditional scanners skip.
