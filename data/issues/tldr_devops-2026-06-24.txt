Linux and Secure Boot certificate expiration (2025) (1 minute read)

The Microsoft 2011 key used to sign the shim first-stage UEFI bootloader expires in September, after which Microsoft will stop signing the bootloader that Linux distributions rely on to boot under Secure Boot. A 2023 replacement key exists but is missing from many systems and often requires a vendor firmware update, so the breakage mainly hits new installs on Secure Boot hardware and adds work for distributors and users.

Decades-Old Squid Proxy Flaw ‘Squidbleed' Can Expose User Data (5 minute read)

Researchers at Calif.io disclosed Squidbleed (CVE-2026-47729), a Heartbleed-style memory leak in Squid proxy's FTP parser that has existed since 1997 and can leak other users' cleartext HTTP request data, including credentials, session tokens, and API keys. The flaw is patched in Squid 7.6 and version 8, and poses the most risk in shared proxy environments like corporate networks and public Wi-Fi, where disabling unused FTP support is the recommended mitigation.

How we found a bug in the hyper HTTP library (1 minute read)

Cloudflare engineers spent six weeks tracing why their rearchitected Images binding intermittently returned truncated payloads, 200-status responses that cut a two-megabyte image down to a few hundred kilobytes, only for larger files. The culprit was a race condition in the hyper Rust HTTP library governing how processed image data streamed back to the client. The fix was four lines of code, a reminder that edge-network rollouts surface failure modes invisible under normal load.

Kops: Safely Extending the eBPF Compilation Pipeline with Native Operations (1 minute read)

Kops is an extension interface that lets userspace compilers and kernel modules add new eBPF operations without touching the kernel core, keeping the trusted computing base minimal. Each operation ships as a verifier-checked proof sequence plus a native emit, the only per-operation addition to the TCB, with Lean 4 proofs confirming the two forms compute the same result. The authors' EInsn operations speed up eBPF microbenchmarks by up to 24% and production workloads by up to 12% on x86-64 and ARM64.

A Comparison of Kubernetes Compliance Standards and Configuration Scanners (1 minute read)

This paper benchmarks eight Kubernetes hardening guidelines into 79 configuration recommendations, then runs ten popular static config scanners against it.
