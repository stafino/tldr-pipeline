Miasma Malware Targets npm Packages and GitHub Actions in Supply Chain Attack (1 minute read)

Researchers tied a new wave of supply chain attacks to the Mini Shai-Hulud/Miasma/Hades malware family, which trojanized two dozen npm packages including LeoPlatform and RStreams, plus a Go module, after compromising a maintainer's npm token and pushing poisoned versions within a six-second window. The malware skips lifecycle hooks in favor of binding.gyp install-time execution, then drops a "Run Copilot" GitHub Actions workflow to steal CI/CD secrets and developer credentials.

Founding members including AWS, Anthropic, GitHub, Google, IBM, Microsoft, and NVIDIA aim to fund and coordinate fixes for small maintainer teams as AI shortens the window between flaw discovery and exploitation.

NVIDIA’s New AI Servers Run on Hotub Coolant and Don’t Need Evaporators (1 minute read)

NVIDIA's Rubin server architecture moves every heat-generating chip onto closed-loop water blocks that run at 45-55 C, eliminating the evaporative chillers that have made AI data centers a target for water-conservation critics. The bigger driver is cost: running that hot lets dry radiators reject heat to ambient air, which NVIDIA says saves roughly $4M a year in cooling for a 50 MW hyperscaler.

Cloudflare teams up with Chrome, Firefox, and Edge on a privacy-first anti-bot protocol (5 minute read)

Cloudflare, Mozilla, Google, Microsoft, and Shopify are co-developing PACT (Private Access Control Tokens), a protocol that issues anonymous tokens to prove a visitor is human or an authorized agent, replacing CAPTCHAs and forced logins without tracking browsing history. The push lands as Cloudflare Radar shows bots now drive roughly 58% of global HTTP requests against 42% from people, and the group plans to submit PACT for formal standardization.

How Cloudflare responded to the “Copy Fail” Linux vulnerability (5 minute read)

Cloudflare detailed how its Linux kernel patching cadence neutralized "Copy Fail" (CVE-2026-31431), a local privilege escalation bug, before public disclosure. An automated weekly build off LTS branches plus a four-week Edge Reboot Release cycle meant the fix was already deployed across most of its 6.12 fleet, with no exposure or service disruption. The retro argues that staying close to upstream LTS is the cheapest insurance against zero-day scramble.

Client-Side Load Balancing at a Million Requests Per Second (5 minute read)

Zalando's engineering team replaced their shared Skipper edge router with an in-process client-side load balancer for the million-requests-per-second internal fan-out behind PRAPI, their product read API serving 25 European markets. They layered on N-ring fade-in, occupancy-based bounded load, and AZ-aware routing with a latency health factor. Owning the routing decision cut cost, made the service resilient to the infrastructure beneath it, and finally exposed root causes that were neither Skipper nor PRAPI.

Privacy-Aware Infrastructure in the AI-Native Era: An Asset Classification Case Study (1 minute read)

Meta's engineering team described how it classifies data assets for privacy enforcement, where a field named "age" might mean a person's age or a cache TTL. The hybrid pattern builds rich context, then uses LLMs only for novel or ambiguous assets, distilling stable behavior into deterministic, versioned rules that humans review before promotion. The result keeps production enforcement low latency, replayable, and auditable while shrinking the LLM's runtime role over time.

Migrating from Proxmox to NixOS and Incus (1 minute read)

The author migrated a multi-node Proxmox homelab cluster to NixOS running Incus, arguing that a version-controlled declarative config should be the single source of truth so any host can be rebuilt byte-for-byte. The trade-off is Nix's steep learning curve, but the payoff is no imperative state drift from one-off debugging fixes that never make it back into the automation. A HN-front-page update concedes Proxmox supports `qm`, `pct`, REST, Terraform, and Ansible, narrowing the real argument to reproducibility over tooling.

Empirical Software Engineering TerraProbe: A Layered-Oracle Framework for Detecting Deceptive Fixes in LLM-Assisted Terraform (1 minute read)

TerraProbe is a five-layer oracle framework for evaluating LLM-assisted Terraform security repairs, applied to 288 fixes from Gemini 2.5 Flash Lite, GPT-4o, and Claude 3.5 Sonnet. The authors find that passing Checkov badly overstates success: targeted findings cleared 83.3% of the time, but full-scanner cleanliness hit 10.4% and 71.4% of plan-compared repairs were deceptive fixes leaving the vulnerability in place. Wildcard IAM grants persisted across all flagged cases, and the rate held across all three models.

Robinhood Cuts Access Approval Time to Support High-Velocity Development (1 minute read)

Robinhood's application security team built Secure Enhanced Remote Approval (SERA), a passkey-based system that orchestrates access approvals from any device without VPNs or managed laptops. Shipped earlier this year, it cut approval time 20% for developers and incident responders, removing a bottleneck that had slowed both incident response and feature rollout. The writeup covers the trade-offs of speeding up access grants without resorting to overpermissioning.

Building a state-of-the-art development platform with Backstage (2 minute read)

The author argues that Backstage solves the developer-portal problem but not the platform problem, since it organizes catalogs, docs, and golden-path templates while assuming an execution layer already exists beneath it. Wiring Backstage directly to CI/CD, GitOps, Kubernetes, and observability creates a fragile "messy middle" of custom integrations. The proposed fix is a control plane that compiles developer and platform abstractions into Kubernetes resources, enforces policies, reconciles drift, and feeds runtime state back to the portal.
