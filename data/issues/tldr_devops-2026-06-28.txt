WAL-RUS: a Rust Rewrite of WAL-G for PostgreSQL Backups (1 minute read)

ClickHouse released WAL-RUS, an open-source Rust rewrite of WAL-G that handles Postgres WAL archival and backups while keeping WAL-G compatibility. The rewrite targets memory predictability, replacing Go's garbage-collected sawtooth footprint with a tighter profile so operators can stop over-provisioning backup memory and hand it back to Postgres for shared buffers and page cache.
