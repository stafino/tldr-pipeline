etcd-operator joins Cozystack with a new v1alpha2 API (1 minute read)

The etcd-operator project was donated to Cozystack and re-implemented from scratch under a new etcd-operator.cozystack.io/v1alpha2 API, dropping the StatefulSet model in favor of driving etcd's native Membership API for MemberAdd, MemberPromote, and MemberRemove directly. The rewrite reconciles each member's Pod and PVC independently and adds learner-mode members one at a time, giving the operator full control over quorum changes during scaling and graceful removal.

OTel and mesh-derived metrics: A 2026 reference (1 minute read)

This CNCF post explains how to combine Linkerd's mesh-derived metrics with an existing OpenTelemetry pipeline, covering the east-west service traffic that application-layer instrumentation misses. Once a workload is meshed, the Linkerd proxy emits golden metrics for every inbound and outbound request on port 4191, with no SDK calls or image rebuilds. The reference shows where mesh metrics overlap with OTel and where they don't, plus a working Collector config and Grafana dashboard to land both layers in the same backend.
