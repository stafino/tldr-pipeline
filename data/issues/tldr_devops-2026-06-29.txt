AMD Strix Halo RDMA Cluster Setup Guide (1 minute read)

This guide walks through building a two-node AMD Strix Halo cluster wired with Intel E810 NICs over RoCE v2 to run distributed vLLM inference with Tensor Parallelism. The payoff is in the data plane: RDMA lets RCCL write tensors node-to-node directly from memory, cutting per-exchange latency from ~70-100µs over TCP to ~5µs. That matters because TP=2 forces a synchronization after every layer, thousands of times per second, so the kernel-bypass path is what makes two boxes behave like one.

What you need to know about Lambda MicroVMs (5 minute read)

AWS launched Lambda MicroVMs, giving teams Firecracker-grade isolation with no shared kernel, sub-second snapshot boots, and stateful sessions that persist memory and disk for up to eight hours. The author argues it finally fits the AI-code-sandbox problem that EC2, ECS, and Lambda functions each solved only partially. The model is per-session VMs with dedicated HTTPS endpoints, bearer-token auth, and automatic suspend on idle so you stop paying when traffic stops.

WAL-RUS: a Rust Rewrite of WAL-G for PostgreSQL Backups (1 minute read)

ClickHouse released WAL-RUS, an open-source Rust reimplementation of WAL-G for PostgreSQL WAL archival and backups, built for WAL-G compatibility and predictable memory usage in resource-constrained environments. Rust avoids Go's garbage-collected "sawtooth" memory profile and unpredictable virtual memory reservation, letting operators provision tighter backup budgets and reclaim memory for Postgres queries, shared buffers, and page cache.

Amazon ECS introduces new high-resolution metrics for faster service auto scaling (1 minute read)

Amazon ECS service auto scaling now supports high-resolution 20-second metrics, cutting the time to trigger scale-out from 363 to 86 seconds and total scale-and-provision time from 386 to 109 seconds in AWS benchmarks. Faster reaction lets teams trim baseline task counts and the preemptive capacity padding that hedges against traffic spikes, lowering compute cost while target tracking replaces the custom step-scaling policies that used to deliver aggressive scaling.
