Zuck saves Meta bucks by reusing memory from old servers with a custom CXL ASIC (1 minute read)

Meta built a custom CXL ASIC that pools DRAM salvaged from decommissioned servers, letting newer hosts borrow that memory over the CXL fabric instead of buying fresh DIMMs. The move targets memory cost and e-waste at hyperscale, extending the useful life of RAM that would otherwise be scrapped and easing capacity pressure on memory-bound workloads like caching and inference.

etcd-operator joins Cozystack with a new v1alpha2 API (1 minute read)

The etcd-operator project was donated to Cozystack and re-implemented from scratch under a new etcd-operator.cozystack.io/v1alpha2 API, dropping the StatefulSet model in favor of driving etcd's native Membership API for MemberAdd, MemberPromote, and MemberRemove directly. The rewrite reconciles each member's Pod and PVC independently and adds learner-mode members one at a time, giving the operator full control over quorum changes during scaling and graceful removal.

The reference shows where mesh metrics overlap with OTel and where they don't, plus a working Collector config and Grafana dashboard to land both layers in the same backend.

A Multi-Dataset Benchmark for Evaluating LLM Agents in Microservice Failure Diagnosis (1 minute read)

This paper introduces two expert-labeled datasets, AIOps2025 and RCA100, for benchmarking LLM agents on microservice failure diagnosis over multimodal observability data. Unlike prior benchmarks that score only the final answer, it grades the reasoning process along three axes: localization (where the fault is), identification (what type), and whether the trace is grounded in real evidence. The 500-plus cases span resource, network, runtime, database, and application-logic faults across HipsterShop and the OpenTelemetry Demo Store, and were validated through competitions with over 6,000 teams.

A native graphical shell for SSH (1 minute read)

Outer Shell is an open-source graphical shell that lets servers and edge devices serve browser-based apps over SSH, with each app running as a dependency-free HTTP server bound to a Unix domain socket rather than a localhost port. Encryption is handled at the SSH layer instead of per-app, and apps register capabilities so they can discover each other, turning a remote box into a navigable desktop without a local GUI.

WSL containers now build and run Linux workloads on Windows (1 minute read)

Microsoft shipped WSL containers in public preview (WSL build 2.9.3), adding a wslc.exe CLI and a C/C++/C# API so Windows developers can build, run, and debug Linux containers, including GPU workloads, without third-party tooling. Enterprise controls land alongside it, with Intune-managed allowlists for permitted container registries and Defender for Endpoint now reading Linux container events across distributions.

Apache Flink 2.3.0 Release Announcement (5 minute read)

The Apache Flink PMC released Flink 2.3.0, which adds SQL changelog operators (FROM_CHANGELOG and TO_CHANGELOG), a redesigned native S3 filesystem built on AWS SDK v2, and application-level lifecycle management across 15 implemented FLIPs. Adaptive partition selection and improved watermark alignment cut backpressure and speed backlog processing, while a reworked SinkUpsertMaterializer lifts performance for some SQL workloads in production deployments.

Introducing Package Proxy: Supply-Chain Safety Checks Without Client-Side Software (5 minute read)

Thinkst released Package Proxy, a Cloudflare Workers-based proxy that runs inline supply-chain checks for npm, pip, uv, and cargo without installing any client-side software on developer machines. The tool blocks packages that fall inside the risky window for known attack patterns, such as recently published versions tied to surreptitious back-doors and low-install typo-squats, and deploys from its GitHub repo straight into Cloudflare.

The "Akrites" vulnerability-mitigation project launches (5 minute read)

Akrites launched as a vulnerability-mitigation project that fixes flaws upstream and confidentially, measuring success by patch deployment rather than publication on the premise that public patches let adversaries reverse-engineer exploits before fixes land. The program keeps undisclosed flaws embargoed, routes fixes through each project's own maintainers, and commits to serve as maintainer of last resort for unmaintained but critical packages.

Google Antigravity agents get full context with GitLab Orbit (3 minute read)

GitLab released Orbit, a lifecycle context graph that Google Antigravity developers can install from the Antigravity MCP Store to give coding agents structured access to projects, pipelines, merge requests, vulnerabilities, and source across a GitLab instance. Orbit indexes the instance into a knowledge graph queried through two MCP tools, and GitLab's internal tests reported agents responding up to 11x faster, using 4.5x fewer tokens, and producing 45x fewer hallucinations.
