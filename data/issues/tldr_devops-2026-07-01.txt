Zuck saves Meta bucks by reusing memory from old servers with a custom CXL ASIC (1 minute read)

Meta built a custom CXL ASIC that pools DRAM salvaged from decommissioned servers, letting newer hosts borrow that memory over the CXL fabric instead of buying fresh DIMMs. The move targets memory cost and e-waste at hyperscale, extending the useful life of RAM that would otherwise be scrapped and easing capacity pressure on memory-bound workloads like caching and inference.

I ported Kubernetes to the browser (1 minute read)

A developer released webernetes, a partial reimplementation of Kubernetes in TypeScript that runs an entire cluster in the browser, handling pod lifecycles, cluster DNS, IP allocation, container garbage collection, and Deployment/ReplicaSet tracking in roughly 140KiB gzipped. Rather than compiling Kubernetes to WebAssembly, which would ship megabytes and fails on browser-unavailable syscalls, the author rebuilt the control-plane logic and swapped registries for a browser-native image API defined in TypeScript. The project spans nearly 100,000 lines across 629 files, written over two months.

OTel and mesh-derived metrics: A 2026 reference (1 minute read)

This CNCF post explains how to combine Linkerd's mesh-derived metrics with an existing OpenTelemetry pipeline, covering the east-west service traffic that application-layer instrumentation misses. Once a workload is meshed, the Linkerd proxy emits golden metrics for every inbound and outbound request on port 4191, with no SDK calls or image rebuilds. The reference shows where mesh metrics overlap with OTel and where they don't, plus a working Collector config and Grafana dashboard to land both layers in the same backend.

Core dump epidemiology: fixing an 18-year-old bug (1 minute read)

OpenAI's data-infrastructure team traced a batch of "impossible" C++ crashes in its Rockset service to two unrelated causes: silent hardware corruption on a single Azure host doing math wrong, and an 18-year-old race condition in GNU libunwind. Rather than chase individual stack traces, the team treated the crashes as a population and built a dataset across every occurrence, letting them separate the two signals. The write-up is a useful model for debugging rare, non-reproducible failures at fleet scale, where per-incident analysis breaks down.

How is AI changing datacenter network fabrics? (1 minute read)

The Register examines how AI training and inference workloads are reshaping datacenter network fabrics, driving the shift toward higher-radix switches, 800GbE links, and lossless RDMA-based topologies built for east-west GPU traffic. The piece argues that traditional oversubscribed leaf-spine designs buckle under synchronized all-to-all collectives, pushing operators toward rail-optimized fabrics and tighter congestion control. For platform teams, the takeaway is that network cost and topology now gate cluster utilization as much as GPU supply does.

A native graphical shell for SSH (1 minute read)

Outer Shell is an open-source graphical shell that lets servers and edge devices serve browser-based apps over SSH, with each app running as a dependency-free HTTP server bound to a Unix domain socket rather than a localhost port. Encryption is handled at the SSH layer instead of per-app, and apps register capabilities so they can discover each other, turning a remote box into a navigable desktop without a local GUI.
