16-Year-Old Linux KVM Flaw Lets Guest VMs Escape to Host on Intel and AMD x86 Systems (1 minute read)

A 16-year-old use-after-free bug in Linux KVM's shadow MMU, dubbed Januscape (CVE-2026-53359), lets a guest VM with root corrupt host kernel memory on both Intel and AMD, the first known guest-to-host escape working on both. The public proof-of-concept panics the host and takes every co-tenant VM down; a fix landed in mainline on June 19, and cloud operators should prioritize patching multi-tenant hosts.

Security researcher Hyunwoo Kim disclosed Januscape (CVE-2026-53359), a use-after-free bug in KVM's shadow MMU that allows guest-to-host VM escape on both Intel and AMD, the first KVM exploit to work on both architectures. It threatens multi-tenant public clouds running untrusted guests, letting a single rented instance panic the host or gain root RCE over every co-located tenant; it was patched in mainline on June 19 after 16 years dormant.

New Januscape Linux flaw allows VM escape on Intel, AMD devices (1 minute read)

Security researcher Hyunwoo Kim disclosed Januscape (CVE-2026-53359), a use-after-free bug in KVM/x86 shadow MMU emulation that lets an attacker with root inside a guest VM execute code on the host or panic the kernel, taking down every co-tenant VM on the same machine. It is the first guest-to-host escape that triggers on both Intel and AMD, and multi-tenant operators should confirm patch commit 81ccda30b4e8 is applied to their host kernels.

Threat Actors Probe Gitea Docker Flaw CVE-2026-20896 13 Days After Disclosure (1 minute read)

Sysdig detected the first in-the-wild exploitation attempt against CVE-2026-20896, a CVSS 9.8 flaw in Gitea's official Docker images that hard-code REVERSE_PROXY_TRUSTED_PROXIES to a wildcard, letting any client that reaches the container spoof an X-WEBAUTH-USER header and authenticate as admin. Gitea 1.26.3 removes the wildcard and makes reverse-proxy auth opt-in, but roughly 6,200 internet-facing instances remain exposed 13 days after disclosure.

Proof-of-Concept Exploit Released for Linux ‘Bad Epoll’ Root Access Vulnerability (1 minute read)

A proof-of-concept exploit is now public for Bad Epoll (CVE-2026-46242), a use-after-free race condition in the Linux kernel's epoll facility that lets unprivileged processes escalate to root on distros running kernel 6.4 or newer, including Pixel 10 phones. The PoC leaks kernel memory and hijacks an indirect call via a ROP chain, and a correct maintainer patch landed only two months after the initial report.

Automatic Upgrades: best practice features for your lakehouse tables (1 minute read)

Databricks released Auto Upgrades for Unity Catalog managed tables, which observes client access patterns over a rolling window, verifies workload compatibility, and then applies best-practice features like Row Tracking via a background ALTER TABLE job with no manual effort. Every change is reversible per table, and once all existing tables in a schema pass verification the feature becomes a default that new tables inherit automatically.

Meta: Recycling memory is magic solution to RAM supply misery (1 minute read)

Meta detailed Vistara, a CXL-based ASIC that exposes DDR4 modules pulled from decommissioned servers as an extra NUMA node on newer DDR5 hosts, letting memory-capacity-bound machines reuse RAM that outlives the servers it shipped in. In Meta's numbers the bridge cut server count by up to 25% for disaggregated ML inference, dropped distributed-cache latency 29%, and reduced job-failure frequency 33%, validating CXL against the AI-driven memory crunch.

Can LLMs Really Recover Microservice Failures? A Recovery-Aware Evaluation of Diagnosis-to-Action Reasoning (1 minute read)

The gap sits in translating evidence into an admissible fix, which is the part that matters for autonomous incident response.

Beyond Compliance: A Large Scale Study on the Completeness and Consistency of the GitHub SBOMs (1 minute read)

This paper evaluates GitHub's auto-generated SBOMs against Syft, Trivy, and the Microsoft SBOM Tool across 10,000 repositories spanning ten language ecosystems. GitHub's SBOMs fail NTIA compliance but reliably carry core metadata, and they match the Microsoft tool while often beating Syft and Trivy on version and license coverage. The catch for anyone using SBOMs for vulnerability or license analysis is that completeness swings hard by ecosystem, driven by how each language manages dependencies.

Event-driven plumbing. When plain SQS + Lambda beats EventBridge Pipes (3 minute read)

The author argues that for SQS-to-Lambda-to-SQS enrichment, plain SQS plus Lambda beats EventBridge Pipes, which charge $0.40 per million requests and bill every 64KB payload chunk as a separate request. Pipes earn their keep for low-code HTTP API enrichment on an event stream, but when your enrichment is already a Lambda, you pay in both money and lost failure granularity for handling your serverless primitives do better. The post walks through a multi-million-document S3 enrichment job to make the case.

Configure Azure Pipelines app in ghe.com (5 minute read)

A developer documenting how to install the Azure Pipelines GitHub app on ghe.com (GitHub Enterprise Managed Users with Data Residency), a step Azure DevOps's service-connection UI assumes exists but never explains. The fix is an undocumented URL, `https://{slug}.ghe.com/apps/external-app/azure-pipelines`, reverse-engineered from the Azure Boards app path after GitHub and Microsoft contacts came up empty. From there the service connection authorizes cleanly and pipelines rewire from Azure Repos to GitHub.

Migrating Counter Service storage: Design choices and learnings (5 minute read)

Grab's anti-fraud team migrated its Counter Service, which handles ~1 billion requests per day at tens of thousands of QPS, off a wide-column store onto Aerospike while decoupling storage from business logic. The writeup covers the reader-side safety scaffolding built for a gradual rollout, a redesigned writer-side data model that collapsed the original four network round-trips per event, and the optimizations surfaced by revisiting access patterns. The decoupling is framed as the reusable payoff, cutting the effort for any future storage swap.

Your Scanner Doesn't Know What Time It Is (3 minute read)

The author argues that cloud security scanners produce compound findings by splicing config snapshots captured minutes apart, asserting a system state that may never have existed at any single moment. A scanner reporting "this IAM role can reach 47 S3 buckets" can be wrong if the policy changed inside the collection window, and nothing flags the inconsistency. The post applies Lamport's 1978 causal-ordering work to argue scanners need consistent-snapshot semantics, not just point-in-time reads.

DNSGlobe – Rust TUI to watch DNS propagate around the world (1 minute read)

DNSGlobe is a Rust terminal UI that queries 34 public DNS resolvers worldwide in parallel, compares their answers, and renders record propagation on a live world map with a watch mode that re-polls until every server agrees.
