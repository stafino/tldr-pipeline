GhostLock, a stack-UAF that has existed in ALL Linux distributions for 15 years (1 minute read)

Security researchers at VEGA disclosed GhostLock (CVE-2026-43499), a stack use-after-free in the Linux kernel's rtmutex PI code that has shipped in every major distribution since Linux 2.6.39 in 2011 and requires no special config or privileges to trigger. The team turned it into a 97% stable local privilege escalation and container escape, earning $92,337 through Google's kernelCTF, and urge unpatched hosts to move to the latest LTS.

Hackers exploit critical auth bypass in Gitea Docker image (1 minute read)

Attackers are actively exploiting CVE-2026-20896, a critical auth bypass in Gitea's official Docker image that ships `REVERSE_PROXY_TRUSTED_PROXIES=*`, letting any client that reaches the container's HTTP port set an `X-WEBAUTH-USER` header and impersonate any user, including admins. Gitea released 1.26.3 and 1.26.4 to fix it, and operators who can't upgrade should scope the trusted-proxies setting to specific reverse-proxy IPs across the roughly 6,200 exposed instances.

Six New U-Boot Flaws Could Let Malicious Images Crash Devices or Run Code at Boot (1 minute read)

Binarly disclosed six flaws in U-Boot's FIT image verification, two of which let a malicious image run attacker code before the bootloader checks its signature, and four that crash the device. Most of the vulnerable code has shipped since v2013.07 across 50-plus stable releases and the many vendor firmwares built on U-Boot, from home routers to data-center BMCs, and no CVEs have been assigned yet.

An update on residential proxies and the scraper situation (1 minute read)

LWN's follow-up on the AI scraper problem reports that request floods now span millions of unique residential and mobile IPs, each hitting a site two or three times with fabricated user-agent strings before rotating away. The traffic comes from malware-infected home devices, increasingly media-streaming boxes, coordinated by command-and-control nodes, which makes IP-based blocking useless. Google's takedown of the IPIDEA proxy network cut LWN's scraper load for a few months, but the reprieve has already ended.

Improving Smart Tiered Cache for Public Cloud Regions (1 minute read)

Cloudflare shipped Smart Tiered Cache for Public Cloud Regions, letting customers pass a cloud region hint so Cloudflare can map anycast or regional-unicast origins to the right region and pick optimal primary and fallback upper tiers. This restores single-closest-tier cache efficiency for public cloud origins, where ambiguous IPs previously forced a fallback to multiple upper tiers and eroded hit ratios.
