Cloudflare WAF protects WordPress applications from two high-severity vulnerabilities (1 minute read)

Cloudflare deployed WAF rules protecting all plan tiers, including free, against two high-severity WordPress flaws: an unauthenticated RCE in the REST API batch endpoint (CVE-2026-63030, critical) and a related SQL injection (CVE-2026-60137, high). OpenSSL HollowByte Flaw Could Freeze Server Memory with 11-Byte TLS Requests (1 minute read)

Okta's Red Team disclosed HollowByte, a denial-of-service flaw where an unpatched OpenSSL server allocates up to 131 KB per TLS handshake header before any body arrives, and glibc's chunk-caching keeps that memory frozen after the attacker disconnects. OpenSSL patched it quietly on June 9 with no CVE or advisory, so scanners can't flag it, and Okta's NGINX tests froze 25% of a 16 GB server without hitting connection limits.

Learning a few things about running SQLite (1 minute read)

Julia Evans documents operational lessons from running SQLite in production behind a Django site, starting with the discovery that a 4000-row FTS5 query dropped from 5 seconds to 0.05 seconds after running ANALYZE to give the query planner table statistics. The write-up also covers a failure mode where slow bulk DELETEs hold a transaction long enough that concurrent workers hit the 5-second busy timeout and crash, a reminder that "SQLite is fine for small sites" still means operating a real database.
