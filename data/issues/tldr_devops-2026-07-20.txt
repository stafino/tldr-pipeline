Malicious cloud customers can bring down the power grid (1 minute read)

Researchers at Zhejiang University showed that malicious cloud tenants can weaponize normal GPU training workloads, timing power swings at synchronization points like checkpointing to push oscillations past 6,000 Hz and match critical grid frequencies. Their Bit2Watt attack drove 46.8% total harmonic distortion and a negative damping ratio on a simulated 1-MW grid, and stays covert because it runs entirely inside authorized workload paths that cloud-provider monitoring is unlikely to flag.

Branching databases like code: a CI/CD pattern for Lakebase, in production at Glaspoort (1 minute read)

Glaspoort, a Dutch fiber operator, documented how it ships schema changes to Lakebase, the serverless Postgres OLTP that backs its project-management app, by branching a production-like database for every PR. Each pull request spins up an ephemeral, isolated copy so reviewers can test against fresh data without stepping on each other, with migrations held as the single source of truth. The trade-off is more short-lived databases to provision and reap, absorbed by treating them as disposable IaC alongside the app's CI/CD.

Google Cloud added readiness probes and service health aggregation to Cloud Run, giving multi-region services instance-level health checks that automatically fail traffic away from unhealthy regions within seconds via a global external load balancer. The capabilities cut the manual incident response that regional failover previously required, and service health works with both single- and multi-region deployments across public internet and VPC-internal traffic.

Cloudflare Internal DNS is now generally available (1 minute read)

Cloudflare made Internal DNS generally available, providing authoritative and recursive private DNS on the same global network and control plane customers already use for public DNS and Zero Trust. It runs on the infrastructure behind 1.1.1.1 with no appliances to rack, folds split-horizon resolution into shared zones with per-view resolver policies, and comes at no additional charge for Enterprise customers on Cloudflare Gateway.
