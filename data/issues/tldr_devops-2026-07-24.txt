BGP ORIGIN attribute manipulation and its impact on the Internet (1 minute read)

Cloudflare found that ~70% of observed BGP paths carry a different ORIGIN attribute than the originating AS set, despite the spec saying no downstream router should modify it. Because path selection prefers the lower ORIGIN value when Local Preference and AS_PATH length tie, this widespread manipulation quietly reshapes how traffic is forwarded across the Internet.

Microsoft blames massive Microsoft 365 outage on maintenance bug (1 minute read)

Downdetector logged 2,403 reports at peak against a baseline of 29, and Microsoft resolved incident MO1437424 by reverting the networking change roughly four hours later.

CIOs beware: DNS KSK rollover could kick off wave of mysterious outages (3 minute read)

The root zone's Key Signing Key rollover begins signing on Oct. 11, 2026, with the old key retired Jan. 11, 2027, and analysts warn the transition could surface a wave of hard-to-trace DNSSEC resolution failures. The exposure sits in the systems that bypass platform standards, such as hardcoded resolvers, stale container base images, sidecar configs, and untouched partner integrations, where no one has thought to ask vendors about DNS updates.

OpenTelemetry has graduated… Now what? (1 minute read)

OpenTelemetry reached CNCF graduated status, joining Kubernetes and Prometheus and cementing its position as the de facto open standard for traces, metrics, and logs. The project is now the second-highest velocity effort in the CNCF with over 12,000 contributions from 2,800 companies, and graduation signals a stable instrumentation layer that lets teams swap observability vendors without rewriting code.

Default Azure Automation Setting Enables Cross-Tenant Identity Takeover (1 minute read)

Microsoft patched CVE-2025-29827, a CVSS 9.9 flaw in Azure Automation where a public-by-default identity setting let an attacker with their own automation account cross the tenant trust boundary and assume another tenant's managed identity. That access could expose credentials and configuration data and let the attacker create, modify, or delete resources across a victim's cloud workloads, though Microsoft reports no known exploitation and has since changed the default to private.

Cloudflare Internal DNS puts public and private DNS on one policy engine (2 minute read)

Cloudflare launched Internal DNS, which extends its public DNS platform to private zones so both run on one control plane, one API, one audit trail, and a single policy engine. Cloudflare Gateway acts as the resolver ahead of every lookup, evaluating zero-trust policies before routing queries to the right DNS view based on source IP, device posture, or network location, with internal zones never assigned public nameservers.

Microsoft explains why its West US Azure and cloud services failed (1 minute read)

Microsoft published a preliminary post-incident review for its July 23 West US outage, attributing five hours of lost connectivity to automated maintenance systems that pulled IP routes from devices outside the planned isolation scope. Traffic entering or leaving the region failed between 14:44 and 19:41 UTC, marking the second major Azure disruption this year, and Microsoft is now advising mission-critical workloads to adopt a multi-region architecture.

ModelExpress: Distributing Model Artifacts at the Speed of Light (1 minute read)

NVIDIA detailed ModelExpress, a system that cuts model-weight startup time by asking whether a compatible copy of the weights already lives on a serving peer before falling back to object storage. When a peer holds the weights in GPU, ModelExpress transfers them GPU-to-GPU over P2P RDMA via NIXL, skipping redundant reads from S3, local disk, and host memory. It moved DeepSeek-V4 Pro weights into a fresh replica in under 10 seconds, dropping cold-start time from 8 minutes to 1 minute 44 seconds.
