RefluXFS: A Linux Kernel Local Privilege Escalation to Root in XFS (CVE-2026-64600) (5 minute read)

Qualys disclosed CVE-2026-64600, a race condition in the Linux kernel's XFS copy-on-write path that lets any local unprivileged user overwrite protected files at the block layer and escalate to host root, even under SELinux Enforcing. The flaw has existed since kernel 4.11 and ships in default RHEL, Oracle Linux, Amazon Linux, and Fedora installs, with no reliable mitigation short of patching and rebooting.

When Kubeflow meets Cilium: Debugging 60% idle GPUs in Kubernetes (1 minute read)

The authors trace a class of Kubernetes failure where GPUs sit idle despite green health checks: Kubernetes and Kubeflow schedule pods topology-agnostically, while a CiliumNetworkPolicy drawn for zone isolation silently blocks the training coordinator from reaching workers in another availability zone. Because each system makes an individually correct decision, no single view shows both pod placement and the network boundary. The symptom ranges from an outright connection block to NCCL AllReduce paying cross-zone latency on every gradient sync.

Characterizing Metastable Faults and Failures (5 minute read)

A June 2026 paper offers the first analytical causal account of metastable failures, the self-sustaining goodput collapses that persist through retry and queue feedback loops even after the original trigger clears. It reframes a metastable fault as a "sin of composition" between individually stable components and ties the recovery mechanism to scheduling. The stakes are concrete: four of AWS's 15 major outages over a decade were metastable, lasting 1.5 to 73 hours, and these emergent failures resist the hardening that already covers obvious faults.

SequenceFI: Non-intrusive Temporal Fault Injection for Microservice Systems (1 minute read)

This paper introduces SequenceFI, a non-intrusive framework that adds timing control to microservice fault injection, triggering faults only when temporal guards on observed message events are met. It targets timing-dependent failures that where-and-what tools miss: faults after state-changing side effects, order-sensitive concurrent responses, and partial failures across repeated downstream calls. On four Kubernetes benchmarks, it hit 100% temporal accuracy and cut end-to-end search time 95.9% versus H-Random, requiring no application code changes.
