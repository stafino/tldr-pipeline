GhostTree Attack Abused Recursive Windows Junctions to Hide Malware (1 minute read)

Researchers disclosed GhostTree, a Windows evasion technique that abuses NTFS junctions pointed at their own parent directory to create recursive loops generating effectively infinite file paths, causing recursive scanners and some EDR products to hang before they ever examine malware in the folder. Junction creation requires only write access and no admin privileges, so defenders should constrain directory recursion depth and flag reparse points that loop back on their own ancestors.

OptinMonster WordPress plugin hacked in CDN supply-chain attack (1 minute read)

Attackers compromised Awesome Motive's CDN by exploiting a known UpdraftPlus flaw on a marketing server holding CDN credentials, then used the stolen API key to inject malicious JavaScript into OptinMonster, TrustPulse, and PushEngage plugin files served to over 1.2 million WordPress sites. The script ran when admins browsed infected pages, stealing auth tokens to create rogue admin accounts and install a self-hiding backdoor with a web shell and PHP execution.

Defenders monitoring network traffic see only outbound connections to genuine Microsoft servers, while the attackers used DLL sideloading and BYOVD drivers to disable security tools and dwelled one to two months.

SimpleHelp RMM flaw could give attackers full access to managed endpoints (CVE-2026-48558) (1 minute read)

SimpleHelp patched it in v5.5.16 and v6.0 RC 2 in late May; Horizon3.ai found roughly 7.2% of the nearly 14,000 internet-exposed servers run the vulnerable OIDC config, so patch and audit Technician accounts now.

Windows version of SprySOCKS Linux malware used to attack govt orgs (1 minute read)

Users cry foul after AMD stripped memory crypto from its consumer CPUs (1 minute read)

AMD silently removed Transparent Secure Memory Encryption (TSME) from its consumer Ryzen CPUs, leaving DRAM contents unencrypted and exposing systems to cold-boot and physical memory-extraction attacks that the feature was meant to mitigate. The change was undocumented and appears deliberate, so defenders relying on at-rest memory encryption for laptops or kiosks should verify TSME support per SKU rather than assume parity with prior generations.

The Proxy Knows Too Much: Sealing LLM API Routers with Attested TEEs (1 minute read)

This paper introduces AEGIS, an attested API router that confines LLM plaintext handling to a hardware enclave, blocking malicious routers that would otherwise act as application-layer men-in-the-middle (rewriting tool calls, swapping in typosquatted dependencies, or exfiltrating secrets). Clients verify the enclave before releasing plaintext, and the host can neither read nor alter traffic. The trusted path is 851 lines, carries three provider-native APIs, and adds roughly six milliseconds of relay overhead per request.

Cross-Silo De-Anonymization Under Local Differential Privacy: Threat Model, Phase Transition, and Coordination Necessity (1 minute read)

This paper models when an attacker can re-identify a person whose records sit in k independent data silos, each protected by local differential privacy. The authors prove a phase transition at k* = Θ(log n / ε²): below it any estimator fails, above it de-anonymization is inevitable for non-coordinated randomized-response mechanisms. The practical takeaway is that per-silo DP guarantees do not bound joint re-identification risk, so defenders must coordinate privacy budgets across silos rather than treat each in isolation.

AI Agent Skill Managers Are a Supply-Chain Surface Most of Them Don't Guard (1 minute read)

Three open-source AI agent skill managers, which install the SKILL.md packages that assistants like Claude Code and Codex read and execute, each passed 2,000 GitHub stars in recent months. Because a skill is natural-language intent run with full file and shell access, every manager is an unguarded software supply-chain channel, yet only one of the three scans skill files for malicious instructions before an agent sees them. Treat third-party skills like any untrusted dependency: review the source, and pin versions before agents run them.

Automated jailbreak attack targeting multiple defense strategies (1 minute read)

This paper introduces UNIATTACK, a black-box framework that builds one-shot jailbreak prompts by extracting high-impact features from existing attacks and refining them with an attacker LLM. Against models running multi-layered defenses, it raises attack success rates by 64.63% to 248.82% over baselines at 0.03% to 4.96% of their cost. The feature-centric, transferable templates generalize across multiple models and safety categories, giving red teams a cheap way to stress-test layered LLM guardrails.
