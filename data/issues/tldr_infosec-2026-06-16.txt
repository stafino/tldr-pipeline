Strategies & Tactics

Transferable Self-Evolving Playbooks for Agentic Security Auditing (1 minute read)

EvoHunt automates audit playbook creation for LLM security agents, replacing human-curated prompts and heuristics with a closed evolution loop: an audit agent runs the playbook, an evaluator scores findings against ground truth, and a reviser commits updates. The authors show evolved playbooks transfer the audit procedure to weaker agents, raising their vulnerability-discovery capability without retraining the underlying model. For defenders, this points toward scalable agentic auditing where procedural knowledge, not just model strength, drives results.

Di5Guise: 5G Privacy with vSIM (1 minute read)

This paper introduces Di5Guise, a 5G privacy architecture that decouples device identity from subscriber identity to stop network-level correlation attacks. Current eSIMs bind every provisioned subscriber profile to a fixed device profile (secret key, certificate, and eUICC ID), letting an operator-level adversary tie rotating identities back to one device and reconstruct a full pattern of life. Its vSIM design breaks that binding at the root, so identity rotation and traffic obfuscation actually hold.

From Third-Party to First-Party: Measuring and Protecting Against Modern Web Tracking Mechanisms (1 minute read)

A new study measuring top sites found that over 54% now deploy first-party tracking (FPT) or server-side tracking (SST), relocating tracking logic into the browser's first-party context or the site's backend to evade client-side detection. The authors introduce a provider-independent detection method and use script-similarity clustering to map the ecosystem, giving defenders and privacy tooling a way to spot tracking that traditional blocklists and client-side tools now miss.

SoK: Taxonomizing the Low-Level Attack Surface of Modern Web Browsers (1 minute read)

This paper systematizes the low-level attack surface of modern web browsers, mapping where memory-corruption flaws arise across input classes and components to expose regions underexplored by current fuzzing and testing. The authors structure the surface along input types and browser components, then chart how memory-corruption bugs cluster within that taxonomy. For AppSec and red teams, it offers a defense-oriented map of which browser subsystems warrant deeper security testing rather than continued effort on already-saturated targets.

Measurement Study of Post-Quantum Readiness of Internet: 2026 (1 minute read)

A measurement study of 32,011 domains found that 49.3% now support hybrid post-quantum key exchange such as MLKEM768 with X25519, but 15.70% still negotiate only TLS 1.2, concentrated in banking and government. The authors show post-quantum adoption is uneven across sectors, leaving the most sensitive infrastructure least prepared for harvest-now-decrypt-later attacks. Defenders in lagging sectors should prioritize TLS 1.3 migration and enable hybrid key exchange before classical ECC and RSA become quantum-breakable.

How memory safety CVEs differ between Rust and C/C++ (1 minute read)

This analysis compares memory-safety CVEs across Rust and C/C++ codebases, finding that Rust's safety guarantees push most remaining memory bugs into `unsafe` blocks or upstream C dependencies rather than safe Rust itself. For security teams weighing a migration, the operational takeaway is to focus auditing effort on `unsafe` boundaries and FFI surfaces, where Rust's compiler protections no longer apply, rather than treating the language as eliminating memory-corruption risk outright.

SPARK: Security Knowledge Priming and Representation-Guided Knowledge Activation for LLM-based Secure Code Generation (1 minute read)

SPARK is an inference-time security harness that reduces LLM-generated code vulnerabilities without retraining, arguing the security knowledge is already latent in pretraining and only needs activation. It retrieves relevant CWE entries per coding task and appends a short structured cue to the prompt, surfacing the model's existing safety representations. The approach sidesteps the compute overhead and data bias of fine-tuning or heavy retrieval-augmented defenses.

The Anatomy of Scam Scenarios: Large-Scale Characterization and Conversation-Aware Detection (1 minute read)

This paper analyzes 102,054 real-world scam conversations to build a hierarchical taxonomy of 18 fine-grained scam scenarios grouped into 6 high-level tactics, characterizing the psychological techniques each scenario systematically deploys. The authors then use this scenario-level mapping to build a conversation-aware detection model for financial-institution customer interactions, flagging active scams mid-conversation so banks can warn and intervene before funds move.

SkillVetBench: LLM-as-Judge for Multi-Dimensional Security Risk Evaluation in Open-Source LLM Agent Skills (1 minute read)

SkillVetBench is a public Hugging Face leaderboard that uses an LLM-as-Judge to vet community-contributed LLM agent skills, targeting instruction-layer and multi-agent risks that code-level scanners structurally miss, such as prompt-injection hijacks, encoded data exfiltration, and harm chained across pipelines. It scores each skill with SARS, a five-dimensional agentic-risk metric, and integrates full CVSS v4.0 vector decomposition plus a side-by-side view of the LLM verdict against the official marketplace rating.

Understanding Scam Trends and Rail Paths from Reddit Self-Disclosure Narratives (1 minute read)

DoubtProbe: Black-Box Jailbreak Defense via Structural Verification and Semantic Auditing (1 minute read)

DoubtProbe is a black-box jailbreak defense that reframes detection as consistency checking under controlled transformation, on the observation that most jailbreaks reorganize a harmful goal rather than remove it. A structural branch reconstructs the request from a normalized representation while a semantic branch audits intent, flagging inputs whose meaning shifts across transformation. The approach targets evolving prompt packaging and structure manipulation that known-attack coverage and prompt-level judgment tend to miss.

A data-driven security quantification framework for IoT-based systems (1 minute read)

This paper introduces a data-driven framework for quantifying cybersecurity risk in safety-critical IoT systems, replacing subjective expert scoring in Attack Tree Analysis with empirical vulnerability data. The authors model system architecture in SysML, derive attack trees from it, and assign exploitation probabilities to Basic Attack Steps from real-world vulnerability datasets. The approach aims to make IoT risk quantification more reproducible across heterogeneous architectures, where probabilistic exploitability data is typically scarce.

A Formal Resilience Framework for Cyber-Physical Embodied Systems under Device-Level Cyberattacks (1 minute read)

This paper proposes a formal dependability framework for embodied cyber-physical systems that folds IDS telemetry into resilience-evaluation predicates, letting defenders assess tolerance to disruption and degradation rather than relying on sensor and actuator drift detection alone. Traditional fault-tolerance models miss the subtle, strategic disruptions cyberattacks cause, which is critical when devices must preserve physical integrity. The authors' approach enables structured reasoning about how attacks affect task execution and supports proactive response before structural damage occurs.

Phantoms and Disclosures: a Causal Framework for Auditing Synthetic Data (1 minute read)

This paper introduces an empirical auditing framework for detecting when synthetic data leaks private records from its training corpus. It distinguishes "true disclosures," where the system reproduces a user's actual data, from "phantom disclosures," where matching records appear incidentally, by partitioning inputs into training and holdout sets and applying statistical hypothesis testing against zero-learning or differential-privacy baselines. The method requires no model access and no canary insertion, making it usable for auditing third-party synthetic datasets.

The Art of Mixology: Mixup-based Obfuscation for Privacy-Preserving Split Learning in Large Language Models (1 minute read)

This paper introduces MIXGUARD, a mixup-based framework for privacy-preserving split learning that lets resource-constrained users train LLMs by offloading heavy layers to a server while keeping raw data local. Robust and Automated Reconfiguration of Byzantine Wide-Area Replication (1 minute read)

The authors propose Beware, a framework that filters fabricated reports, computes robust weight distributions, and uses machine learning to converge on Byzantine-resilient configurations.

did:crdt: Coordination-Free Decentralised Identifiers via Signed CRDTs (1 minute read)

By the CALM Theorem the state-merge path is confluent and the signed-delta path needs only causal delivery, eliminating the fees, latency, and coordination that blockchain-anchored and Sidetree methods impose.
