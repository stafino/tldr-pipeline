GhostTree Attack Abused Recursive Windows Junctions to Hide Malware (1 minute read)

Researchers disclosed GhostTree, a Windows evasion technique that abuses NTFS junctions pointed at their own parent directory to create recursive loops generating effectively infinite file paths, causing recursive scanners and some EDR products to hang before they ever examine malware in the folder. Junction creation requires only write access and no admin privileges, so defenders should constrain directory recursion depth and flag reparse points that loop back on their own ancestors.

ShellGames: Speculative LLM-Driven SSH Deception (1 minute read)

ShellGames is an LLM-based SSH honeypot that aims to sustain credible, long-lived attacker sessions where prior LLM deception falls short. It combines chain-of-thought prompting, memory for state coherency, speculative command execution to cut latency, sandbox routing for complex commands, and subversion detection, reporting 0.898 command accuracy, 0.918 sequence consistency, and 0.95 robustness against probing. A 20-person user study rated its realism comparable to a real shell and ahead of traditional honeypots on perceived command coverage.

OTRO: Oblivious Tokenization Path with Square-Root ORAM (1 minute read)

Researchers introduced OTRO, an oblivious tokenization path that closes a side-channel gap in confidential LLM serving, where prior work recovered full user prompts from CPU tokenizer memory-access patterns on production Intel TDX. Standard PathORAM mitigations impose roughly 13x tokenizer slowdown and 10-58% higher time-to-first-token. OTRO uses replicated square-root ORAM instances with epoch-based rotation and KV-cache-aware tokenization to cap TTFT overhead at 4.5% while reducing observable leakage, a practical option for teams running LLMs inside CPU and GPU TEEs.

SNAS: A Multi-Layer Defense-in-Depth Architecture for Secure Egress in Sandboxed Workloads (1 minute read)

This paper presents SNAS (Secure Network Access in Snowpark), a production architecture for policy-driven egress control from sandboxed UDF workloads while preserving multi-tenant isolation. SNAS combines eBPF packet filtering, GENEVE overlay networks, and distributed egress proxies, plus an eBPF bandwidth limiter using the Earliest Departure Time algorithm and safeguards against connection limits and port exhaustion. It is deployed across all Snowflake regions, supporting petabyte-scale transfers and latency-sensitive external integrations.

PARSE: Provenance-Aware Retrieval Sanitization for Professional Domain LLM Agents (1 minute read)

PARSE, a new prompt-injection defense, argues that paraphrasing and other defenses validated on synthetic benchmarks fail on real enterprise documents, where authority language and factual content interleave. Tested on 122 tasks across financial, legal, medical, scientific, and DevOps domains using real SEC filings, Federal Register rules, and GitHub postmortems, PARSE cut attack success from 25.4% to 15.6% at 86.9% utility by classifying sentences by injection likelihood and verifying fact preservation. The authors urge teams to evaluate defenses on domain-matched real documents, not synthetic proxies.

An AI Security Agent for Banking: Multi-Vector Fraud and AML Detection Across Retail and Corporate Accounts (1 minute read) [flag]

This paper presents an AI security agent for retail and corporate banking that fuses LSTM behavioral models, velocity monitors, and graph-based counterparty analysis across parallel transaction and session streams to catch both signature-based fraud and behavioral crimes like BEC and AML layering. On a synthetic log of 237,669 transactions and 113,508 sessions, it hit F1 of 0.787 (transaction) and 0.867 (session), beating rule-based (0.562/0.733) and LSTM-only (0.655/0.713) baselines, with Critical-tier response under 0.43 ms at p95.

SoK: AI-Augmented Binary Reversing (1 minute read)

This paper systematizes a decade of AI-augmented binary reversing, analyzing 144 papers published since 2015 and organizing them into 22 reversing domains spanning vulnerability discovery, malware investigation, and firmware auditing. The authors introduce a unified taxonomy connecting traditional analysis techniques, binary-derived artifacts, learning paradigms, and the emerging roles of LLMs and agentic systems. For reverse engineers, it maps where ML approaches are maturing and where persistent evaluation gaps still undercut reliability.

A Red-Team Study of Anthropic Fable 5 & Opus 4.8 Models (1 minute read) [flag]

A red-team study using the HackAgent framework tested Anthropic's Fable 5 and Opus 4.8 against four jailbreak families across 7,826 harmful intents, with every apparent success re-adjudicated by a three-judge majority vote. Both models neutralize static obfuscation but remain vulnerable to adaptive iterative attacks: tree-of-attacks broke Opus 4.8 on 11.5% of intents and Fable 5 on 6.1% worst-case, yielding 1,620 and 702 panel-confirmed harmful completions. The authors conclude that even hardened frontier models stay reliably breakable under sustained automated pressure.

AI Agent Skill Managers Are a Supply-Chain Surface Most of Them Don't Guard (1 minute read)

Three open-source AI agent skill managers, which install the SKILL.md packages that assistants like Claude Code and Codex read and execute, each passed 2,000 GitHub stars in recent months. Because a skill is natural-language intent run with full file and shell access, every manager is an unguarded software supply-chain channel, yet only one of the three scans skill files for malicious instructions before an agent sees them. Treat third-party skills like any untrusted dependency: review the source, and pin versions before agents run them.

Safety, Security, and Cognitive Risks in Neuro-Symbolic AI (1 minute read)

This paper maps the attack surface of neuro-symbolic AI, which pairs neural perception with symbolic reasoning, across five layers and extends MITRE ATLAS with 11 NeSy-specific tactics. The authors show that symbolic-layer attacks are both potent and stealthy: single-axiom OWL edits hit 93.3% symbolic-integrity-violation success while staying 100% consistent under Pellet, yet held-out STIX detection performs at random-guessing level. Defenders adopting explainable hybrid systems should treat knowledge graphs and ontologies as exploitable assets requiring integrity controls.

Bifrost: Hybrid TEE-FHE Inference for Privacy-Preserving Transformer and LLM Serving (1 minute read)

Bifrost is a hybrid TEE-FHE architecture for confidential LLM inference that provisions secrets only to an attested CPU TEE, keeping the accelerator, drivers, and host stack outside the trusted computing base. It uses FHE for ciphertext-only delegation of linear layers while running non-linear, KV-state, and refresh operations inside the TEE. The authors report 9.25x to 9.91x projected latency cuts on GPT-2 and LLaMA 3, with a Bifrost+ prefill/decode split reducing TTFT up to 53.4x.

Evaluating Open-Source LLMs for Multi-Label ATT&CK Technique Classification on CTI Reports (1 minute read)

This paper benchmarks seven open-source LLMs (8B to 236B parameters) on multi-label MITRE ATT&CK classification using 2,076 human-annotated sentences from 83 complex, unstructured CTI reports. The best model scored only 0.22 micro-averaged F1, with larger models correlating to higher scores but prompt strategy and temperature showing no significant gains. The authors conclude current open-source LLMs are insufficient for production-grade ATT&CK tagging, and release the dataset as a reproducible baseline.

Verifying the Rust Standard Library (1 minute read)

A crowdsourced campaign integrated formal-verification tools into the CI of a fork of the Rust standard library, targeting the unsafe code that underpins memory safety but had only been validated through testing and Miri's dynamic checks. The authors report machine-checked proofs ruling out a subset of undefined behaviors, including out-of-bounds access, null and dangling pointer dereferences, and use of uninitialized memory, and frame the unproven remainder as open challenges for the formal-methods community.

Differential Privacy of Gaussian Process Posterior Sampling (1 minute read)

This paper shows that releasing posterior sample paths from a Gaussian process is differentially private on its own, since the sampling randomness substitutes for the noise that standard DP mechanisms add externally. The authors derive explicit Rényi-DP bounds, finding that meaningful privacy hinges on effective ridge regularization, and confirm the predicted leakage with membership-inference attacks. For stronger guarantees, calibrated GP noise serves as an additional tunable privacy knob.

Learning Red Agent Policy from Observations for Neurosymbolic Autonomous Cyber Agents (1 minute read)

This paper proposes an imitation-learning technique that lets a neurosymbolic cyber-defense agent infer a red agent's actions and policy from network observations and its own defender actions, despite the attacker's moves being unobservable in a partially observable environment. By predicting red behavior across discrete states and actions, the method gives autonomous defenders a way to estimate intrusion level and adapt, achieving high prediction accuracy against diverse simulated attacker policies.

Cordon: Semantic Transactions for Tool-Using LLM Agents (1 minute read)

This paper introduces Cordon, a transactional runtime that stages and validates an LLM agent's irreversible tool effects before commit, replacing per-call guardrails with a task-scoped execution boundary. It executes reversible mutations in shadow state, queues outward actions in an effect outbox, and validates the composed flow before releasing anything. In evaluation, Cordon caught cross-step violations that existing defenses missed and cut irreversible-effect failures while preserving benign task completion at modest approval and latency overhead.
