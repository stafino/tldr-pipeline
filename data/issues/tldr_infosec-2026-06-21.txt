Unpatchable 'usbliter8' Exploit Breaks Apple A12 and A13 SecureROM Boot Chain (1 minute read)

Security researchers at Paradigm Shift published usbliter8, an unpatchable SecureROM exploit achieving arbitrary code execution on Apple's A12 and A13 chips via a buffer underflow in the Synopsys DWC2 USB controller, abusing the DART IOMMU's bypass mode to overwrite SRAM. The flaw is burned into silicon and requires physical DFU-mode access over USB, so treat lost or seized iPhone XS through 11-series and SE devices as fully compromisable below the signed boot chain.

AutoJack Attack Lets One Web Page Hijack AI Agent for Host Code Execution (1 minute read) [flag]

Microsoft researchers detailed AutoJack, an exploit chain that turns an AI browsing agent into a remote-code-execution vector by chaining three flaws in AutoGen Studio's MCP WebSocket: localhost trust, skipped authentication, and an unallowlisted command parameter. A single attacker page loaded by the agent, via planted link or prompt injection, runs arbitrary host commands unauthenticated; the vulnerable handler shipped only in pre-releases 0.4.3.dev1 and 0.4.3.dev2, with the fix in GitHub main at commit b047730.

Users cry foul after AMD stripped memory crypto from its consumer CPUs (1 minute read)

AMD silently removed Transparent Secure Memory Encryption (TSME) from its consumer Ryzen CPUs, leaving DRAM contents unencrypted and exposing systems to cold-boot and physical memory-extraction attacks that the feature was meant to mitigate. The change was undocumented and appears deliberate, so defenders relying on at-rest memory encryption for laptops or kiosks should verify TSME support per SKU rather than assume parity with prior generations.

The Gentlemen RaaS Uses GentleKiller EDR Framework Targeting 400 Security Processes (1 minute read)

ESET notes the group operationalizes newly disclosed BYOVD PoCs within days of release, so defenders should enforce driver blocklists and monitor for signed-but-anomalous driver loads ahead of encryptor deployment.

Strategies & Tactics

Build your own vulnerability harness (1 minute read)

Cloudflare details how it built a model-agnostic vulnerability harness for Project Glasswing, deliberately swapping models across the pipeline so one model handles discovery and a different one validates, cross-checking findings through distinct reasoning. The architecture persists hundreds of investigations across runs, traces vulnerabilities through cross-repo dependencies, and filters thousands of raw candidates down to a triaged queue of actionable fixes.

MosaicLeaks: Can your research agent keep a secret? (5 minute read)

MosaicLeaks introduces a deep-research benchmark showing that agents combining private documents with web retrieval leak sensitive enterprise data through their outbound query logs, where an observer reassembles fragments via the mosaic effect without ever seeing the documents. The authors found leakage is common across tested models and that training purely for task accuracy worsens it. Their privacy-aware RL method, PA-DR, cut answer and full-information leakage from 34.0% to 9.9% while raising strict chain success from 48.7% to 58.7%.

Linux eliminates the strncpy API after six years of work, 360 patches (1 minute read)

Linux 7.2 removed the strncpy() API from the kernel after a six-year, 362-commit campaign to migrate every caller off it. The function was a persistent source of bugs due to counter-intuitive NUL-termination semantics and redundant zero-fill overhead. Kernel code should now use strscpy() for NUL-terminated copies, strscpy_pad() or strtomem_pad() for zero-padded fields, or memcpy() for known-length copies.

Your browser tab could become encrypted storage for someone else’s files (1 minute read)

A research paper from IENYC's Gregory Magarshak describes Safecloud, a decentralized storage design where browser tabs act as encrypted storage nodes: files are chunked and encrypted on the owner's device before leaving, so storage nodes ("Drops") hold only ciphertext in IndexedDB and routing nodes ("Jets") never see keys. Every key derives from a single 32-byte root secret, and three overlapping trees handle integrity, confidentiality, and authorization over one addressing path, enabling content-addressed dedup and single-key segment streaming without exposing plaintext to any participant.

How Dropbox uses MCP and Dash to close the design-to-code security gap (5 minute read)

Dropbox built a system that combines MCP, foundational LLMs, and its Dash content index to automatically pull the relevant threat model during code review and check whether code changes meet the security requirements agreed on during design. Because Dash already indexes years of security reviews and engineering docs, the system closes the design-to-code gap without engineers manually linking threat models to pull requests. The post covers the architecture and what months of threat-model analysis revealed.

How attackers are jailbreaking LLMs with CTF framing and how to catch them (5 minute read)

Sysdig's Threat Research Team documented a "jailbreak-to-deploy" pattern where attackers frame exploit requests to their own upstream LLMs as CTF challenges or CVE-hunting exercises, coaxing the models past safety training into writing working exploit code they ship nearly verbatim. The framing leaks into telemetry: CVE-templated strings like ctf-litellm-cve42271-mcp-stdio/1.0 appeared in User-Agents, password fields, and AWS roleSessionNames across campaigns hitting PraisonAI, LiteLLM, FastGPT, Open-WebUI, and Gotenberg. Hunt for those baked-in CTF/CVE labels as detection artifacts.
