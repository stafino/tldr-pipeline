AMD will reinstate memory encryption on Ryzen 9000 CPUs via BIOS update in July (1 minute read)

AMD will reinstate Transparent Secure Memory Encryption (TSME) on non-PRO Ryzen 9000 desktop CPUs via a July BIOS update, reversing a quiet removal in AGESA 1.2.7.0 that disabled the firmware-level RAM encryption present since 2020. TSME defends against cold-boot attacks where an attacker with physical access extracts sensitive data from memory, so affected fleets should apply the July microcode and confirm encryption status post-update.
