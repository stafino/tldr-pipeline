Hundreds of AI-powered iOS apps found exposing credentials (1 minute read)

Researchers at Wake Forest University analyzed 444 iOS apps with LLM features and found 282 (64%) exposing exploitable credentials, including 136 leaking authentication tokens, 92 allowing unauthenticated backend access, and 54 hardcoding plaintext API keys. The leakage was systemic rather than niche, affecting apps with up to 2.3 million ratings, and 55% routed traffic through custom developer backends that provider-side mitigations alone cannot fix.

Encrypted DNS still tells an eavesdropper where to look (1 minute read)

Encrypted DNS over TLS, HTTPS, QUIC, and CoAP hides query contents but leaves plaintext headers that mark a flow as DNS, letting an eavesdropper on a wireless link single it out to block, profile, or analyze it. A new study trained a Random Forest classifier on 296 IoT scenarios and found addresses, ports, and monotonic counters leak the most. The authors propose eliding headers via SCHC compression and CoAP block-wise transfer to even out packet lengths.

Agent Beacon: Open-source telemetry layer for AI agents (1 minute read)

Asymptote Labs released Beacon, an open-source (MIT, mostly Go) telemetry layer that configures OpenTelemetry export and hooks for AI coding agents like Claude Code, Codex CLI, and Cursor, then normalizes their file edits, command execution, tool use, and approval decisions into a single JSON event log. It runs locally without an account, binds its dashboard to loopback, offers redaction and retention modes, and can forward events into customer-managed SIEMs.
