FFmpeg fixes PixelSmash flaw in widely used video decoder (1 minute read)

FFmpeg patched PixelSmash (CVE-2026-8461, CVSS 8.8), a heap out-of-bounds write in the MagicYUV decoder triggered by a malicious AVI, MKV, or MOV file processed through libavcodec. JFrog achieved full RCE against a Jellyfin 10.11.9 server via its automatic library scan; Kodi, OBS, PhotoPrism, Nextcloud, and desktop thumbnail generators are also exposed, with RCE viable when ASLR is disabled or chained around.

Researchers at Spur scanned 6,038 LG and Samsung smart TV apps and found 2,058 of them, nearly half on LG, bundling residential proxy SDKs from vendors like Bright Data that route strangers' internet traffic through the TV's home connection. A new unpatchable flaw in Apple chips opens the door to an iPhone jailbreak (1 minute read)

Paradigm Shift, a Barcelona offensive-security firm, published "usbliter8," a Boot ROM flaw with a proof-of-concept affecting Apple's A12 and A13 chips in the iPhone XS, XR, and 11. 29-Year-Old Squid Proxy Bug 'Squidbleed' Can Leak Cleartext HTTP Requests (1 minute read) [flag]

Squidbleed (CVE-2026-47729), a 29-year-old heap over-read in Squid's FTP directory-listing parser, lets any client already permitted to use the proxy leak another user's cleartext HTTP request, including Authorization headers and session tokens, by pointing the proxy at an attacker-controlled FTP server. The bug is live in Squid's default config and proof-of-concept code is public; patch and verify the guard lands in FtpGateway.cc rather than trusting version numbers, since distros ship their own backports.

FortiBleed campaign used custom FortiGate sniffer to steal credentials (1 minute read)

SOCRadar reports that the FortiBleed campaign has targeted more than 430,000 FortiGate firewalls since February 2026, deploying a Golang tool called FortigateSniffer that abuses the legitimate FortiOS "diagnose sniffer packet" command to harvest credentials from RADIUS, NTLM, Kerberos, and LDAP traffic. The initial-access broker behind it gains admin access via credential stuffing and brute-force, so defenders should rotate FortiGate credentials and audit SSH sessions for sniffer activity.

Prompt Injection as Role Confusion (1 minute read)

This writeup argues prompt injection is fundamentally a role-confusion problem: an LLM receives system prompts, user messages, tool outputs, and its own reasoning as one undifferentiated token stream, and the role tags meant to partition that stream are a weak signal the model can be made to misread. The authors use this framing to construct new attacks, explain mechanistic-interpretability findings, and predict when injections succeed, arguing that hardening role separation matters more than patching individual payloads.

Linux and Secure Boot certificate expiration (2025) (1 minute read)

The Microsoft 2011 key used to sign the shim first-stage UEFI bootloader expires in September, after which Microsoft will stop signing shim with it, breaking Secure Boot installs of new Linux distributions on affected systems. A replacement key has existed since 2023 but may not be present in firmware and can require a vendor firmware update that may never ship. Defenders running Secure Boot fleets should audit firmware key databases now and push fwupd/LVFS updates before the cutover.

Never Give Them Your Face (1 minute read)

The author argues that online age-verification laws are functionally identity-verification mandates: to confirm no minor is present, a service must check everyone, turning a rule written for teenagers into a universal ID checkpoint backed by government documents or facial scans. The piece notes that almost none of these systems accept a document attesting only to being over eighteen, because the captured data — name, birthdate, document number, face — is the point, and unlike a password, a face cannot be reset once breached.
