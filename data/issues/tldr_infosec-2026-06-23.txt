One-Prompt Censorship Evasion via Generative Diffusion Models (1 minute read)

This paper reframes censorship evasion as a semantic image-to-image editing task, introducing FlowPaint, a diffusion-based framework that reshapes censored network traffic into benign-looking patterns from a single natural-language prompt. The authors report it outperforms existing evasion tools against both rule-based middleboxes and learning-based traffic classifiers, with users switching between censorship paradigms by varying the instruction rather than hand-tuning mechanism-specific configs. For defenders, it signals that DPI and ML traffic classifiers face an adversary that generalizes across modalities without manual fitting.

On Good Authority: Release-Authority Measurement for Registry-Mediated Package Ecosystems (1 minute read)

This paper introduces a predecessor-aware release-authority record that flags when a package's public publishing path (publisher, repository, workflow, provenance, signing) changes from the prior release, surfacing review cues that dependency graphs miss. Across 43,100 predecessor comparisons spanning npm, PyPI, Maven Central, and RubyGems, the rules identified 204 release-path discontinuities, and practitioners rated 20 of 30 sampled triggers as warranting immediate review. The signals don't catch same-path compromise or unchanged compromised CI, which need separate evidence.

VeriPort: Automated and Verified Patch Backporting at Scale (1 minute read)

VeriPort is an agentic system that automatically backports security patches to every affected version of an open-source dependency, building an evidence chain to confirm each patch blocks exploitation and preserves functionality. It resolved 95.3% of 128 BackportBench tasks, beating Claude Code by 22.7 points, and generated over 5,000 verified patches across 169 high- and critical-severity CVEs. In the process it flagged 2,100 versions wrongly listed as affected and 127 previously unidentified vulnerable versions, prompting upstream corrections to 23 advisories.

RAVEN: Agentic RAG for Automated Vulnerability Repair (1 minute read)

RAVEN is an agentic RAG framework for automated vulnerability repair that pairs a multi-faceted retrieval pipeline over historical fixes with a Curator Agent that pulls cross-file dependencies to patch vulnerabilities local context alone can't resolve. It runs on open-source LLMs in a fully local, low-GPU setup and hit an 83.13% repair success rate across 160 real-world CVEs spanning two languages, unseen CWE categories, and out-of-distribution settings, beating prior state-of-the-art frameworks at negligible cost.

What You See Is Not What You Execute: Memory-Based Runtime SBOM Generation for Supply Chain Security (1 minute read)

MEM-SBOM is a memory forensics framework that generates software bills of materials directly from the runtime state of Python applications, recovering loaded modules from the interpreter's internal structures without requiring prior instrumentation. Built as Volatility 3 plugins and tested against 51 real-world apps, it hit 100% extraction accuracy and surfaced runtime packages that metadata- and filesystem-based SBOM tools miss. The payoff for defenders is a forensically sound view of what code actually executed during incident response, including which dependencies reach vulnerable functions.

DE-FIVE: Detecting Malicious Image Prompts via Fourier Features and Image Vector Embeddings (1 minute read)

This paper introduces DE-FIVE, a training-free framework for detecting malicious image prompts against vision language models, targeting adversarial perturbations and indirect prompt injection delivered through crafted images. It pairs a black-box detector operating on Fourier-domain features with a white-box detector that uses image vector embeddings from the visual encoder, derived from only a few-shot malicious set. The authors report it consistently outperforms state-of-the-art baselines without retraining or deploying additional classifiers.

Safety in Self-Evolving LLM Agent Systems: Threats, Amplification, and Case Studies (1 minute read)

This paper analyzes the security of self-evolving LLM agent systems, which autonomously update their own parameters, memory, tools, and architecture. Using a Module-Lifecycle Attack Surface matrix, the authors find that 17 of 25 cells face critical threats with no effective mitigation, and that evolution-native frameworks hit a 100% attack persistence rate (40/40 payloads) while co-located scanners blocked just 2.5%. The core finding: self-evolution converts session-bounded attacks into lineage-persistent ones that self-amplify across generations, making static defenses structurally inadequate.

TROPT: An Open Framework for Unifying and Advancing Discrete Text Optimization (1 minute read)

TROPT is the first open-source framework that unifies discrete text-trigger optimization, the search for input sequences that steer a model toward an attacker-specified objective and the basis for most LLM jailbreaks. It ships with 30+ recipes built from 15+ optimizers (white-box to black-box) and 15+ losses, letting red teams compare techniques head-to-head and port attacks across domains like corpus-poisoning embedding models. The authors' large-scale experiments surface potent but underadopted jailbreak strategies defenders should expect to see weaponized.

Public Diffusion Models, Private Images: Key-Controlled Inversion for Conditional Reconstruction (1 minute read)

This paper proposes key-controlled inversion, an access-control scheme for white-box diffusion models where published parameters let anyone invert intermediate latents back to the original input image. The authors turn diffusion's exponential error propagation into a defense by injecting key-dependent noise into the inversion formula, so only the correct key reconstructs the image and any other key yields unrecognizable output. They prove the resulting ciphertext distribution is IND-CPA secure with adversary advantage exponentially small in a tunable security parameter.
