Embedding Forbidden Text in Spyware to Discourage AI Analysis (1 minute read)

A malware developer is prepending JavaScript block comments containing fake nuclear and biological weapons policy text to spyware payloads, aiming to derail LLM-based triage that feeds file headers to a model without isolating them as untrusted data. The trick can trigger refusal, context pollution, or premature classification in naive LLM-first scanners, but it does not evade YARA rules, entropy checks, AST parsing, or behavioral detection.

Cisco Unified CM flaw actively exploited to drop webshells (CVE-2026-20230) (1 minute read)

CVE-2026-20230, an unauthenticated SSRF in Cisco Unified Communications Manager, is being actively exploited via Tor to chain into webshell deployment and remote code execution, with honeypots observing automated sweeps after a public PoC dropped this week. The attack abuses WebDialer SSRF to deploy a rogue Apache Axis service and write JSP shells under /platform-services/axis2-web/; apply Cisco's June 3 patches or disable the WebDialer service to mitigate.

Cordyceps CI/CD Flaws Expose 300+ GitHub Repositories to Supply-Chain Attacks (1 minute read)

Novee Security disclosed Cordyceps, a class of CI/CD misconfiguration in which untrusted pull requests trigger privileged GitHub Actions workflows, enabling command injection, credential theft, and supply-chain compromise exploitable by any unauthenticated user with a free account. A scan of roughly 30,000 high-impact repositories found 300+ fully exploitable, including projects at Microsoft, Google, Apache, and Cloudflare; audit workflows that run on `pull_request_target` and strip write permissions and secrets from untrusted PR triggers.

CISA warns of max severity Ubiquiti flaws exploited in attacks (1 minute read)

CISA added four actively exploited flaws to its KEV catalog, including three max-severity Ubiquiti UniFi OS bugs (CVE-2026-34908, -34909, -34910) that Bishop Fox chained for unauthenticated RCE with elevated privileges, plus a critical Lantronix EDS5000 command-injection (CVE-2025-67038). BOD 26-04 gives federal agencies three days to patch; apply Ubiquiti's May updates and Lantronix firmware 2.2.0.0R1, and run Bishop Fox's detection script to find vulnerable instances.
