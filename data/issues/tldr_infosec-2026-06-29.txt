Public PoC Released for Critical libssh2 CVE-2026-55200 Client-Side SSH Flaw (1 minute read)

A public PoC is out for CVE-2026-55200 (CVSS 9.2), a critical integer-overflow-to-heap-write in libssh2's ssh2_transport_read() that lets a malicious SSH server corrupt memory on a connecting client with no credentials or interaction, with possible RCE. The flaw affects all releases through 1.11.1 and is embedded in curl, Git, PHP, and firmware updaters, many statically linked, so inventory linked binaries and rebuild rather than relying on distro package updates.

Data Exposure Flaws Threaten Dify AI Platform Used by 1 Million Apps (1 minute read)

Dify 1.14.2 patches all four flaws; operators running multi-tenant instances should update immediately and apply WAF rules to block exploitation.

OpenAI Unveils GPT-5.6 Sol as Its Most Advanced Cybersecurity AI (1 minute read)

OpenAI released a limited preview of GPT-5.6 Sol, its most capable cybersecurity model, which matched Mythos Preview on ExploitBench while using roughly a third of the output tokens. Sol skews toward defensive work like vulnerability identification and patch development, isolating bugs in the Chromium and Firefox codebases without building a full-chain exploit, and ships behind real-time bio and cyber input classifiers.

Perplexity Bumblebee Review: The Supply Chain Scanner Your Dev Machine Needs (3 minute read)

Perplexity open-sourced Bumblebee, a read-only Go supply-chain scanner that inventories installed packages, editor and browser extensions, and MCP configs, then matches them against known-compromised advisories without ever invoking a package manager. It emits NDJSON and scanned 847 packages across 9 ecosystems in under 4 seconds, filling the developer-machine blind spot that CI, SBOM, and EDR tools miss, including poisoned claude_desktop_config.json MCP servers.

DarkMoon: Open-source AI pentesting platform (1 minute read)

DarkMoon, an open-source AI pentesting platform, runs end-to-end security assessments by pairing an LLM orchestrator (OpenCode) with an MCP control layer that exposes an allow-list of 50+ tools, including Nuclei, sqlmap, BloodHound, and NetExec, inside an isolated Docker container. The LLM never executes commands directly, and specialized sub-agents for web apps, Active Directory, Kubernetes, and network protocols map coverage to ISO 27001, NIST SP 800-115, and MITRE ATT&CK.
