Mozilla warns of indirect prompt injection risk in AI coding agents (1 minute read) [flag]

Mozilla's 0DIN team demonstrated an indirect prompt injection attack in which a malicious GitHub repo steers AI coding agents like Claude Code into running setup commands that fetch a reverse-shell payload at runtime via a DNS TXT record, leaving nothing malicious in the repo for code review or static analysis to catch. Treat setup instructions in unfamiliar repos as untrusted code regardless of agent recommendations, and design agents to surface what a command actually executes rather than its literal string.

‘DirtyClone’ Linux Kernel Vulnerability Leads to Root Access (1 minute read)

JFrog published technical details and a PoC for DirtyClone (CVE-2026-43503, CVSS 8.8), a Linux kernel local privilege escalation flaw in the networking stack's socket-buffer handling that lets any local user gain root. The bug is a variant of the DirtyFrag/Fragnesia family affecting Debian, Fedora, and Ubuntu with unprivileged user namespaces enabled; only kernels updated to v7.1-rc5 with the complete fix chain are protected, so partially patched LTS branches remain exposed.

Microsoft Removes 119 Edge Extensions That Hid Malware in Images and Fonts (1 minute read)

Microsoft removed 119 malicious Edge extensions tied to a single threat actor it tracks as StegoAd, which used steganography to hide payloads inside PNG, WebP, and WOFF2 image and font files, then activated days post-install to steal credentials and run ad fraud. Active since at least 2021, the extensions posed as ad blockers, VPNs, and translators across an install base of up to 2.6 million, using server-side validation, DevTools detection, and execution gates to evade analysis.

FCC escalates China crackdown by closing equipment import loophole (1 minute read)

The rules take effect in July and extend Chairman Brendan Carr's China crackdown, which already covers foreign-made routers, blanket carrier authorizations, and subsea cable supply chains following the Salt Typhoon intrusions.
