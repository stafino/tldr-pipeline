Public PoC Released for Critical libssh2 CVE-2026-55200 Client-Side SSH Flaw (1 minute read)

A public PoC is out for CVE-2026-55200 (CVSS 9.2), a critical integer-overflow-to-heap-write in libssh2's ssh2_transport_read() that lets a malicious SSH server corrupt memory on a connecting client with no credentials or interaction, with possible RCE. The flaw affects all releases through 1.11.1 and is embedded in curl, Git, PHP, and firmware updaters, many statically linked, so inventory linked binaries and rebuild rather than relying on distro package updates.

Abandoning Abstractions: Manually Crafting EtherNet/IP Packets Almost Broke Me (2 minute read)

The author built an EtherNet/IP and CIP sandbox from scratch using raw sockets, a Linux loopback, and a cpppo simulator, deliberately dropping Scapy and protocol wrappers to learn how PLCs actually parse control-plane traffic at the wire level. Basic tag reads (0x4C) and writes (0x4D) worked, but fragmented transfers (0x52/0x53) broke silently under manual offset and session-state tracking. The takeaway for OT researchers: abstraction libraries hide the encapsulation and CIP object model where real protocol bugs and detection gaps live.

DarkMoon: Open-source AI pentesting platform (1 minute read)

DarkMoon, an open-source AI pentesting platform, runs end-to-end security assessments by pairing an LLM orchestrator (OpenCode) with an MCP control layer that exposes an allow-list of 50+ tools, including Nuclei, sqlmap, BloodHound, and NetExec, inside an isolated Docker container. The LLM never executes commands directly, and specialized sub-agents for web apps, Active Directory, Kubernetes, and network protocols map coverage to ISO 27001, NIST SP 800-115, and MITRE ATT&CK.
