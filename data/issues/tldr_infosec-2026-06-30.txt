Factoring RSA Keys with Many Zeros (1 minute read)

Researchers analyzing badkeys data found a class of weak RSA keys with regularly spaced blocks of all zeros, including certificates issued to Yahoo, Verizon, and NetApp devices, plus SSH hosts running EnterpriseDT's CompleteFTP (RSA keys from v10.0.0-12.0.0, DSA keys from v10.0.0-23.0.4). The sparse moduli are factorable, and the recurrence of the same failure across independent implementations suggests more vulnerable products remain undiscovered and worth scanning for.

Researchers Demo New Claude Code Attack Using Harmless-Looking Repositories to Hijack Developer Machines (1 minute read) [flag]

Mozilla's 0Din researchers demonstrated an indirect prompt-injection attack against Claude Code in which a benign-looking repository's setup notes trigger an install error whose message instructs the agent to run a recovery command that executes setup.sh, which fetches a base64-encoded payload from a DNS TXT record and spawns a reverse shell. A Lightweight Post-Quantum Authentication Framework for 5G Base Station Bootstrapping (1 minute read)

This paper introduces EMULSION, a symmetric-key framework for authenticating 5G base station SIB broadcasts, which 5G leaves unprotected and open to fake base station attacks, MITM interception, and spoofed emergency alerts. It uses a TESLA-style HMAC chain anchored by a per-epoch MAYO post-quantum signature, fitting authentication into a single SIB packet with no fragmentation while covering SIB1 through SIB21. On an over-the-air testbed it ran 33x faster than ML-DSA and 12x faster than FN-DSA, both of which exceed 5G packet-size limits.

On the Internet, Nobody Knows You're an LLM Bot: Unmasking Web Agents with Multi-Layer Fingerprinting (1 minute read)

Researchers deployed honeysites guarded by CAPTCHAs, proof-of-work, and Cloudflare's anti-bot defenses, then pointed six LLM-based web agents at them, finding that some bypassed every mechanism tested. The counterintuitive takeaway for defenders: stealth and anti-detection add-ons usually raise an agent's detectability rather than lower it, so layered fingerprinting beats blocklists for spotting LLM-driven traffic.

A Multi-task Mixture of Experts Framework for Malware Classification, Packing Detection, and Family Attribution (1 minute read)

A Hybrid Framework For Crypto-Ransomware Detection In Enterprise Shared Storage (1 minute read)

Words Speak Louder Than Code: Investigating Cognitive Heuristics in LLM-Based Code Vulnerability Detection (1 minute read)

This paper shows that LLM-based vulnerability detectors are susceptible to the same cognitive heuristics that bias humans, with susceptibility holding code fixed and varying only surrounding context. Across eight models and three languages, framing effects (via task objectives) flipped verdicts 33.2% of the time, anchoring 23.5%, and author-attribution halo 18.4%, with semantic-reasoning bugs more affected than pattern-matchable ones. A black-box cognitive attack suppressed up to 97% of previously detected vulnerabilities, so treat LLM scanner context as an untrusted, manipulable input.

COHORT: Collaborative Orchestration for Hardening via Offensive Replay on Emulated Topologies (1 minute read)

COHORT is a multi-agent LLM framework that automates enterprise network mitigation, a task that normally takes analysts weeks of expert work. It proposes mitigations as real device commands on a GNS3 emulator running vendor firmware, then validates each via offensive replay, re-executing the original adversary against the mitigated network and checking that legitimate connectivity survives. Across three topologies and four attack scenarios, 46.7% of generated mitigations both disrupted the attack and preserved connectivity, 4.4x the single-agent baseline.

Multi-Level Distributional Entropy for Explainable Network Intrusion Detection (1 minute read)

This paper introduces Multi-Level Distributional Entropy (MDE), a framework that derives interpretable entropy features from flow-level summary statistics without raw packet access or training data, matching conventional IDS features (weighted F1 0.708-0.989) across four benchmarks. Full metric reporting exposes failure modes that aggregate F1 hides: on CICIDS-2018, F1=0.74 conceals a 0.48 detection rate, and under temporal shift fixed thresholds collapse to DR=0.082. Defenders should track detection rate, not just F1, when validating ML detectors.

SurrogateShield: Beyond Redaction for High-Utility, Privacy-Preserving LLM Interactions (1 minute read)

SurrogateShield is a client-side proxy that swaps detected PII for locally generated, type-consistent surrogate values before queries hit third-party LLM APIs, then restores the originals in the response so no real PII crosses the network. A three-stage detection cascade covering 22 PII types hit 98.87% F1, and surrogate substitution preserved far more semantic utility than placeholder redaction, raising BERTScore from 81.59% to 94.85%. Surrogate-to-original mappings stay in an AES-256-GCM ShadowMap that never leaves the device.

Uncovering Similar but Different Packages in PyPI and Potential Security Threats (1 minute read)

A large-scale study of 200K PyPI packages (one-third of the repository) found widespread package replication, where new packages duplicate most of an existing codebase under a different maintainer. The authors identified 256 previously unknown replicated vulnerable packages that current detection tools miss, plus seven new malicious packages that cloned popular projects and added code injection. Defenders should treat replication as a supply-chain blind spot and extend dependency scanning to flag near-duplicate packages, not just known-bad names.

Forensic Trajectory Signatures for Agent Memory Poisoning Detection (1 minute read)

Researchers studying LLM agents under persistent memory poisoning found a behavioral invariant: successful exfiltration requires a memory_recall_fact call before email_send_email, a transition benign sessions rarely produce. A Random Forest over 19 trajectory features hits AUC 0.9904 and generalizes to GPT-4.1 and GPT-4o without retraining. For defenders, this means tool-call logs alone can flag memory-channel attacks and distinguish them from prompt injection, which leaves a separate trajectory signature.

Proofs of Ownership for Machine Learning Models (1 minute read)

This paper formalizes proofs of ownership for stolen ML models as a three-party game among an owner, a thief, and a judge, where the owner perturbs a model and attaches a proof, and the judge decides whether a suspect model derives from it. The authors prove a dichotomy for black-box classifiers: under standard cryptographic assumptions, ownership is provable if and only if the concept class is not self-correctable. The result gives a precise boundary for when model-theft attribution is achievable.
