Catching ransomware on the wire before it locks the file server (1 minute read)

Researchers at La Trobe University built a ransomware detection framework that reads SMB file-sharing traffic on the wire rather than relying on endpoint agents, closing the blind spot where a compromised client encrypts files on a remote server. The method segments traffic using fixed-size SMB control packets, such as the 260-byte directory enumeration and 410-byte new-file response, to reconstruct client file operations and fingerprint ransomware families by note size, without inspecting packet contents or touching endpoints.

What the AI patch gap means for enterprise security (1 minute read)

Tuskira analyzed findings from Anthropic's Claude Mythos Preview, which verified 1,596 vulnerabilities across hundreds of open-source projects in nine weeks at a 90.8% true-positive rate. Discovery ran at roughly 25 verified bugs a day while credited fixes moved at about 1.5, a 16.5-to-1 deficit that widens the open-issue backlog daily. Some 95% of disclosures had no public advisory at snapshot, so defenders should track upstream commits directly and operate at discovery cadence rather than waiting on advisory feeds.

Where Severity Scores Go Wrong: “Just Add Prototype Pollution” (1 minute read)

JFrog's Security Research team found that CVSS severity scores routinely overstate real-world risk, reassessing 2025 NVD data and downgrading 96% of critical and 65% of high-severity CVEs after weighing practical exploitation requirements. Prototype pollution is the case study: the 10 CVEs Axios has disclosed since April 2026 all require a separate, pre-existing prototype pollution flaw in the process to exploit, yet carry inflated scores that ignore that dependency. Prioritize patching against actual exploitability and environmental constraints, not headline CVSS numbers.

Detecting Adversarial Evasion Attacks Against Autoencoder-Based Network Intrusion Detection Systems (1 minute read)

This paper proposes two complementary detectors for adversarial evasion attacks against autoencoder-based network intrusion detection systems, targeting PANDA-style attacks that convert packet sequences into invertible grayscale images to run masked FGSM against NIDS anomaly scoring. The Residual Localisation Detector tracks spatial concentration of reconstruction errors in the inter-arrival time region, while the Feature-Space Perturbation Consistency Detector checks packet-level timing features directly. Both hit F1 ≥ 0.99 against adversarial traffic across IoT devices in the UQ-IoT dataset.

SoK: Attack and Defense Landscape of Mobile On-device AI Systems (1 minute read)

This paper presents the first systematization of knowledge (SoK) on security for mobile on-device AI (MoAI) systems, which run inference locally instead of in the cloud and thereby expose models to attacks stemming from local storage. The authors map the attack and defense landscape across MoAI security pillars, then flag unresolved gaps for teams shipping on-device inference to reason about model extraction, tampering, and other risks that don't exist in cloud-hosted deployments.

SessionBound: Turning Enterprise Task Approval into Budgeted Database Sessions (1 minute read)

SessionBound converts approved enterprise tasks into short-lived, budgeted database sessions for AI agents, so an agent can generate SQL freely but every query must execute inside a signed task token's boundary. Its SessionBoundDB runtime enforces safe views, row scope, denied fields, and query and disclosure budgets at the database layer rather than trusting an LLM to judge safety. A PostgreSQL prototype cleared a 24-scenario suite with p50 overhead of roughly 1.4ms.

HARC: Coupling Harmfulness and Refusal Directions for Robust Safety Alignment (1 minute read)

This paper introduces HARC (Harmfulness-And-Refusal Coupling), a fine-tuning method that pairs an LLM's separable harmfulness and refusal directions across both prompt and response token positions to harden safety alignment. The authors show jailbreaks work by suppressing one direction before generation, and that models often recognize harmful content mid-response even after missing it at the prompt. HARC beat six training-time and inference-time baselines on the robustness-capability-usability trade-off, and the directions transferred across five model families and two scales without architecture-specific tuning.

Knowledge-Enhanced Agentic Vulnerability Repair (1 minute read)

KeaRepair is an agentic automated vulnerability repair system that grounds patch generation in verified program facts and knowledge mined from historical vulnerability-patch pairs. A tool-augmented agent runs ReAct-style diagnosis, then generates patches validated through a closed loop of compilation, PoC replay, and test-suite execution. Paired with Gemini-3.1-Pro, it repaired 46 of 55 reproducible C/C++ vulnerabilities (83.64%), including six that no baseline could fix, signaling that remediation is closing the gap on AI-accelerated discovery.

Minos: A Multi-Agent Collaborative Framework for Provenance-Based Backward Tracking (1 minute read)

This paper introduces Minos, a multi-agent framework that reframes provenance-based backward tracking for APT forensics as an LLM-driven reasoning process, coordinating four specialized agents under a finite state machine to prune the search space instead of exhaustive graph traversal. Across 14 attack scenarios in five public datasets, it reached 0.92 recall and 0.64 precision while producing attack subgraphs 49% more compact than prior methods, and it emits interpretable reasoning traces to support forensic auditing.

Federated Sovereign Transport Protocol (FSTP): Verifiable Coordination Without Disclosure (1 minute read)

This paper introduces the Federated Sovereign Transport Protocol (FSTP), a federation transport layer that makes data confinement a structural property rather than an operator policy. A synchronization agent with a formally closed output type set prevents raw internal data from appearing in any federation message, enforced by the Rust type system at compile time instead of runtime checks. A contextual identity model derives unlinkable per-relationship identifiers to block cross-context correlation, and a Blocklace event substrate provides tamper-evident partial ordering that supports erasure without breaking the hash chain.

Distill to Detect: Exposing Stealth Biases in LLMs through Cartridge Distillation (1 minute read)

New method distills the logit-distribution gap between a suspect LLM and its base to surface stealth preferential biases invisible to text inspection.

Know Thy Neighbor: Cross-TEE Mutual Attestation (1 minute read)

This paper introduces Hema, a formally-verified protocol for mutual attestation between trusted-application instances running on the same or different TEE types. The current default requires each TEE to embed the full attestation software stack of every other TEE it talks to (SGX carrying TrustZone's stack and vice versa), which Hema replaces with a single cross-TEE handshake. For teams running confidential-computing workloads across heterogeneous enclaves, this cuts the attestation surface each TEE must trust and verify.

No Country for Old Privacy: The Evolving Challenges of Anonymity in Bitcoin (1 minute read)

A longitudinal study of Bitcoin's second-generation anonymization protocols (CoinJoin, CoinSwap, CoinShuffle, Stealth Addresses) found detectable privacy usage stays below 1% of network transactions, with sharp declines following key regulatory events. The authors identified 5.94M CoinJoin and 23.3M CoinSwap transactions via heuristic filters but found no standardized Stealth Address adoption, suggesting privacy-seeking users have migrated to less detectable methods rather than abandoning anonymity. For chain-analysis and threat-intel teams, on-chain visibility of these protocols understates real obfuscation activity.
