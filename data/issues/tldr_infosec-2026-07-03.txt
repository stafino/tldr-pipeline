Agentic AI Used to Conduct Ransomware Attack via Langflow (1 minute read)

Threat actor JadePuffer exploited CVE-2025-3248 (CVSS 9.8), a critical missing-authentication flaw in the Langflow LLM framework, to gain arbitrary Python code execution on an internet-exposed instance, then used the LLM itself to run reconnaissance, harvest credentials, and pivot to a production server before deploying ransomware. CISA flagged the bug as exploited in May, so patch exposed Langflow instances and rotate any secrets reachable from them.

Researchers say an AI agent just ran a ransomware attack from start to finish, with no human at the keyboard (1 minute read)

Sysdig documented what it calls the first end-to-end ransomware attack run entirely by an AI agent, tracked as JADEPUFFER, which breached a network via a patched year-old Langflow RCE, harvested credentials, deployed a backdoor, and encrypted a production database. The LLM generated its encryption key, printed it once, and never saved or exfiltrated it, so paying the Bitcoin ransom recovers nothing; patch exposed Langflow instances and rotate any API keys or cloud credentials stored on them.

Flock Cameras Can Surveil Cars Without License Plates (1 minute read)

Flock's "Vehicle Fingerprint" lets police identify and track cars using decals, bumper stickers, roof racks, and temporary tags rather than plates, and its "multi geo search" surfaces vehicles that habitually travel together. The technique predates AI and mirrors NSA cell-phone co-location tracking Snowden disclosed. For defenders, the takeaway is that plate-obfuscation offers little protection, and anyone with broad location-data access, not just law enforcement, can build the same movement graphs.
