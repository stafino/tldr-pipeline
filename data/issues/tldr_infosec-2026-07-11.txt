GhostLock, a stack-UAF that has existed in ALL Linux distributions for 15 years (1 minute read)

GhostLock (CVE-2026-43499) is a Linux kernel stack use-after-free in the rtmutex PI code that has shipped in every major distribution since kernel 2.6.39 in 2011, exploitable by an unprivileged local user with only CONFIG_FUTEX_PI and standard threading syscalls. VEGA built a 97% reliable privilege escalation and container escape from it; the fix landed in Linux 7.1, so unpatched hosts should upgrade to the latest LTS.

Researcher Details WhatsApp-to-Host Attack Chain Using Three OpenClaw Flaws (1 minute read)

A researcher chained three now-patched OpenClaw flaws (GHSA-hjr6-g723-hmfm and GHSA-9969-8g9h-rxwm, both CVSS 8.8; GHSA-575v-8hfq-m3mc, CVSS 8.4) to achieve host code execution from a single external WhatsApp message, with no prior foothold required. The path-traversal bug lets a container bind-mount /home to bypass denylisted directories and read every user's SSH keys, so operators should upgrade to version 2026.6.6 and rotate exposed credentials.

Hackers exploit critical auth bypass in Gitea Docker image (1 minute read)

Attackers are actively exploiting CVE-2026-20896, a critical authentication bypass in Gitea's official Docker image that trusts the X-WEBAUTH-USER header from any source IP because it ships REVERSE_PROXY_TRUSTED_PROXIES=*, letting an unauthenticated client impersonate any user, including admins. Upgrade to 1.26.4 immediately; if patching isn't possible, restrict REVERSE_PROXY_TRUSTED_PROXIES to trusted proxy IPs, as roughly 6,200 instances sit exposed on the public web.

Injective Labs GitHub Compromise Pushes Wallet-Key-Stealing npm Packages (1 minute read)

Attackers compromised the Injective Labs SDK GitHub repo via commits from an established contributor's account, publishing malicious @injectivelabs/sdk-ts@1.20.21 to npm that steals cryptocurrency wallet private keys and mnemonic seed phrases through a fake trackKeyDerivation() telemetry function. The poisoned version was pinned across 17 additional @injectivelabs packages to hit transitive users, and while npm has deprecated it, the release artifacts remain downloadable from GitHub, so audit lockfiles and rotate any keys derived on affected versions.

CrowdStrike identifies five new AI prompt injection threats (1 minute read)

CrowdStrike added five new prompt injection techniques to its taxonomy, including Trigger-Activated Rule Addition, Cognitive Token Suppression, Algorithmic Payload Decomposition, Special Token Injection, and Unwitting User Context-Data Injection. Several stage a payload across multiple innocuous-looking inputs or hide instructions inside uploaded documents and forwarded emails, defeating single-prompt filtering. CrowdStrike recommends threat modeling every source of model context, expanding testing, and extending detection engineering to cover composite attacks.
