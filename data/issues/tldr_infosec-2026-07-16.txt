Cursor Flaw Lets Malicious Cloned Repositories Trigger Windows Code Execution (1 minute read)

Cursor on Windows executes any `git.exe` sitting in a cloned repository's root when the project opens, spawning the binary with `git rev-parse --show-toplevel` and re-running it for the session's duration. Mindgard reported the unauthenticated RCE to Cursor on December 15, 2025 and published details this week with no patch or advisory available; the flaw was confirmed against 3.2.16 in April and is believed to persist through the current 3.11 release.

Daxin Resurfaces in Taiwan Alongside Stupig Pre-Login SYSTEM Backdoor (1 minute read)

Symantec found Daxin, the China-linked kernel rootkit last documented in 2022, running on a Taiwan manufacturer's host alongside Stupig, a previously unreported backdoor that trojanizes a keyboard-layout DLL loaded by winlogon.exe to execute SYSTEM commands from the logon screen without generating a logon audit event. 11 Old Microsoft-Signed Linux UEFI Shims Could Let Attackers Bypass Secure Boot (1 minute read)

ESET disclosed 11 old Microsoft-signed Linux UEFI shim bootloaders (mostly shim 0.9 and earlier) that let an attacker execute untrusted code during boot, bypassing Secure Boot on any system trusting the Microsoft Corporation UEFI CA 2011 certificate regardless of installed OS. Microsoft revoked the affected shims in its June 2026 Patch Tuesday update, so apply the DBX revocations to block deployment of bootkits like BlackLotus, Bootkitty, and HybridPetya.

OAuth Client ID Spoofing Lets Attackers Validate Stolen Microsoft Entra Credentials (1 minute read)

Proofpoint disclosed OAuth client ID spoofing, an evasion technique used by at least two threat actors, including UNK_CustomCloak, to enumerate accounts and validate stolen credentials against Microsoft Entra ID without generating a successful sign-in event. Attackers send ROPC-flow requests with syntactically valid but unregistered client IDs to the OAuth token endpoint, reading AADSTS error codes to confirm usernames and passwords while leaving only an application ID and no app name in sign-in logs.

AsyncAPI npm packages infected with credential-stealing malware (1 minute read)

An attacker compromised two AsyncAPI GitHub repositories on July 14 and pushed trojanized commits that the projects' own release workflows published to npm via OIDC trusted publishing, giving five malicious packages valid SLSA provenance across 2.25M weekly downloads. The three-stage payload pulls a modular framework from IPFS and beacons over HTTP, Nostr, Ethereum contracts, and libp2p to steal credentials and auth keys. Provenance attestations do not prove a package is clean.

Microsoft’s Secure Boot has been broken for a decade and no one noticed until now (1 minute read)

ESET researchers found 11 UEFI Secure Boot shims, at least one dating to 2013, that Microsoft signed but never revoked despite known flaws, letting an attacker with a copy of any old shim bypass Secure Boot without a new exploit. The unrevoked binaries affect both Windows and Linux devices and enable persistent bootkits that survive OS reinstalls and drive swaps, so defenders should apply the updated UEFI revocation (DBX) lists.

Blocking Slow-Burn Attacks: Contextual Policies in Omnigent (1 minute read)

Databricks demonstrates a contextual policy in Omnigent that blocks indirect prompt-injection "slow-burn" attacks, where a malicious goal is split into individually benign steps (read a doc, read another, summarize, send) that no single-action guardrail catches. The stateful policy tracks accumulated risk across the whole session and blocks the outbound step once the agent has read too much sensitive material. The guard is tamper-resistant: the agent has no tool to disable it, adding policies requires human approval, and any denial wins when policies combine.

Protecting Privacy in an AI Era (1 minute read)

Daniel Solove argues in the Wall Street Journal that consent-and-control privacy regimes have failed, and that regulators should hold companies accountable for design decisions the way they do food and drug manufacturers. His proposed measures include mandatory data minimization, fiduciary duties over collected data, liability for negligent technological design and for algorithmic harm, and multi-stakeholder review before deployment. The shift would move privacy compliance from consent-flow engineering to demonstrable limits on collection and use.

The scaffolding, not the frontier model, drove the successful runs.
