WordPress Core "wp2shell" RCE flaws get public exploits, patch now (1 minute read)

TP-Link Kasa cameras leaked home GPS via unauthenticated UDP for 6 years (1 minute read)

TP-Link Kasa EC71 cameras responded to unauthenticated UDP probes on the local network with device details including the owner's GPS coordinates, a flaw that shipped in the firmware for roughly six years before disclosure. Any attacker on the same network segment, or an app with local network permission, could geolocate a household without credentials. TP-Link has not published a fix, so segment the cameras onto an isolated VLAN.

Qubes OS Security in the Public Record (1 minute read)

A longitudinal analysis of 109 Qubes Security Bulletins (2011-2025) and the project's Xen Security Advisory tracker found that 87 of 109 QSBs (79.8%) trace to Xen, CPU/microarchitectural, or other upstream components rather than Qubes-core logic, with 113 of 464 XSAs affecting Qubes. Post-2018 annual disclosure rates are statistically flat, and S-shaped vulnerability discovery models did not beat a rolling-mean baseline on short-horizon forecasts. The measurable risk in compartmentalized architectures sits in the upstream trust anchors, not the isolation layer.

VulnHunter: Capital One's agentic AI code security tool (1 minute read)

Capital One open-sourced VulnHunter, an agentic AI code security tool that analyzes source from an attacker perspective to identify exploitable defects, map attack paths, and propose targeted remediations.
