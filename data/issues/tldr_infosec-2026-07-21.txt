Weaponizing And Defending The React Flight Protocol: Deserialization Sinks In RSCs (1 minute read)

The author dissects React's Flight protocol as a deserialization system, arguing CVE-2025-55182 (React2Shell, CVSS 10.0 unauthenticated RCE, tied by Sysdig to North Korean actors) was a symptom rather than a one-off parsing bug in getOutlinedModel and getChunk. SlotGuard: Stop Oversharing Private Local Context in LLM Agent Transcri (1 minute read)

LLM agents leak file paths, emails, and API keys into provider-bound transcripts whenever tool output, shell logs, and file reads get appended. SlotGuard rewrites those bindings into typed slots with format-preserving synthetic values and restores raw data only inside the trusted runtime, removing all 20,814 sensitive characters and dropping credential leakage to 0.0% across 852 planted secrets. Task success stayed near raw-transcript levels, while generic redaction fell to 2.5%.

Insecure Coding Preferences in Long-Term Memory: Security Risks for LLM-based Code Generation (1 minute read)

Researchers tested ChatGPT, Gemini, Qwen, and Grok across Python, C, C++, Go, and JavaScript and found that insecure coding preferences stored in long-term memory raise vulnerable-code generation rates by 2.7 to 50.3 percentage points, with warning rates lagging vulnerability rates by 5.4 to 14.0 points. Poisoned memories persist across rephrased prompts and resist overwriting through normal interaction. Memory-level safety filtering caught 100% of risky entries and restored baseline behavior, while appending security requirements cut functional correctness by up to 15.9 points.

Physical Time-Lock Puzzles (1 minute read)

This paper introduces Physical Time-Lock Puzzles, which anchor solving delay to the hardware-bounded evaluation latency of noisy PUFs instead of sequential computation. The authors argue existing time-lock schemes are really step-locks, since hardware acceleration collapses delay guarantees meant to hold for decades. A composite puzzle design absorbs PUF noise, and an FPGA prototype using a ring oscillator PUF matched the predicted delay windows.

Rate-Distortion Function for Encrypted Traffic Side-Channel Defense (1 minute read)

This paper defines a rate-distortion function for encrypted traffic side-channel defenses, giving a provable lower bound on leakage rate for a given QoS cost budget instead of empirical parameter tuning. The optimal defense takes an exponential-tilting form, and the curve marks the exact Pareto frontier. Measured against it, Front, WTF-PAD, and TrafficSliver sit 0.028, 0.034, and 0.124 bits above optimal, quantifying how much padding budget current website-fingerprinting defenses waste.

Measuring and Evaluating the Performance of Generative AI Models for Scam Detection (1 minute read)

The authors benchmarked nine LLMs against a fine-tuned BERT classifier on a new dataset of real-world scams spanning multiple formats and topics, testing whether general-purpose models hold up in scam detection without task-specific fine-tuning. Larger models scored higher, but prompting strategy closed much of the gap for smaller ones, and LLMs generalized better than the fine-tuned baseline to scams not seen in training. The dataset and evaluation framework are released for teams validating LLM-backed detection in email gateways or fraud-monitoring pipelines.

How Jailbreak Attacks Inform Safety Alignment: A Defender-Centric, Shapley-Based Evaluation of Jailbreak Contributions (1 minute read)

This paper argues that attack success rate is the wrong metric for jailbreak evaluation, since an attack that breaks a model isn't necessarily useful as red-teaming data for safety training. The authors introduce A-MESS, which computes AttackSHAP, a Shapley-based score attributing marginal safety utility to individual attacks, then selects compact attack subsets under a query budget. ASR rankings correlate weakly with defender-centric utility, so red-team suites picked by ASR likely waste alignment training budget.

Adaptive Adversaries: A Multi-Turn, Multi-LLM Benchmark for LLM Agent Security (1 minute read)

This paper introduces a 21-scenario benchmark for adaptive multi-round attacks against LLM agents, where an autonomous attacker observes defender responses and pivots across turns. Scoring only the first attacker turn yields 0-1% attack success, while 15 rounds of adaptation yields 5.4-14.0%, and pooling three frontier attacker models surfaces 1.4-2.2x as many unique successful attacks as the best single one. Defender rankings disagree sharply across scenarios (Kendall's W = 0.19), so single-benchmark model selection for agent security is unreliable.

A Multi-Model Hybrid Defense Approach Against White-box Adversarial Attacks in Computer Network Traffic (1 minute read)

This paper tests a hybrid defense for ML-based NIDS combining adversarial training with Gaussian data augmentation, targeting FGSM and Carlini & Wagner white-box evasion. Teams running model-driven traffic classification should assume an attacker with model access can flip malicious flows to benign unless the classifier is hardened.
