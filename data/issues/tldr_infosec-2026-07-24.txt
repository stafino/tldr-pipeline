Bing Images Flaws Let Crafted SVGs Run Commands as SYSTEM on Microsoft's Servers (1 minute read)

XBOW disclosed two critical flaws (CVE-2026-32194 and CVE-2026-32191, both CVSS 9.8) in Bing's image-processing tier, where a crafted one-pixel SVG piped through an ImageMagick delegate ran commands as SYSTEM on Windows workers and root on Linux across Microsoft's fleet. Microsoft patched server-side before the March advisories, so there is no customer action; teams running ImageMagick-compatible conversion on attacker-controlled uploads should disable delegates and keep workers off the network.

BGP ORIGIN attribute manipulation and its impact on the Internet (1 minute read)

Cloudflare analyzed the BGP ORIGIN attribute, a mandatory path attribute that should stay fixed after the originating AS sets it, and found that roughly 70% of observed paths across multiple vantage points carried a different ORIGIN value than the origin AS assigned. Because routers break path-selection ties by preferring the lower ORIGIN value, this widespread manipulation quietly reshapes how traffic is forwarded, giving operators a lever that can steer routing without touching AS_PATH or Local Preference.

Updated Cyber Threat Actor Naming System (2 minute read)

Google Threat Intelligence Group is rolling out a unified threat-actor naming schema, replacing the separate systems Mandiant and TAG maintained after the two teams merged into GTIG. The new convention uses two-word cryptonyms: the first word identifies the actor (often carried over from prior public reporting), and the second categorizes the cluster by motivation, attribution, or activity type. GTIG notes that cross-vendor comparisons remain imperfect since no two organizations share the same visibility, so defenders should treat the scheme as a mapping aid, not ground truth.

Slopsquatting, Phantom Domains, and HalluSquatting Are the Same AI Attack (1 minute read)

Researchers at Tel Aviv University, Technion, and Intuit showed that slopsquatting, phantom domains, and HalluSquatting are one attack: AI coding agents fetch and execute hallucinated package, repo, and skill names as if verified. The hallucinations are predictable, repeating identical fake names up to 85% of the time for repository requests and 100% for skill installs across Cursor, Copilot, Cline, and Gemini CLI. Defenders should gate every agent-suggested identifier against an allowlist or registry check before the agent resolves it.

Microsoft tightens Windows enterprise activation security (1 minute read)

Microsoft is making TPM-backed attestation mandatory for the Windows KMS volume-activation service, replacing software-only trust with a hardware-rooted chain of trust between clients and the KMS host. The change blocks attackers from copying or impersonating a KMS server and lands with the next Windows Server LTSC release. Admins should inventory KMS hosts now, confirm TPM support and Windows Server certification, and watch for readiness messaging starting August 2026 before enforcement begins.
