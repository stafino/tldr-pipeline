Decades-Old Squid Proxy Flaw ‘Squidbleed’ Can Expose User Data (1 minute read)

Researchers at Calif.io disclosed Squidbleed (CVE-2026-47729), a Heartbleed-style memory leak in the Squid open source proxy that has persisted since 1997 and can expose other users' cleartext HTTP request data, including credentials, session tokens, and API keys. The bug is patched in Squid 7.6 and 8 and matters most in shared-proxy environments like corporate networks and public Wi-Fi, where IT teams should patch or disable FTP support.

New Exploit Bypasses Apple’s Boot Defenses, Affects Millions of iPhones (1 minute read)

European research firm Paradigm Shift disclosed Usbliter8, an unpatchable BootROM exploit in Apple's SecureROM that grants full code execution before the OS loads on iPhones with A12 and A13 chips, including the XS, XR, and 11, plus S4 and S5 Apple Watches. The flaw requires physical USB access and cannot reach Secure Enclave-protected user data, but it hands forensics vendors a durable unlock path on millions of 2018-2019 devices that no patch can close.

Lessons from the VMwars – nothing virtual about the Broadcom vs Tesco slugfest (1 minute read)

The practical takeaway for IT leaders is to model migration costs and contractual exit terms before the next renewal, not after the invoice arrives.
