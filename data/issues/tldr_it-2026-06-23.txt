Microsoft fixes AutoGen Studio flaw that enabled code execution (1 minute read)

Microsoft patched a vulnerability chain dubbed AutoJack in AutoGen Studio, the GUI for its open-source multi-agent framework, which could let a malicious webpage trick a developer's browsing agent into executing arbitrary PowerShell or Bash commands on the host. The flaw was fixed before any PyPI release, so only developers who built from the main GitHub branch during a narrow window were exposed.

Fortinet: Oracle and other goliaths in firewall fallout as GPU 'democratization' blamed for mass hack (1 minute read)

Attackers compromised roughly 74,000 internet-facing FortiGate devices, nearly half of all publicly exposed Fortinet firewalls, harvesting enterprise credentials from organizations including Oracle, Samsung, Siemens, PwC, Accenture, and a NATO defense contractor. Researchers tied the campaign to a rented 45-GPU cluster used to crack and generate password variants at scale, evidence that cheap on-demand GPU access has lowered the bar for credential attacks on enterprise edge infrastructure.

A new unpatchable flaw in Apple chips opens the door to an iPhone jailbreak (1 minute read)

Paradigm Shift, a Barcelona-based offensive security firm, published "usbliter8," a proof-of-concept exploit for an unpatchable Boot ROM flaw in Apple's A12 and A13 chips that affects the iPhone XS, XR, and 11. Because the flaw sits in immutable silicon, no software update can fix it, leaving hardware refresh the only mitigation for older fleets and handing forensics vendors like Cellebrite another physical-access path into seized devices.

FFmpeg fixes PixelSmash flaw in widely used video decoder (1 minute read)

FFmpeg patched PixelSmash (CVE-2026-8461), a high-severity heap out-of-bounds write in the MagicYUV decoder that JFrog used to achieve remote code execution on a Jellyfin media server through its normal library-scan pipeline. Any application built on libavcodec is exposed, including Kodi, OBS Studio, Nextcloud, PhotoPrism, and Linux desktop thumbnail generators, so IT teams should prioritize updating FFmpeg across self-hosted media and file-preview infrastructure.

FortiBleed campaign used custom FortiGate sniffer to steal credentials (1 minute read)

SOCRadar reported that the FortiBleed campaign has compromised FortiGate firewalls since at least February 2026 using "FortigateSniffer," a Golang tool that abuses FortiOS's built-in diagnose sniffer command to harvest RADIUS, NTLM, Kerberos, and LDAP credentials across 24 protocols. The finding contradicts Fortinet's claim that the incident is only a collection of old leaked credentials, indicating an active operation against more than 430,000 internet-exposed devices.

Microsoft and Chevron plan one of the largest gas-powered data center projects in US (1 minute read)

Microsoft and Chevron will develop Project Kilby, a 2.67-gigawatt natural gas plant in West Texas dedicated to a Microsoft-operated AI and cloud data center under a 20-year power purchase agreement. The dedicated buildout signals that Microsoft is prioritizing guaranteed AI compute capacity over its 2030 carbon-neutral pledge, since the plant could emit more than 13 million tons of CO2 annually.

ShapedPlugin WordPress Pro Plugins Backdoored in Supply Chain Attack (1 minute read)

Attackers compromised ShapedPlugin's build and distribution pipeline, injecting backdoor code into Pro releases of Product Slider Pro for WooCommerce, Real Testimonials Pro, and Smart Post Show Pro delivered through the vendor's licensed update channels, with one flaw rated CVSS 10.0. The malware harvests wp-config.php credentials, admin accounts, SMTP secrets, and recent WooCommerce order data, making this a procurement-level reminder to audit third-party plugin supply chains, not just core platforms.

Linux and Secure Boot certificate expiration (2025) (1 minute read)

Microsoft's 2011 key used to sign the shim UEFI bootloader for Secure Boot expires in September, after which it will no longer sign the first-stage loader that Linux distributions rely on to boot under Secure Boot. A replacement key has existed since 2023 but is absent from many systems and may require a vendor firmware update, leaving distributors and IT teams to handle new installs and patch coverage before the deadline.
