Cisco Unified CM flaw actively exploited to drop webshells (CVE-2026-20230) (1 minute read)

Attackers are actively exploiting CVE-2026-20230, an unauthenticated SSRF flaw in Cisco Unified Communications Manager, to drop webshells and gain remote code execution on the underlying server. Cisco patched the bug on June 3 and a public PoC landed this week, so unpatched IP telephony deployments should upgrade now or disable the vulnerable WebDialer service as an interim mitigation.
