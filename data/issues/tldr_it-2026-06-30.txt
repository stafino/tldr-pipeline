US Supreme Court Just Blew Up EU-US Data Transfers (1 minute read)

The US Supreme Court ruled in Trump v. Slaughter that the FTC may no longer be independent, undercutting the EU-US Data Privacy Framework, which names the "independent" FTC as enforcer 259 times. Max Schrems and noyb are pressing the European Commission to withdraw the adequacy decision, which would reopen the transfer-mechanism scramble for every EU company that outsourced personal data to US cloud providers like AWS, Microsoft, and Google.

CISA: Windows BlueHammer flaw now exploited by ransomware gangs (1 minute read)

CISA added the Windows BlueHammer flaw (CVE-2026-33825), a Microsoft Defender privilege-escalation bug, to its Known Exploited Vulnerabilities catalog and confirmed ransomware gangs are now exploiting it to reach the SAM database and escalate to SYSTEM. Microsoft patched the flaw in April after a researcher leaked proof-of-concept code, so unpatched Windows fleets remain the exposure enterprise IT teams need to close.

Amazon EKS now supports control plane egress through your VPC (5 minute read)

AWS added customer-routed control plane egress to Amazon EKS, letting teams route Kubernetes API Server outbound traffic—admission webhook callbacks, OIDC discovery lookups, and aggregate API server requests—through an ENI in their own VPC under existing security group, endpoint policy, and Network Firewall controls. The feature targets regulated buyers in government, financial services, and healthcare, and supports org-wide enforcement via AWS Organizations Service Control Policies using the eks:controlPlaneEgressMode condition key.

Nissan says Oracle PeopleSoft break-in may have spilled payroll records, SSNs (1 minute read)

Nissan disclosed that an intrusion into its Oracle PeopleSoft system may have exposed employee payroll records and Social Security numbers, marking another breach tied to enterprise HR and ERP platforms. The incident pressures Oracle PeopleSoft customers to audit their on-prem deployments and patch cadence, since legacy ERP systems holding sensitive personnel data remain high-value targets even as vendors push migrations to managed cloud.

In Less Than 24 Hours, Attackers Weaponize Cisco CUCM Flaw (1 minute read)

Attackers began exploiting CVE-2026-20230, a critical SSRF flaw in Cisco Unified Communications Manager that grants unauthenticated remote attackers root access, less than 24 hours after SSD Secure Disclosure published a full exploit chain. CUCM serves roughly 30 million users globally, and Cisco shipped fixed versions on June 3, urging admins to patch deployments running the optional WebDialer service rather than wait on the 8.6 CVSS score.

Critical SimpleHelp flaw exploited to deploy new stealer malware (1 minute read)

Hackers are exploiting a critical authentication-bypass flaw in SimpleHelp (CVE-2026-48558) to create privileged technician accounts and deploy two previously undocumented tools, the TaskWeaver loader and Djinn Stealer, an cross-platform infostealer focused on developer and cloud credentials. Because SimpleHelp is RMM software used by MSPs and IT helpdesks, a single compromised server hands attackers a trusted admin channel into every managed endpoint, and roughly 1,000 exposed servers were running the vulnerable OIDC configuration at disclosure.

How the Senate’s AI AGENT Act could reshape enterprise AI governance (2 minute read)

Senator Mark Warner released a discussion draft of the AI AGENT Act, which would require providers of consumer "custodial user agents" to register with the FTC and link each agent to an authorizing user before accessing large online platforms. For enterprises, the user-linkage requirement is the consequential part, forcing CIOs and CISOs to build continuous traceability for automated decisions rather than absorbing it through existing supplier review and identity systems.

A Hybrid Framework For Crypto-Ransomware Detection In Enterprise Shared Storage (1 minute read)

The authors propose a hybrid framework for detecting crypto-ransomware that spreads from compromised clients to network drives and file servers, where endpoint detection tools stay blind because the servers themselves show no behavioral change. The approach extracts Indicators of Compromise from network traffic to extend existing EDRs and IDSs, and trains an ML model that the paper reports catches evasive variants at 99.64% precision with a 0% false-negative rate before encryption does meaningful damage.

What the post-quantum executive order really demands of CISOs (1 minute read)

The author argues the new post-quantum executive order reframes PQC as a readiness and ownership problem for CISOs, not a future cryptography exercise. Federal high-value systems must move key establishment to PQC by Dec. 31, 2030 and digital signatures by Dec. 31, 2031, deadlines that already sit inside enterprise procurement and architecture-review horizons. The more urgent risk is "Harvest Now, Decrypt Later" collection, which means long-lived sensitive data is functionally exposed today regardless of the transition timeline.

How to Govern Autonomous Agents in Enterprise AI Factories (1 minute read)

NVIDIA published its Secure Agent Workspace Reference Design, which moves agent execution off the employee's laptop, browser, or IDE and into a managed VM where identity, credentials, network access, runtime policy, and audit are enforced centrally. The argument for IT leaders is that always-on agents touching internal systems need the same governed perimeter as any privileged workload, with the endpoint reduced to a presentation layer and the VM as the isolation boundary.
