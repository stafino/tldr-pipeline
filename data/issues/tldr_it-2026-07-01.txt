ChocoPoc malware delivered via trojanized exploits on GitHub (1 minute read)

Sekoia identified at least seven trojanized proof-of-concept exploit repositories on GitHub delivering ChocoPoC, a Python-based RAT that runs shell commands, steals browser credentials, and exfiltrates data through abused Mapbox datasets. Rather than embedding malware in the exploit code, the campaign hides it in PyPI dependencies named frint and skytext, the latter pulled 2,400 times and aimed at security researchers testing trending CVEs.

Microsoft Removes 119 Edge Extensions That Hid Malware in Images and Fonts (1 minute read)

Microsoft removed 119 malicious Edge extensions tied to a single threat actor it calls StegoAd, which hid payloads inside PNG, WebP, and WOFF2 files using steganography to evade scanners before stealing credentials and running ad fraud. The campaign had been live since 2021 across a 2.6 million install base, exposing how thinly the Edge Add-ons store vets the ad blockers, VPNs, and translators enterprise users sideload onto managed browsers.

How is AI changing datacenter network fabrics? (1 minute read)

The Register examines how AI training and inference workloads are reshaping datacenter networking, as dense GPU clusters push east-west traffic and lossless, high-bandwidth fabrics to the center of infrastructure design. The piece argues that the old spine-leaf assumptions no longer hold, with 400/800GbE, RoCE, and InfiniBand now gating cluster performance. For IT buyers, the takeaway is that network fabric, not just compute, is becoming a primary constraint and line item in AI buildouts.

Infosec professionals sour on automated pentesting tools (1 minute read)

Security professionals have turned sharply against fully autonomous penetration testing, with the share willing to hand pentests entirely to automated tools falling from 29% last year to 9% now. The reversal signals that early enthusiasm for AI-driven offensive security has run into accuracy and trust limits, and vendors selling autonomous pentesting are likely to face longer procurement cycles as buyers insist on human oversight in the loop.

How to Govern Autonomous Agents in Enterprise AI Factories (1 minute read)

NVIDIA published its Secure Agent Workspace Reference Design, which moves agent execution off the employee's laptop, browser, or IDE and into a managed VM where identity, credentials, network access, runtime policy, and audit are enforced centrally. The argument for IT leaders is that always-on agents touching internal systems need the same governed perimeter as any privileged workload, with the endpoint reduced to a presentation layer and the VM as the isolation boundary.
