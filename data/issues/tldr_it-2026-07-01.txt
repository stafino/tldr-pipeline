Amazon EKS now supports control plane egress through your VPC (5 minute read)

AWS added customer-routed control plane egress to Amazon EKS, letting teams route Kubernetes API Server outbound traffic—admission webhook callbacks, OIDC discovery lookups, and aggregate API server requests—through an ENI in their own VPC under existing security group, endpoint policy, and Network Firewall controls. The feature targets regulated buyers in government, financial services, and healthcare, and supports org-wide enforcement via AWS Organizations Service Control Policies using the eks:controlPlaneEgressMode condition key.

Oracle shipped a fix in its May 2026 Critical Patch Update, but the ongoing campaign follows Clop and ShinyHunters intrusions into Oracle EBS and PeopleSoft, making immediate patching an urgent priority for enterprise IT teams.

In Less Than 24 Hours, Attackers Weaponize Cisco CUCM Flaw (1 minute read)

Attackers began exploiting CVE-2026-20230, a critical SSRF flaw in Cisco Unified Communications Manager that grants unauthenticated remote attackers root access, less than 24 hours after SSD Secure Disclosure published a full exploit chain. CUCM serves roughly 30 million users globally, and Cisco shipped fixed versions on June 3, urging admins to patch deployments running the optional WebDialer service rather than wait on the 8.6 CVSS score.

BlueHammer Vulnerability Exploited in Ransomware Attacks (1 minute read)

CISA updated its Known Exploited Vulnerabilities catalog to confirm that BlueHammer (CVE-2026-33825), a Microsoft Defender privilege-escalation flaw, is now being used in ransomware campaigns, though Microsoft's advisory still stops short of confirming in-the-wild exploitation. The bug was one of several dropped early by a researcher feuding with Microsoft over its patch handling, and Huntress observed zero-day exploitation before the April 14 fix shipped.
