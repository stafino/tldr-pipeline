Amazon EKS now supports control plane egress through your VPC (5 minute read)

AWS added customer-routed control plane egress to Amazon EKS, letting teams route Kubernetes API Server outbound traffic—admission webhook callbacks, OIDC discovery lookups, and aggregate API server requests—through an ENI in their own VPC under existing security group, endpoint policy, and Network Firewall controls. The feature targets regulated buyers in government, financial services, and healthcare, and supports org-wide enforcement via AWS Organizations Service Control Policies using the eks:controlPlaneEgressMode condition key.

CISA Warns of Actively Exploited Microsoft SharePoint Vulnerability (1 minute read)

CISA added CVE-2026-45659, a high-severity SharePoint Server deserialization bug patched out-of-band in late May, to its Known Exploited Vulnerabilities catalog, giving federal agencies three days to patch under BOD 26-04. The flaw lets an authenticated attacker with only Site Member permissions run arbitrary code, and it affects Subscription Edition, 2019, and 2016 servers that anchor enterprise document sharing and intranets.

Enterprise Tech In, Shell Out (Progress Kemp LoadMaster Uninitialized Heap to Pre-Auth RCE CVE-2026-8037) (5 minute read)

Progress disclosed CVE-2026-8037, a pre-authentication remote code execution flaw in Kemp LoadMaster, its edge load balancer and application delivery controller, exploitable by any unauthenticated attacker who can reach the API on GA builds v7.2.63.1 and older or LTSF v7.2.54.17 and older. Progress, the vendor behind the widely exploited MoveIt breaches, has patched the issue, and enterprises running LoadMaster at the network edge should update immediately or disable the API.

SharePoint RCE CVE-2026-45659 Added to CISA KEV After Active Exploitation (1 minute read)

CISA added SharePoint Server flaw CVE-2026-45659 (CVSS 8.8), a deserialization remote-code-execution bug, to its Known Exploited Vulnerabilities catalog after confirming active exploitation, and set a July 4 patch deadline for federal agencies. Microsoft fixed the flaw in May 2026 for Subscription Edition, Server 2019, and Enterprise Server 2016, but any authenticated user with Site Member permissions can trigger it, so on-prem SharePoint operators should prioritize the update.

In Less Than 24 Hours, Attackers Weaponize Cisco CUCM Flaw (1 minute read)

Attackers began exploiting CVE-2026-20230, a critical SSRF flaw in Cisco Unified Communications Manager that grants unauthenticated remote attackers root access, less than 24 hours after SSD Secure Disclosure published a full exploit chain. CUCM serves roughly 30 million users globally, and Cisco shipped fixed versions on June 3, urging admins to patch deployments running the optional WebDialer service rather than wait on the 8.6 CVSS score.

Gartner forecasts that "agentic arbitrage" will reshape up to $234B in enterprise application spending through 2030, roughly 20% of projected SaaS outlays, as AI agents execute work across systems and bypass the user interfaces vendors monetize. The report argues this breaks the link between seat growth and revenue, threatening dashboard- and per-user pricing models. What Meta, Oracle moves say about data center economics (3 minute read)

Oracle's latest 10-K and Meta's reported plan to resell excess compute signal that hyperscalers are shifting from an AI capacity land grab to managing the financial risk of the buildout. Oracle's filing details how supply-chain delays, permitting limits, and third-party defaults could inflate the cost of commitments like its $300B Stargate deal with OpenAI. With the top four hyperscalers set to spend up to $725B on AI infrastructure this year, the pressure is now on return, not just scale.

Mapping out your unknown: A threat hunter's guide to Salesforce (5 minute read)

Datadog's Security Research team published the first part of a threat-hunting guide for Salesforce, documenting how recent campaigns abuse OAuth connected apps, compromised third-party integrations, and hijacked SSO sessions to enumerate and exfiltrate data. The post maps each attacker step to a MITRE ATT&CK tactic and provides audit-log queries tuned to Salesforce's logging tiers, from Event Log Files to Real-Time Event Monitoring. For enterprises with dozens of connected apps per Salesforce tenant, it is a practical baseline for detection coverage most teams currently lack.
