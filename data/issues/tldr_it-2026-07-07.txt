Linux Kernel Vulnerability Allows VM Escape on Intel and AMD Systems (1 minute read)

Security researchers disclosed CVE-2026-53359, a use-after-free flaw in the KVM hypervisor's shadow MMU code, dubbed Januscape, that lets an attacker with root on a guest VM escape to the host and run code as root or crash every tenant on the same machine. It is the first KVM VM-escape known to work on both Intel and AMD, sat dormant for 16 years, and was patched in the mainline kernel on June 19.

Fake IT support calls on Microsoft Teams push EtherRAT malware (1 minute read)

Attackers are impersonating corporate IT support over Microsoft Teams voice calls to trick employees into granting remote control and installing EtherRAT, a Node.js remote-access trojan that hands over full system control, per Palo Alto Networks' Unit 42. The campaign chains phishing emails, legitimate tools like AnyDesk and HopToDesk, and Ethereum smart contracts for resilient C2, and follows a wave of Teams-based intrusions pushing Microsoft to add new tenant-level protections.

Microsoft to enable Windows settings backup by default for orgs (1 minute read)

Microsoft will switch the Windows settings backup policy from disabled to enabled by default on Entra-joined and Entra hybrid-joined systems running Windows 11 26H2, excluding EU DMA-regulated regions and sovereign cloud environments. Admins keep full control through Intune or Group Policy, where explicit enable or disable settings override the default, and restore behavior still requires manual configuration.

Microsoft testing new Cloud Rebuild Windows 11 recovery feature (1 minute read)

Microsoft began testing Cloud Rebuild in Windows 11 Insider Experimental builds, a recovery option that performs a full OS reinstall from Windows Update, pulling both the target image and device drivers so a machine returns to a known-good state even when it won't boot. It joins Point-in-Time Restore and Quick Machine Recovery under Microsoft's Windows Resiliency Initiative, cutting the need for USB media, custom images, or on-site IT access to recover fleet devices.
