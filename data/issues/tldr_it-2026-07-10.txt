Microsoft Patches RoguePlanet Defender Flaw That Can Grant SYSTEM Privileges (1 minute read)

Microsoft patched RoguePlanet (CVE-2026-50656, CVSS 7.8), a race-condition privilege-escalation flaw in the Defender malware protection engine that could spawn a SYSTEM-level shell, nearly a month after the researcher Chaotic Eclipse disclosed it publicly. The fix ships automatically via engine version 1.1.26060.3008 with no admin action required, but the exploit worked on fully June-patched Windows regardless of whether real-time protection was enabled.

Microsoft patches RoguePlanet Defender zero-day vulnerability (1 minute read)

Microsoft patched a Microsoft Defender zero-day tracked as CVE-2026-50656, dubbed RoguePlanet, which let attackers win a race condition to spawn a SYSTEM-level command prompt on fully updated Windows 10 and 11 machines. The fix ships automatically through Malware Protection Engine version 1.1.26060.3008 rather than a manual patch, but it landed amid a public feud with the researcher who disclosed it and whom Microsoft has threatened with legal action.

SAP makes it easier for customers to shop for legacy product support, ending EU antitrust probe (1 minute read)

SAP agreed to loosen the terms tying customers to its own maintenance for legacy products, resolving a European Commission antitrust investigation into whether the vendor locked out third-party support rivals. Enterprises running older SAP systems gain more room to shop maintenance contracts with providers like Rimini Street, which lowers the switching cost of staying on legacy software rather than migrating to S/4HANA.

Ubuntu emphasizes Arm64 support – and gets Rustier (1 minute read)

Canonical is elevating Arm64 to first-class status in Ubuntu and swapping more core utilities for memory-safe Rust reimplementations, signaling where its LTS base is headed. For enterprise fleets, the Arm64 commitment strengthens the case for Ampere and Graviton deployments, while the Rust shift means shops standardized on GNU coreutils should test compatibility before the next LTS lands.

AT&T, T-Mobile US, and Verizon ditch SMS for network-based authentication (1 minute read)

AT&T, T-Mobile US, and Verizon launched a network-based authentication service through the Aduna venture, using a single API to verify a mobile number against its carrier, device, and SIM and replace SMS one-time passcodes. The move targets OTP interception and SIM-swap fraud tied to $15.9B in U.S. consumer losses last year, while giving developers one integration across all three carriers instead of per-operator deals.

SAP agreed to European Commission commitments that let on-premises ERP customers buy maintenance and support from third-party providers, clarify original licensing terms, and terminate license and support fees in defined scenarios, with the changes binding globally for 10 years. Entra passkey enrollment vishing targets Microsoft 365 users (1 minute read)

Intel and AMD systems at risk from virtualization vulnerability (1 minute read)

Security researchers disclosed Januscape (CVE-2026-53359), a 16-year-old KVM hypervisor flaw that lets a single rented VM escape to the x86 host and either crash or take root control of every other tenant on the same physical machine. The bug has been patched, but it hits Intel and AMD hosts across public clouds while sparing Arm64 KVM, giving IT teams another urgent hypervisor-patching cycle to schedule.

Proof-of-Concept Exploit Released for Linux ‘Bad Epoll' Root Access Vulnerability (5 minute read)

Researchers published a working proof-of-concept for Bad Epoll (CVE-2026-46242, CVSS 7.8), a use-after-free race condition in the Linux kernel's epoll facility that lets unprivileged processes gain root on servers, desktops, and Android devices running kernel 6.4 or newer. The exploit was confirmed on Pixel 10 hardware, and a complete fix landed only two months after the initial report, leaving enterprise Linux fleets exposed to patch-management urgency.

AI Coding Agents Found Triggering Endpoint Security Rules Built to Catch Attackers (1 minute read)

Sophos analyzed a week of its own June 2026 endpoint telemetry and found AI coding agents including Claude Code, Cursor, and OpenAI Codex tripping behavioral detection rules built to catch human intruders, with credential access making up 56.2% of blocked activity. The agents were doing ordinary developer work, but actions like DPAPI browser-credential decryption and living-off-the-land payload fetches are indistinguishable from attacks, leaving security teams to tune endpoint policy for legitimate agent behavior.
