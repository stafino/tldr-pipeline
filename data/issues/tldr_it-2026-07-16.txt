Microsoft’s Secure Boot has been broken for a decade and no one noticed until now (1 minute read)

ESET researchers found that Microsoft left 11 vulnerable UEFI Secure Boot shims signed and unrevoked for up to 13 years, letting a novice attacker bypass the boot protection on both Windows and Linux devices using nothing but an old, still-trusted binary. Because the flaw lets bootkits persist through OS reinstalls and drive swaps, IT teams relying on Secure Boot as a firmware-integrity baseline now need to audit revocation lists and push UEFI dbx updates across their fleets.

Two SonicWall SMA 1000 Zero-Days Exploited, One Could Enable Admin Commands (1 minute read)

SonicWall confirmed active exploitation of two SMA 1000 zero-days, including CVE-2026-15409, a maximum-severity 10.0 SSRF flaw, and CVE-2026-15410, a post-authentication bug that lets attackers run OS commands as administrator. CISA added both to its Known Exploited Vulnerabilities catalog, giving federal agencies until July 17 to patch, and SonicWall is urging customers to apply fixes and run forensic checks for compromise indicators.

TS-2026-009: Insecure argument handling in Tailscale SSH permitted root access (1 minute read)

Tailscale disclosed TS-2026-009, a flaw in which Tailscale SSH accepted usernames with leading dashes that Linux passed to getent(1) as flags, letting an attacker connecting as `-i` dump the passwd file and open an interactive root session in violation of ACLs. Nodes running Tailscale SSH on versions before 1.98.9 are affected, and IT teams should upgrade to 1.98.9 or newer to close it.

SAP Patches CVSS 9.9 NetWeaver ABAP Flaw That Could Expose or Modify Data (1 minute read)

SAP patched CVE-2026-44747, a CVSS 9.9 out-of-bounds write flaw in NetWeaver Application Server ABAP that lets an authenticated attacker corrupt memory to read, modify, or take down SAP systems. The July update also fixes two CVSS 9.1 critical bugs, including default OAuth credentials in Commerce Cloud that an unauthenticated attacker can use to pull a valid token and alter data.
