Inc Ransomware Exploits SonicWall SMA Zero-Days (1 minute read)

HollowByte DDoS flaw bloats OpenSSL server memory with 11-byte payload (1 minute read)

Okta's Red Team disclosed HollowByte, an unauthenticated OpenSSL flaw that lets an 11-byte TLS handshake payload force the server to allocate large memory buffers it never fills, permanently bloating resident memory until the process restarts. OpenSSL silently patched the bug with no CVE assigned and backported the fix to older releases, so IT teams running NGINX, Apache, Node.js, or PostgreSQL should prioritize upgrading the embedded library.
