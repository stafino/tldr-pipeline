WordPress Core "wp2shell" RCE flaws get public exploits, patch now (1 minute read)

Public proof-of-concept exploits are now available for "wp2shell," a chain of two WordPress Core flaws (CVE-2026-63030 and CVE-2026-60137) that gives unauthenticated attackers remote code execution on stock installs running 6.9.0-6.9.4 and 7.0.0-7.0.1. WordPress has forced auto-updates to 7.0.2 and 6.9.5, but IT teams running self-hosted or version-pinned instances need to patch manually across an estimated 500M sites.

Critical ServiceNow code execution flaw now exploited in attacks (1 minute read)

Attackers are exploiting CVE-2026-6875, a pre-auth sandbox-escape RCE in the ServiceNow AI Platform, days after ServiceNow patched hosted instances and shipped fixes for self-hosted deployments on July 13th. ServiceNow's advisory still says it is unaware of active exploitation, so self-hosted customers running the platform that serves 85% of the Fortune 500 should patch on their own timeline rather than the vendor's.

Major glitch in Matrix as UK government projects watchdog rates shared services cluster red (1 minute read)

The UK government's projects watchdog gave a red rating to Matrix, the shared services cluster consolidating back-office ERP and HR systems across multiple Whitehall departments, meaning delivery appears unachievable without a reset of scope, schedule, or budget. Multi-department ERP consolidation is the same pattern vendors sell to large enterprises, and the red flag lands on the delivery model rather than the software itself.

Critical NGINX Vulnerability Can Crash Workers and May Allow Remote Code Execution (1 minute read)

F5 patched CVE-2026-42533, a heap buffer overflow in nginx's script engine that lets an unauthenticated attacker crash worker processes with crafted HTTP requests and, where ASLR can be bypassed, potentially execute code. Fixes landed in nginx 1.30.4, 1.31.3, and NGINX Plus 37.0.3.1, but F5 has not yet published fixed builds for Ingress Controller, Gateway Fabric, App Protect WAF, or Instance Manager.

Microsoft confirms Windows Server Update Services sync delays (1 minute read)

Microsoft confirmed a WSUS synchronization bug affecting Windows 10 1607 and later and Windows Server 2012 and later, with sync timeouts starting July 13 and blocking update deployment through WSUS and Configuration Manager. Saturday's mitigation restored new and rebuilt WSUS installations, but previously affected servers still need manual removal of the bad metadata, and Microsoft has not shipped those steps yet.

Windows KB5121767 OOB update fixes shutdowns on some Dell PCs (1 minute read)

Microsoft released out-of-band updates KB5121767 and KB5121768 to fix unexpected shutdowns, thermal throttling, and battery drain on Dell PCs running Windows 11 25H2, 24H2, and Enterprise LTSC 2024, caused by a conflict between the new Windows USB-C Connection Manager and Intel's IPF Processor Participant driver. Autopatch fleets with hotpatching get the fix automatically, while everyone else needs to pull it manually and only on affected devices.

Auditors tell UK government to do the math before banking on £45B AI savings (1 minute read)

The UK's National Audit Office told the government it lacks the baseline data and measurement framework to verify the £45B in annual savings ministers have projected from AI adoption across public services. The finding lands as departments negotiate AI contracts with Microsoft, Google, and other vendors, and it raises the odds that procurement teams get asked to justify spend against benchmarks nobody has actually established.

High hardware costs see Red Hat offer a two-server edge rig, no mini-PC required (1 minute read)

Red Hat introduced a two-server edge configuration for OpenShift, dropping the third control-plane node that its compact cluster deployments previously required as hardware costs climb. IT teams running hundreds of distributed sites can cut per-location server spend without falling back to single-node clusters, which trade away the failover that retail, manufacturing, and telco edge workloads depend on.

The audit trail CIOs need before the next cyber crisis (2 minute read)

The author argues that DORA, NIS2, and the SEC's cyber disclosure rules have shifted CIO liability from operating controls to proving, after the fact, that leadership decisions matched the risk evidence available at the time.
