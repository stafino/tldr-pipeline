Two SonicWall SMA1000 zero-days, CVE-2026-15409 and CVE-2026-15410, were exploited by threat actor UTA0533 for roughly three weeks before SonicWall issued hotfixes on July 14, per Volexity, which found the group deploying custom KnuckleBall malware alongside a Java webshell and open-source proxy. CISA has added both flaws to its KEV catalog, bringing SonicWall's total to 17, and enterprises running the remote-access appliances should apply the hotfixes and hunt for the published IoCs.

Microsoft 365 calendars become spy drop boxes in HOLLOWGRAPH campaign (1 minute read)

Attackers in the HOLLOWGRAPH campaign are abusing Microsoft 365 calendar entries as covert dead drops, hiding stolen data and command-and-control instructions inside event fields that blend into normal Graph API traffic. The technique lets exfiltration ride legitimate Microsoft 365 channels that most enterprise security tooling trusts by default, forcing IT teams to treat Graph API activity and calendar writes as monitored surfaces rather than benign background noise.

HOLLOWGRAPH malware turns Microsoft 365 calendars into an espionage channel (1 minute read)

Group-IB researchers documented HOLLOWGRAPH, espionage malware that uses a compromised Microsoft 365 mailbox and the Graph API to hide command-and-control traffic inside calendar events dated to 2050, with DNS tunneling to refresh Entra ID credentials. The campaign hit 12 systems tied to Israeli entities and shows how trusted Microsoft services and legitimate Graph API access can carry data exfiltration past defenses watching for conventional C2 channels.
