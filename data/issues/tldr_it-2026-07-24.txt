Microsoft blames massive Microsoft 365 outage on maintenance bug (1 minute read)

Microsoft attributed Thursday's Microsoft 365 and Azure outage to a bug in its automated network maintenance system that removed IP routes from more devices than intended in the West US region. The incident degraded SharePoint, Teams, OneDrive, Copilot, and Defender for over three hours until Microsoft reverted the change, a reminder for IT teams to test failover paths for single-region Microsoft dependencies.

Cloudflare Internal DNS puts public and private DNS on one policy engine (2 minute read)

Cloudflare launched Internal DNS, which extends its existing public DNS platform to private DNS so both run on one control plane, one API, one audit trail, and a single policy engine. Cloudflare Gateway becomes the resolver, evaluating zero-trust policies before routing each query to the right DNS view, which lets enterprises already on Cloudflare retire separate internal DNS appliances and split-horizon setups without a second console.
