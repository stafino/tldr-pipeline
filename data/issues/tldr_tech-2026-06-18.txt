Show HN: cuTile Rust: Safe, data-race-free GPU kernels in Rust (1 minute read)

cuTile Rust is an Nvidia research project for writing memory-safe, data-race-free GPU kernels in idiomatic Rust, extending ownership across the GPU launch boundary by partitioning mutable tensors into disjoint pieces while sharing immutable ones. A `#[cutile::module]` macro captures each kernel's AST and JIT-compiles it through CUDA Tile IR into a cubin, supporting synchronous launches, async pipelines, and CUDA graph replay.

RFC 10008: The new HTTP Query Method (1 minute read)

The IETF published RFC 10008, defining a new HTTP QUERY method for sending safe, idempotent requests with a body, filling the gap between GET and POST. Unlike GET, it carries query input as request content rather than in the URI, and unlike POST, it is explicitly safe and idempotent, so requests can be automatically retried or cached without risk of partial state changes.
