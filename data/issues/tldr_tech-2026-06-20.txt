Intel and AMD's new ACE CPU extensions bring an efficient AI-oriented instruction set to x86 — a new design makes matrix multiplication more power- and density-efficient (1 minute read)

Intel and AMD released the full specification for ACE, a shared x86 instruction set that adds dedicated matrix-multiplication silicon on top of existing AVX10 registers to run AI workloads on the CPU more power-efficiently. ACE performs up to 16x as many operations per input vector as AVX10 and is implementation-agnostic, letting frameworks like PyTorch and TensorFlow target one code path instead of vendor-specific variants.
