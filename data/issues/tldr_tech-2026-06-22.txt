Deno Desktop (1 minute read)

Deno desktop, shipping in canary v2.9.0, turns any Deno project from a single TypeScript file to a Next.js app into a self-contained desktop binary that bundles your code, the runtime, and a web rendering engine. It auto-detects frameworks like Next.js, Astro, and SvelteKit, defaults to the OS webview for small binaries, uses in-process bindings instead of IPC, and ships built-in binary-diff auto-updates.
