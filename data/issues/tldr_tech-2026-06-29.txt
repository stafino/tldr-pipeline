Obfuscation: building the final boss of cryptography (Part I) (5 minute read)

Vitalik Buterin's latest post explains indistinguishability obfuscation, a cryptographic technique that scrambles a program so anyone can run it but no one can read its internal logic or secrets. The cost YAGNI was never about (1 minute read)

Kent Beck argues that YAGNI was never about the cost of writing code, so cheap LLM generation doesn't retire it. The real cost is speculative structure: building ahead of a feature commits you to a guess, and even a correct guess leaves you worse off than waiting, since the value lies in keeping the option to build the right structure once you actually know what's needed.
