Memory Safe Context Switching (1 minute read)

Fil-C now supports the ucontext APIs (getcontext, setcontext, makecontext, swapcontext) and longjmp/setjmp in a memory-safe way, so misuse that would corrupt the stack in ordinary C instead panics the program. Restoring a dangling stack, freeing a stack still referenced by a context, or swapping into the currently executing context all become safety errors rather than the silent, untraceable crashes that attackers can exploit.

Popping the GPU Bubble (1 minute read)

Moondream's Photon inference engine tackles GPU bubbles, the idle gaps where the GPU waits on CPU housekeeping between tokens during autoregressive decoding. Their fix, pipelined decoding, launches the next token's GPU work while the CPU is still committing the current one, overlapping the two kinds of work so forwards run back-to-back instead of waiting on each baton pass.
